# 주문 · 결제

장바구니에 담았거나 상품 상세에서 바로 고른 상품을 **실제로 주문하고 결제**하는 화면입니다. 주소는 `/checkout` 이며, 결제는 **토스페이먼츠** 결제창에서 이뤄집니다.

로그인하지 않은 상태로 주문을 시작하면 **구매 방법 선택** 화면(`/checkout/choose`)이 먼저 나옵니다. 여기서 회원으로 로그인해 주문하거나, **비회원으로 주문**할 수 있습니다. 비회원 주문은 쿠폰·포인트·회원 등급 할인을 받을 수 없습니다.

주문서에 보이는 금액은 참고용이고, **최종 결제 금액은 저장 시점에 서버가 다시 계산**합니다. 고객 화면에서 값을 조작해도 결제 금액은 바뀌지 않습니다.

---

## 이 화면에 무엇이 보이나

**주문자 정보** — 이름 / 연락처 / 이메일
**배송지 정보** — 받는 분 / 연락처 / 우편번호 · 기본 주소 · 상세 주소 / **배송 메모**(선택형)
  - `주문자 정보와 동일` 체크박스로 배송지를 한 번에 채울 수 있습니다.
**할인 적용** — 쿠폰 / 배송비 쿠폰 / 포인트 / 회원 등급 할인
**결제 금액 요약** — 상품 금액 · 배송비 · 할인 합계 · 최종 결제금액

결제하기를 누르면 토스 결제창으로 이동하고, 성공하면 **주문 완료** 화면, 실패하면 **결제 실패** 화면이 나옵니다.

---

## 이 화면은 관리자에서 이렇게 만들어집니다

| 고객 화면의 이 부분 | 관리자 어느 기능으로 만드나 | 비고 |
|---|---|---|
| 배송비 금액, 지역 할증(제주·도서산간) | 배송비 정책 | 우편번호를 입력하면 할증이 다시 계산됩니다 |
| 무료배송 기준 | 배송비 정책의 무료배송 기준금액 | |
| **쿠폰** 드롭다운에 뜨는 목록 | 쿠폰 관리 | 유효기간 내 · 미사용 쿠폰만 나옵니다 |
| **배송비 쿠폰** 드롭다운 | 쿠폰 관리 (배송비 쿠폰으로 만든 것) | 주문 쿠폰과 각각 1장씩 쓸 수 있습니다 |
| 쿠폰 코드 입력칸의 동작 | 쿠폰 관리에서 **코드 입력형**으로 발급한 쿠폰 | |
| **포인트** 입력칸의 보유 잔액 | 포인트 관리 | |
| 포인트 최소 사용 단위 | 사이트 설정의 **포인트 최소 사용금액** | |
| 결제 후 적립되는 포인트 | 사이트 설정의 **구매 적립률(%)** + 멤버십 등급별 추가 적립 | 관리자가 0을 저장하면 "적립 없음"입니다 |
| 회원 등급 할인 · 등급 무료배송 | 멤버십 관리 | 비회원 주문에는 적용되지 않습니다 |
| 결제창이 뜨는 것 자체 | 사이트 설정의 **토스페이먼츠 키**(클라이언트 키 / 시크릿 키) | 키가 없으면 결제가 진행되지 않습니다 |
| 주문이 쌓이는 곳 | 판매(주문) 관리 | 결제 완료 시 주문 상태가 `결제완료`가 됩니다 |

> **결제 수단(카드·계좌이체·간편결제)의 종류는 이 쇼핑몰 관리자에서 고르지 않습니다.** 토스페이먼츠 상점 설정에서 결정되며, 관리자에서는 **키를 넣어 연동을 켜고 끄는 것**만 합니다.

> **쿠폰·포인트·특가는 실제 결제 금액을 깎습니다.** 테스트로 만든 100% 할인 쿠폰이나 과도한 적립률을 그대로 두면 그대로 팔립니다. 오픈 전에 쿠폰 관리와 적립률을 반드시 정리하세요.

## 사업자(B2B) 주문은 결제하지 않습니다

승인된 사업자가 **기업 구매 모드**로 주문하면 주문서가 이렇게 바뀝니다.

| 일반 주문 | 기업 주문 |
|---|---|
| 쿠폰·포인트 사용 | **쿠폰·포인트 입력란이 사라집니다**(전용가는 계약가이므로 중복 적용하지 않습니다) |
| 최종 결제금액 | **공급가액 / 부가세**를 나눠 표시하고 **주문 금액**으로 표기 |
| — | **발주번호(PO) · 납기 희망일 · 세금계산서 발행 · 요청사항** 입력란 추가 |
| `결제하기` → 결제창 | **`주문 요청하기`** → **접수 완료 화면**(결제창 없음) |

접수 뒤 절차는 **담당자 확인 → 승인·입금 안내 → 입금 확인 → 출고** 입니다. 접수 완료 화면에 **입금 계좌와 입금 기한**이 안내되고, 같은 내용이 이메일로도 발송됩니다.

주문번호는 일반 주문의 `ORD-` 대신 **`B2B-`** 로 시작합니다.

자세한 내용은 [사업자(B2B) 구매](/manual/user/b2b)를 보세요.

## 안 보이거나 비어 있을 때

**"결제하기를 눌러도 결제창이 안 뜹니다"**
사이트 설정에 **토스페이먼츠 클라이언트 키 / 시크릿 키**가 비어 있습니다. 두 값을 모두 넣어야 결제창이 열립니다.

**"쿠폰 드롭다운이 비어 있습니다"**
① 그 회원에게 발급된 쿠폰이 없거나, ② 유효기간이 지났거나, ③ 이미 사용했거나, ④ 쿠폰이 **다른 몰 전용**으로 만들어졌습니다. 쿠폰 관리에서 몰 지정과 유효기간을 확인하세요.

**"쿠폰이 목록에는 있는데 '○○원 이상 구매 시 사용 가능'이라고 뜹니다"**
쿠폰의 **최소 주문금액** 조건을 아직 못 채웠습니다. 쿠폰 관리에서 조건을 확인하세요.

**"포인트 입력칸이 안 보이거나 쓸 수 없습니다"**
비회원 주문에서는 포인트를 쓸 수 없습니다. 회원인데도 못 쓴다면 보유 포인트가 **최소 사용금액**보다 적은 경우입니다.

**"배송비가 예상과 다릅니다"**
우편번호를 넣는 순간 **지역 할증**이 반영됩니다. 배송비 정책의 지역(권역) 설정을 확인하세요.

**"등급 할인이 안 들어갑니다"**
① 비회원 주문이거나, ② 멤버십 관리에서 등급 혜택이 설정되지 않았거나, ③ 쿠폰과 등급 할인의 **중복 사용 방식**이 "중복 불가"로 되어 있을 수 있습니다.
