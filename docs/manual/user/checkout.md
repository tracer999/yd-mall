# 주문 / 결제

장바구니에 담은 상품을 **실제로 주문하고 결제**하는 단계입니다. **구매 방법 선택**(회원/비회원) → **주문서 작성** → **결제** → **주문 완료** 순으로 진행됩니다.

---

## 1. 구매 방법 선택 (비회원인 경우)

- **로그인하지 않은 상태**에서 **주문하기**를 누르면, **"회원으로 주문" / "비회원으로 주문"** 을 고르는 화면이 나올 수 있습니다.
- **회원으로 주문:** 로그인 화면으로 이동한 뒤, 로그인하면 주문서 작성 단계로 갑니다.
- **비회원으로 주문:** 로그인 없이 **주문자 이름·연락처·이메일·배송지** 등을 직접 입력하고 결제할 수 있습니다. (쇼핑몰이 비회원 주문을 허용한 경우)

---

## 2. 주문서 작성

- **주문할 상품 목록, 수량, 금액**이 표시됩니다.
- **수령인 정보:** 받는 분 이름, 휴대폰 번호, 받을 주소(우편번호, 주소, 상세주소)를 입력합니다. 회원은 **저장된 주소**가 자동으로 채워질 수 있습니다.
- **배송 메시지:** 택배 기사님께 전할 말 (예: "문 앞에 놓아 주세요")을 적을 수 있습니다.
- **쿠폰:** 보유한 **쿠폰**이 있으면 여기서 선택해 할인을 받을 수 있습니다. 적용 가능 금액·조건은 쿠폰마다 다릅니다.
- **포인트:** **적립금·포인트**를 사용할 수 있는 쇼핑몰은 사용할 포인트를 입력합니다. 최소 사용 단위·최대 사용 한도가 있을 수 있습니다.
- **총 결제 금액**이 **상품 금액 − 쿠폰 할인 − 포인트** 로 계산되어 표시됩니다.
- **주문하기** 또는 **결제하기** 버튼을 누르면 **결제 단계**로 넘어갑니다.

---

## 3. 결제하기

- **결제창**이 열리면, **카드·계좌이체·간편결제** 등 선택 가능한 수단 중 하나를 골라 결제를 진행합니다.
- **토스페이먼츠** 등 결제 대행사를 쓰는 쇼핑몰은 결제창에서 **카드 번호·유효기간·비밀번호** 등을 입력합니다.
- 결제가 **성공**하면 **주문 완료** 페이지로 이동합니다.
- 결제가 **실패**하면 (한도 초과, 카드 오류 등) **결제 실패** 페이지가 나오고, 다시 시도하거나 다른 결제 수단을 선택할 수 있습니다.

---

## 4. 주문 완료

- **주문 번호**와 **주문 내용·배송지·결제 금액** 등이 표시됩니다.
- **주문 번호**는 나중에 **배송 조회·문의**할 때 필요하므로 메모해 두거나 이메일·문자로 받은 내용을 보관하세요.
- **마이페이지**에서 주문 내역을 볼 수 있는 쇼핑몰은, 여기서 **주문 조회** 링크를 눌러 확인할 수 있습니다.

---

## 쿠폰 코드 입력 (있는 경우)

- **쿠폰 코드**를 따로 입력하는 칸이 있으면, **프로모션 코드** 또는 **쿠폰 번호**를 넣고 **적용** 버튼을 누릅니다.
- 적용되면 **할인 금액**이 반영되고, 적용 가능 조건(최소 주문 금액 등)에 맞지 않으면 안내 메시지가 나옵니다.

---

## 주의사항

- **재고**가 주문 처리 중에 부족해지면 결제 후 "재고 부족"으로 주문이 취소될 수 있습니다. 이 경우 결제 취소·환불 절차가 진행됩니다.
- **결제 완료 후** 주소·수량 변경이 필요하면 **1:1 문의**나 **고객센터**로 연락하는 것이 좋습니다. 쇼핑몰에 따라 배송 전 변경이 가능할 수 있습니다.
