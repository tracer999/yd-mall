# 장바구니

**구매를 생각하는 상품**을 일단 **담아 두었다가**, 나중에 **한꺼번에 주문**할 수 있는 기능입니다. **로그인한 회원**만 장바구니를 사용할 수 있습니다.

---

## 장바구니에 담기

1. **상품 상세 페이지**에서 **수량**을 선택합니다.
2. **장바구니 담기** 버튼을 클릭합니다.
3. 담기가 완료되면 **"장바구니에 담았습니다"** 같은 안내가 나오거나, **장바구니 페이지**로 이동할 수 있습니다.
4. **상단 메뉴의 장바구니 아이콘**에 **숫자**가 표시되면, 그 수만큼 상품(또는 종류)이 담겨 있다는 뜻입니다.

**재고가 부족한 경우** "재고가 부족합니다" 같은 메시지가 나오고, 수량을 줄이거나 다른 상품을 선택해야 합니다.

---

## 장바구니 목록 보기

1. **상단 메뉴의 장바구니 아이콘**을 클릭하면 **장바구니 페이지**로 이동합니다.
2. **담아 둔 상품 목록**이 나옵니다. 각 항목마다 **이미지, 상품명, 단가, 수량, 소계** 등이 보입니다.
3. **수량 변경:** 수량을 바꾸고 **변경** 또는 **적용** 버튼을 누르면 금액이 다시 계산됩니다. 재고보다 많이 넣으면 안내가 나올 수 있습니다.
4. **삭제:** 더 이상 사지 않을 상품은 **삭제** 버튼을 눌러 장바구니에서 뺄 수 있습니다.
5. **총 수량·총 금액**이 하단에 표시됩니다.

---

## 주문하기 (장바구니에서)

1. 장바구니에서 **주문할 상품**을 확인합니다. (필요 없으면 수량 조정·삭제)
2. **주문하기** 또는 **전체 구매** 버튼을 클릭합니다.
3. **주문/결제** 페이지로 이동합니다. 여기서 **수령인·배송지·결제** 등을 입력하고 결제를 진행합니다.

**장바구니가 비어 있으면** 주문하기 버튼을 눌러도 주문이 되지 않습니다. 먼저 상품을 담아 두세요.

---

## 장바구니와 로그인

- 장바구니는 **로그인한 회원**만 사용할 수 있습니다. 로그인하지 않고 "장바구니 담기"를 누르면 **로그인** 화면으로 이동할 수 있습니다.
- **다른 기기**에서 로그인해도 **같은 계정**이면 같은 장바구니 내용이 보입니다. (로그아웃하면 해당 기기에서는 장바구니가 비어 보일 수 있습니다.)
