# 장바구니

구매를 생각 중인 상품을 **일단 담아 두었다가 한꺼번에 주문**하는 화면입니다. 주소는 `/cart` 입니다.

장바구니는 **로그인한 회원만** 사용할 수 있습니다. 비로그인 상태에서 장바구니를 열면 로그인 화면으로 넘어갑니다. 같은 계정으로 로그인하면 다른 기기에서도 같은 장바구니가 보입니다.

담긴 상품의 가격은 **담을 당시 가격이 아니라 지금 가격**으로 계산됩니다. 그 사이 쇼핑특가가 시작되거나 끝나면 장바구니 금액도 함께 바뀝니다.

---

## 이 화면에 무엇이 보이나

- 담아 둔 상품 목록 (이미지 · 상품명 · 단가 · 수량)
- 수량 **+ / −** 버튼, 항목별 **삭제**
- 상품 금액 합계 / 배송비 / 합계 금액
- **무료배송까지 얼마 남았는지** 알려 주는 게이지 (예: "12,000원 더 담으면 무료배송")
- **주문하기** 버튼
- 장바구니가 비어 있으면 **상품 보러가기** 버튼만 보입니다

수량을 재고보다 많이 올리면 담기지 않고 재고 부족 안내가 뜹니다. 재고는 **옵션(SKU) 단위**로 확인하므로, 같은 상품이라도 선택한 옵션에 따라 담을 수 있는 수량이 다릅니다.

---

## 이 화면은 관리자에서 이렇게 만들어집니다

장바구니는 고정 화면이라 레이아웃을 편집하는 메뉴가 없습니다. 대신 **화면에 표시되는 값**들이 아래 관리자 기능에서 결정됩니다.

| 고객 화면의 이 부분 | 관리자 어느 기능으로 만드나 | 비고 |
|---|---|---|
| 상품명 · 이미지 · 판매가 | 상품 관리 | 장바구니는 상품 원본을 그대로 따라갑니다 |
| 담을 수 있는 최대 수량 | 상품 관리 (옵션 · SKU 재고) | 옵션별 재고가 기준입니다 |
| 할인된 단가로 보이는 상품 | 쇼핑특가 관리 | 특가가 걸리면 장바구니 금액도 특가로 바뀝니다 |
| 배송비 금액 | 배송비 정책 | 기본 배송비를 표시합니다 |
| "○○원 더 담으면 무료배송" 게이지 | 배송비 정책의 **무료배송 기준금액** | 기준금액이 없으면 게이지가 사라집니다 |
| 목록에서 사라진 상품 | 상품 관리에서 판매 상태를 내린 경우 | 판매중지 상품은 담기지 않습니다 |

> **장바구니에 표시되는 배송비는 "기본 배송비"입니다.** 제주·도서산간 같은 **지역 할증은 배송지가 정해지는 주문서에서** 더해집니다. 그래서 장바구니 합계와 최종 결제금액이 다를 수 있고, 이는 정상 동작입니다.

## 안 보이거나 비어 있을 때

**"장바구니를 눌렀더니 로그인 화면이 나옵니다"**
정상입니다. 장바구니는 회원 전용입니다.

**"무료배송 게이지가 안 나옵니다"**
배송비 정책에 **무료배송 기준금액**이 설정되어 있지 않습니다. 값을 넣으면 게이지가 생깁니다.

**"배송비가 계속 0원으로 나옵니다"**
배송비 정책의 기본 배송비가 0으로 저장돼 있거나, **다른 몰의 정책**을 고쳤을 수 있습니다. 배송비 정책은 몰마다 따로 저장되므로 편집 중인 몰을 확인하세요.

**"상품을 담으려는데 계속 재고 부족이라고 나옵니다"**
상품 관리에서 해당 **옵션(SKU)의 재고**를 확인하세요. 상품 전체 재고가 남아 있어도 선택한 옵션 재고가 0이면 담기지 않습니다.

**"장바구니 금액과 주문서 금액이 다릅니다"**
① 지역 배송비 할증, ② 주문서에서 적용한 쿠폰·포인트·회원 등급 할인 때문입니다. **최종 금액은 항상 주문서 기준**이며 서버가 다시 계산합니다.
