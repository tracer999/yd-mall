# GA4 이벤트 안내 (사용자 사이트)

쇼핑 과정에서 다음 GA4 이벤트가 자동 전송됩니다. 별도 조작 없이 기본 동작합니다.

## 전송되는 이벤트
- **view_item**: 상품 상세 페이지를 볼 때
- **add_to_cart**: 상품 상세에서 장바구니 버튼을 눌러 추가할 때 (선택한 수량 포함)
- **begin_checkout**: 상품 상세에서 "구매하기"로 결제 단계로 넘어갈 때 (수량/금액 포함)
- **purchase**: 결제를 완료하고 주문 완료 페이지에 도달했을 때 (주문번호와 품목 목록 포함)

## 전송되는 주요 데이터
- 상품 ID/slug, 상품명, 브랜드(공급사), 카테고리명
- 단가(회원가는 회원가), 수량, 총 금액, 통화(KRW)
- 주문 완료 시 주문번호와 구매 품목 리스트

## 주의 사항
- GA4 측정 ID가 관리자 설정에 입력되어 있어야 이벤트가 전송됩니다.
- 광고 차단 확장 프로그램 사용 시 전송이 차단될 수 있습니다.
