# 사업자(B2B) 구매

**사업자 고객이 기업 전용가로 구매하는 흐름**입니다. 별도의 B2B 쇼핑몰이 있는 것이 아니라, **같은 쇼핑몰·같은 상품 화면**에서 로그인한 고객이 승인된 사업자일 때만 몇몇 부분이 바뀝니다.

일반 고객이 보는 화면은 **전혀 달라지지 않습니다.** 사업자 회원이 한 명도 없고 전용가를 넣은 상품이 없으면 이 기능은 화면에 나타나지 않습니다.

## 이 화면에 무엇이 보이나

### 1) 회원가입 · 로그인

- **회원가입** 화면에 `사업자 회원으로 가입하기` 링크가 있습니다. 누르면 일반 가입 항목에 **사업자 정보 입력란**이 붙습니다 — 상호 · 사업자등록번호 · 대표자명 · 업태/종목 · 사업장 주소 · 세금계산서 이메일 · 담당자 · **사업자등록증 사본 첨부**.
- **로그인** 화면에 **`사업자 회원으로 로그인` 체크박스**가 있습니다.
  - 체크하고 로그인 → **기업 구매 모드** (전용가·사업자 주문 절차)
  - 체크 안 함 → **개인 구매 모드** (승인된 사업자라도 일반 판매가로 구매)
  - 사업자 회원이 아닌 계정이 체크하면 **로그인이 거부**되고 안내가 뜹니다.

### 2) 상품 목록 · 상세

승인된 사업자로 로그인했을 때만 바뀝니다.

| 일반 고객 | 승인된 사업자 |
|---|---|
| 판매가 110,000원 | **기업 전용가** — 공급가 70,000원 + VAT 7,000원 (일반가 대비 30%↓) |
| — | 최소 주문수량 10개 |
| `장바구니` `바로구매` | `장바구니` **`주문 요청`** |

- 상품 목록·검색 결과 카드에는 **`기업가` 뱃지**와 전용가가 표시됩니다.
- **승인 전(심사 중)** 에는 전용가가 보이지 않고 "승인 후 확인 가능" 안내만 나옵니다.

### 3) 장바구니

- 상단에 **"기업 구매 장바구니입니다"** 안내가 뜹니다.
- 버튼이 `주문 요청하기` 로 바뀌고, **`견적 요청하기`** 버튼이 하나 더 생깁니다.
- **개인 구매 상품과 기업 구매 상품은 한 장바구니에 담기지 않습니다.** 섞으려 하면 사유가 표시됩니다.
- 최소 주문수량에 못 미치면 담기지 않고 이유를 알려 줍니다.

### 4) 주문서 — 결제하지 않습니다

일반 주문은 결제창으로 넘어가지만, **기업 주문은 접수만 됩니다.**

- **쿠폰·포인트 입력란이 사라집니다.** ("기업 전용가가 적용된 주문입니다" 안내)
- **공급가액 / 부가세**가 나눠서 표시됩니다.
- **발주번호(PO) · 납기 희망일 · 세금계산서 발행 여부 · 요청사항** 입력란이 생깁니다.
- 버튼이 **`주문 요청하기`** 이고, 누르면 결제창이 아니라 **주문 접수 완료 화면**으로 갑니다.

접수 완료 화면에는 다음 절차(담당자 확인 → 입금 안내 → 입금 확인 → 출고)와 **입금 계좌·입금 기한**이 안내됩니다.

### 5) 마이페이지

- **현재 구매 자격**(개인 구매 / 기업 구매)이 표시되고, **전환 버튼**이 있습니다.
  - 전환하려면 **장바구니를 비워야** 합니다.
- **사업자 정보** — 신청 상태(심사 대기 / 승인 / 반려 / 정지 / 계약 만료)와 등록 내용
- **견적함** — 진행 중인 견적 목록 (기업 구매 모드일 때만 보입니다)

### 6) 견적 (대량 주문 단가 협의)

장바구니에서 `견적 요청하기` 로 시작합니다.

1. 수량과 **희망 단가**를 적어 요청
2. 담당자가 검토 후 **단가·배송비·납기·유효기간을 제안**
3. 고객은 **수락** 하거나 **재제안**(다시 협의)
4. 수락하면 단가가 확정되고 **[이 견적으로 주문하기]** 로 주문 전환

- 오간 **협상 메시지**와 **변경 이력(v1, v2, v3…)** 이 모두 남습니다.
- **견적서 PDF** 를 언제든 내려받을 수 있습니다.
- **유효기간이 지나면** 자동 만료되어 주문할 수 없습니다.

## 이 화면은 관리자에서 이렇게 만들어집니다

| 고객 화면의 이 부분 | 관리자 어느 기능으로 만드나 | 비고 |
|---|---|---|
| **기업 전용가** 금액 | 상품 등록/수정 화면의 `B2B(사업자) 판매` 섹션 — **할인율(%)** | 판매가가 바뀌면 전용가도 따라 움직입니다 |
| **최소 주문수량** | 같은 섹션 | 1이면 1개부터 |
| **공급가·부가세 계산**(과세/면세) | B2B 관리 > 기업회원 승인의 `과세 구분` | **거래처마다** 정합니다 |
| 거래처마다 **다른 할인** | B2B 관리 > 거래처 할인 | 상품 할인율에 **더해집니다** |
| 수량별 단가·납기 등 **세부 조건** | B2B 관리 > 견적 관리 | 미리 설정하지 않고 그때그때 협의합니다 |
| 사업자 **승인 여부** | B2B 관리 > 기업회원 승인 | 승인 전에는 전용가가 안 보입니다 |
| **계약 기간** | 같은 화면의 거래 조건 | 기간이 지나면 자동으로 일반가로 돌아갑니다 |
| **공급가 + VAT** 로 나눠 보이는 것 | B2B 관리 > B2B 설정의 `B2B 가격 표기` | |
| **쿠폰·포인트가 사라지는 것** | 같은 설정의 `쿠폰·포인트 사용 허용` | 기본은 사용 안 함 |
| **입금 계좌 안내 · 입금 기한** | 같은 설정 | 접수 화면과 안내 메일에 그대로 나옵니다 |
| **주문이 승인·확정되는 것** | B2B 관리 > B2B 주문 | 담당자가 승인해야 입금 안내가 갑니다 |
| **견적 제안 단가·유효기간** | B2B 관리 > 견적 관리 | |
| 승인·반려·주문 단계별 **안내 메일** | 시스템 설정 > SMTP 메일 설정 | 발송을 끄면 메일이 나가지 않습니다 |

## 고객이 자주 묻는 것

| 고객 질문 | 답 · 확인할 곳 |
|---|---|
| "로그인했는데 전용가가 안 보여요" | ① 로그인할 때 **`사업자 회원으로 로그인`을 체크**했는지 ② 아직 **승인 대기**인지 ③ 그 상품에 전용가가 **설정되어 있는지** |
| "승인은 언제 되나요" | 담당자가 사업자등록증을 확인한 뒤 승인합니다. 결과는 **이메일로 발송**됩니다 |
| "결제가 안 돼요" | 기업 주문은 **즉시 결제가 아닙니다.** 접수 → 승인 → 입금 안내 순서입니다 |
| "장바구니에 같이 안 담겨요" | 개인 구매 상품과 기업 구매 상품은 **섞을 수 없습니다.** 장바구니를 비우고 다시 담아야 합니다 |
| "개인으로도 사고 싶어요" | 마이페이지에서 **개인 구매로 전환**할 수 있습니다(장바구니를 비운 뒤) |
| "세금계산서는 어떻게 받나요" | 주문서에서 **세금계산서 발행**을 체크하면 담당자가 발행합니다. 수신 주소는 사업자 정보에 등록된 이메일입니다 |
| "계산서가 아니라 세금계산서로 받고 싶어요" | 담당자가 **기업회원 승인 > 과세 구분**을 바꿔야 합니다. 거래처 단위로 정해집니다 |
