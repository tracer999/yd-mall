# 홈(메인 페이지)

쇼핑몰 주소로 접속했을 때 **가장 먼저 보이는 페이지**입니다. 홈은 **여러 개의 구역(섹션)이 위에서 아래로 쌓인 화면**이며, 어떤 구역을 몇 번째에 놓을지는 쇼핑몰마다 다릅니다. 그래서 아래 내용 중 **일부는 보이지 않을 수도 있습니다.**

---

## 홈에서 볼 수 있는 구역들

### 메인 배너

- 화면 **맨 위의 큰 이미지**가 슬라이드로 여러 장 넘어갑니다. 좌우 화살표나 아래쪽 점을 눌러 직접 넘길 수 있습니다.
- 배너를 클릭하면 **연결된 페이지**(기획전, 이벤트, 특정 상품 등)로 이동합니다.
- 접속 직후 **팝업 창**이 뜰 수 있습니다. **"오늘 하루 보지 않기"** 를 체크하고 닫으면 그날은 다시 보이지 않습니다.

### 특장점 안내

- 무료배송·정품보장 같은 **쇼핑몰의 강점**을 아이콘과 함께 한 줄로 보여 줍니다.

### 베스트 랭킹

- **판매량과 좋아요를 합산해 매긴 인기 상품 순위**입니다.
- 순위를 **탭**(전체·카테고리별 등)으로 나눠 보여 주기도 합니다. 탭을 누르면 그 기준의 순위로 바뀝니다.
- 더 자세한 순위는 상단 메뉴의 **베스트**에서 볼 수 있습니다. ([상품](products.md))

### MD 추천 상품 / 상품 목록

- 담당자가 고른 상품이 **가로로 넘기는 슬라이드**나 **바둑판 목록**으로 보입니다.
- **더보기** 링크를 누르면 해당 상품을 모아 놓은 목록 페이지로 이동합니다.

### 쇼핑특가 캐러셀

- **기간·시간이 정해진 한정 할인 상품**만 모아 보여 줍니다.
- 특가 기간이 끝나면 이 구역은 **화면에서 사라집니다.**

### 퀵메뉴 / 혜택 모음

- **퀵메뉴:** 자주 찾는 메뉴로 바로 가는 **아이콘 버튼 줄**입니다. 눌러서 해당 페이지로 이동합니다.
- **혜택 모음:** 특가 상품과 이벤트 배너를 크고 작은 타일로 묶어 보여 줍니다.

### 프로모션 배너 · 브랜드 · 카테고리 바로가기

- **프로모션 배너:** 이벤트·기획전 배너가 여러 장 나열됩니다.
- **브랜드 캐러셀:** 브랜드 로고가 가로로 넘어갑니다. 로고를 클릭하면 **브랜드 상세관**으로 이동합니다. ([브랜드관](brands.md))
- **카테고리 바로가기:** 카테고리별 대표 상품을 묶어 보여 줍니다.

### 최근 본 상품

- 내가 **최근에 본 상품**이 표시됩니다. 본 상품이 없으면 이 구역은 나오지 않습니다.
- 로그인하지 않아도 기록되며, **최대 10개**까지 보관됩니다.

### 카카오 문의 버튼

- 화면 **오른쪽 아래에 노란 카카오 버튼**이 떠 있습니다. 누르면 카카오톡 상담 채널이 새 탭으로 열립니다.
- 홈 중간에 **"카카오채널로 문의하기"** 안내 구역이 따로 있을 수도 있습니다.

## 상단 메뉴(GNB)로 이동하기

화면 위쪽 메뉴에서 아래 페이지로 갈 수 있습니다. **쇼핑몰 설정에 따라 일부 메뉴는 없을 수 있습니다.**

| 메뉴 | 내용 |
|------|------|
| **카테고리** | 왼쪽 끝의 **≡ 카테고리** 버튼. 누르면 전체 분류가 펼쳐집니다. |
| **쇼핑특가** | 시간·기간 한정 할인 상품 |
| **베스트** | 인기 상품 순위 |
| **신상품** | 최근 판매를 시작한 상품 |
| **추천** | 나에게 맞는 상품·MD 추천 |
| **기획전** | 주제별로 상품을 모아 놓은 전시 |
| **이벤트&혜택** | 응모 이벤트, 상시 혜택, 멤버십 안내 |
| **브랜드** | 브랜드 검색·브랜드 상세관 |
| **전문관** | 상시 운영되는 테마별 전문 매장 |
| **공동구매** | 기간 한정 전용가 상품 |
| **아울렛** | 이월·임박·전시 상품 등을 싸게 |
| **쇼핑라이브** | 방송을 보며 구매 |
| **쿠폰** | 쿠폰을 받는 곳 |

- 메뉴 옆에 **NEW·HOT·SALE** 같은 작은 표시가 붙을 수 있고, 지금 보고 있는 메뉴는 밑줄로 강조됩니다.
- 메뉴를 눌렀을 때 **"준비 중"** 안내 화면이 나오면, 그 메뉴에 **지금 보여 줄 내용이 하나도 없다는 뜻**입니다. 안내 화면의 버튼으로 다른 메뉴로 이동할 수 있습니다.

### 검색

- 헤더 가운데의 **검색창**에 검색어를 넣고 돋보기 버튼을 누르면 검색 결과가 나옵니다. ([검색](search.md))

### 장바구니 표시

- 헤더의 **장바구니 아이콘**에 담긴 수량이 **빨간 동그라미 숫자**로 표시됩니다.

---

## PC에서 보이는 "바로접속" 레일

- **화면이 넓은 PC**에서는 오른쪽 가장자리에 세로로 된 **바로접속 레일**이 나타납니다.
- 항목: **로그인/마이쇼핑 · 장바구니 · 찜 · 찜한 브랜드 · 최근 본 상품 · (설정 시) 멤버십·앱 다운로드 · 맨 위로(TOP)**
- 맨 위의 **바로접속 ON/OFF** 를 누르면 레일이 **접혔다 펼쳐집니다.** 접어 둔 상태는 다음에도 그대로 유지됩니다.
- **최근 본 상품**을 누르면 옆으로 목록이 펼쳐집니다.
- 레일이 보이지 않는 좁은 화면에서는, 스크롤을 조금 내리면 오른쪽 아래에 **↑ TOP 버튼**이 나타납니다.

---

## 휴대폰에서 보이는 화면 아래 고정 바

- 화면 **맨 아래에 4개 버튼**이 항상 붙어 있습니다: **홈 · 카테고리 · 장바구니 · 마이**
- **장바구니**에는 담긴 수량이 숫자로 표시됩니다.
- **마이** 는 로그인 전에는 **"로그인"** 으로 보입니다.
- **카테고리**를 누르면 **화면 전체를 덮는 카테고리 화면**이 열립니다.
  1. 큰 분류를 고르면 그 아래 분류가 나옵니다. (최대 3단계)
  2. 각 단계마다 **"○○ 전체보기"** 를 누르면 그 분류의 상품 목록으로 갑니다.
  3. 왼쪽 위 **← 뒤로**, 오른쪽 위 **X** 로 닫습니다.

---

## 로그인과의 관계

- 홈은 **로그인하지 않아도** 볼 수 있습니다.
- 장바구니 담기·찜·주문·쿠폰 받기·이벤트 참여 등은 **로그인**이 필요합니다.
