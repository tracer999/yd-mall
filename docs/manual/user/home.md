# 홈(메인 페이지)

쇼핑몰 주소로 접속했을 때 **가장 먼저 보이는 페이지**입니다. 배너, 카테고리별 상품, 신상품, 인기 상품 등을 한눈에 볼 수 있습니다.

---

## 화면에 무엇이 나오나요?

### 메인 배너

- **상단의 넓은 이미지**가 슬라이드처럼 여러 장 보일 수 있습니다.
- 배너를 클릭하면 **연결된 주소**(이벤트 페이지, 특정 상품 등)로 이동합니다.
- 관리자가 **배너 관리**에서 등록·수정한 내용이 그대로 노출됩니다.

### 카테고리별 상품

- **카테고리(분류)** 단위로 구역이 나뉘어 있고, 각 구역에 **해당 카테고리 상품**이 일정 개수만큼 보입니다.
- 예: "건강식품" 구역, "선물세트" 구역 등.
- **해당 카테고리 더 보기** 또는 카테고리 이름을 클릭하면 그 카테고리의 **상품 목록 페이지**로 이동합니다.

### 신상품

- **최근 등록된 상품**이 일정 개수만큼 표시됩니다.
- 상품 이미지·이름·가격을 보고 **클릭하면 상품 상세**로 갑니다.

### 베스트 / 인기 상품

- **조회수가 많거나 인기 있는 상품**이 일정 개수만큼 표시됩니다.
- 마찬가지로 클릭하면 **상품 상세**로 이동합니다.

---

## 여기서 할 수 있는 일

- **배너 클릭** → 연결된 페이지로 이동
- **상품 클릭** → 상품 상세 페이지에서 가격·설명·재고 확인 후 장바구니 담기·구매
- **카테고리 선택** → 해당 카테고리 상품만 모아서 보기
- **전체상품 메뉴** 클릭 → 상품 전체 목록으로 이동

메인 페이지는 **로그인하지 않아도** 볼 수 있습니다. 장바구니 담기·주문·문의 등은 로그인이 필요할 수 있습니다.
