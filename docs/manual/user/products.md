# 상품 목록 / 상품 상세

고객이 **상품을 고르고 자세히 보는** 화면입니다. 쇼핑몰에서 가장 많이 열리는 두 화면이며, 결제가 시작되는 지점도 여기입니다.

**상품 목록**은 카테고리·브랜드·검색·기획전 등 어디서 들어와도 같은 화면을 씁니다. 정렬 탭과 필터가 붙어 있고, 상품 카드가 격자로 나열됩니다.

**상품 상세**는 `/products/{주소이름}` 으로 열립니다. 이미지·가격·옵션·설명·리뷰·추천 상품이 한 화면에 들어갑니다.

## 이 화면에 무엇이 보이나

**상품 목록**

- 정렬 탭 — 인기상품 · 낮은가격 · 높은가격 · 판매량 · 최근등록 · 상품평
- **빠른 필터 줄** — 전체 · 와이디몰특가 · 기간임박할인 · 온라인판매가능 · **B2B 상품** (바로 아래 항목 참고)
- **필터 패널** — 가격·브랜드·할인율 등으로 좁히기 (바로 아래 항목 참고)
- 상품 카드 — 이미지, 이름, 브랜드, 가격(할인 시 정가에 취소선), 뱃지

**상품 상세**

- 대표 이미지 + 추가 이미지
- 상품명 · 브랜드 · 가격 · 할인율
- 옵션 선택 (옵션 상품인 경우) — **재고가 없는 옵션에는 이름 옆에 `(품절)`이 붙습니다**
- 수량 선택, 장바구니 담기 / 바로 구매, 찜
- **탭 5개** — 상세설명 · 배송·반품 · 원산지·품질 · 리뷰 · 문의하기 (바로 아래 항목 참고)
- 함께 보면 좋은 상품
- 카카오톡 문의 버튼 (설정한 경우)

## 빠른 필터 줄 (상단 버튼)

정렬 탭 바로 아래에 **둥근 버튼 줄**이 있습니다. 자주 쓰는 조건 하나를 한 번에 거는 용도입니다.

| 버튼 | 무엇이 보이나 |
|---|---|
| **전체** | 아무 조건도 걸지 않은 상태 (기본) |
| **와이디몰특가** | `와이디몰특가` 뱃지가 달린 상품 |
| **기간임박할인** | `기간임박할인` 뱃지가 달린 상품 (**뱃지 만료일이 지나면 자동으로 빠집니다**) |
| **온라인판매가능** | 온라인 판매 상품으로 지정된 상품 |
| **B2B 상품** | **사업자에게도 판매하는 상품** — 상품 관리에서 `B2B(사업자) 판매`를 켠 상품만 |

- **이 줄의 버튼은 한 번에 하나만 걸립니다.** 다른 버튼을 누르면 앞의 조건은 자동으로 풀립니다. `전체`를 누르면 모두 해제됩니다.
- 아래 **필터 패널**(가격·브랜드 등)과는 **함께** 걸립니다. 예를 들어 `B2B 상품` + `3만원 이하`를 같이 걸 수 있습니다.
- **B2B 상품 버튼과 카드의 `B2B` 뱃지는 로그인하지 않아도 누구에게나 보입니다.** "사업자 구매가 가능한 상품"이라는 표시일 뿐이고, **기업 전용가 금액은 승인된 사업자에게만** 보입니다. 자세한 내용은 [사업자(B2B) 구매](/manual/user/b2b) 를 보세요.
- 이 필터 줄은 **화면을 내려도 따라오지 않습니다.** 목록 맨 위에 그대로 머뭅니다.

## 상품 목록 필터

빠른 필터 줄 아래에 **필터 패널**이 붙습니다. 카테고리·브랜드·검색 결과 등 상품 목록을 쓰는 화면이면 어디서나 같은 방식으로 동작합니다.

### 기본으로 붙는 필터

| 필터 | 고르는 방식 | 동작 |
|---|---|---|
| **가격** | `3만원 이하` · `3~5만원` · `5~10만원` · `10~20만원` · `20만원 이상` 중 **여러 개** | 고른 구간을 **합쳐서** 보여 줍니다(3만원 이하 + 20만원 이상 = 둘 다) |
| **가격 직접 입력** | 최소·최대 금액을 직접 타이핑 | 구간 버튼보다 **직접 입력이 우선**합니다. 한쪽만 넣어도 됩니다 |
| **브랜드** | **지금 조건에서 상품이 있는 브랜드 상위 30개** + 각 건수 | 여러 개 고르면 합집합입니다 |
| **할인율** | `10% 이상` · `20% 이상` · `30% 이상` · `50% 이상` | 여러 개 고르면 **가장 낮은 기준**이 적용됩니다(10%↑와 30%↑ = 10%↑) |
| **상품 태그** | 상품에 달린 뱃지 | |
| **재고** | 켜기/끄기 | 켜면 품절 상품이 빠집니다 |
| **혜택** | `특가` · `아웃렛` | 진행 중인 쇼핑특가 상품, 아웃렛 노출 상품 |

**필터끼리는 AND 입니다.** 가격과 브랜드를 함께 고르면 둘 다 만족하는 상품만 남습니다. 같은 필터 안에서 값을 여러 개 고르면 그것들은 OR(합집합)입니다.

### 화면에서 알아 두면 좋은 것

- 고른 조건은 패널 맨 위에 **칩**으로 모여 표시됩니다. 칩의 `×` 를 누르면 그 조건만 풀리고, **전체 해제**로 한 번에 지웁니다.
- 값이 많은 필터는 **`+N개`** 로 접혀 있습니다. 눌러 펼칩니다.
- **브랜드 목록은 고정이 아닙니다.** 지금 걸린 다른 조건에서 실제로 상품이 있는 브랜드만, 건수와 함께 나옵니다. 그래서 조건을 바꾸면 브랜드 목록도 같이 바뀝니다.
- **브랜드관(`/brands/…`)에서는 브랜드 필터가 나오지 않습니다.** 이미 그 브랜드로 좁혀진 화면이라 의미가 없기 때문입니다.
- 필터를 걸면 **주소(URL)에 그대로 남습니다.** 그 주소를 복사해 공유하면 상대도 같은 필터 결과를 봅니다.
- **배송(무료배송·오늘출발) 필터는 아직 나오지 않습니다.** 상품별 배송 데이터가 쌓이면 자동으로 나타납니다.

### 카테고리마다 필터가 다릅니다

위 표의 필터는 **모든 카테고리에 공통**으로 붙습니다. 그 밖에 색상·사이즈·소재 같은 **카테고리별 속성 필터**는 상품에 그 속성 값이 실제로 쌓인 만큼만 나타납니다.

> **지금은 속성 필터가 화면에 보이지 않는 것이 정상입니다.** 상품 속성 값이 아직 비어 있어 시스템이 자동으로 숨기고 있습니다. 값이 채워지면 코드 수정 없이 나타납니다.

> **카테고리별로 어떤 필터를 붙일지 고르는 관리자 화면은 아직 없습니다.** 현재는 공통 필터가 자동으로 붙는 단계까지 만들어져 있습니다. 카테고리별 지정과 상품 속성 입력 화면은 다음 단계에서 추가됩니다.

## 상품 상세의 탭 5개

상세 화면 아래쪽은 **탭 5개**로 나뉩니다. 어느 탭에 무엇이 들어가고, 그 내용을 **어디서 채우는지**가 다릅니다.

| 탭 | 무엇이 들어가나 | 내용을 어디서 채우나 |
|---|---|---|
| **상세설명** | 상품 상세 설명(이미지·본문), 관련 영상, AI 추천 포인트 | 상품 관리 |
| **배송·반품** | 배송 방법·배송비·배송 기간, 교환/반품 신청 기간·비용 부담·불가 사유 | **배송비는 배송정책 관리에서** / 나머지는 표준 문구 (고정) |
| **원산지·품질** | 상품명·상품코드·브랜드·제조사·원산지·품질보증기준·A/S 연락처 | 상품 관리 + 사이트 설정 |
| **리뷰** | 구매 후기 목록, 평균 별점, 후기 작성 버튼 | 고객이 작성 / 관리자 리뷰 관리에서 숨김·삭제 |
| **문의하기** | 1:1 문의 작성 버튼 | 고객이 작성 |

**알아 두면 좋은 것**

- **배송비는 손님에게 안내되는 금액과 결제 시 청구되는 금액이 항상 같습니다.** 두 곳이 같은 배송정책을 읽기 때문에, 배송정책 관리에서 금액을 바꾸면 이 탭도 함께 바뀝니다.
- 제주·도서산간 추가 배송비는 **무료배송 조건을 넘겨도 부과**되며, 그 사실이 탭에 함께 안내됩니다.
- **리뷰 탭에는 숨김 처리한 후기가 나오지 않습니다.** 리뷰 관리에서 숨기면 즉시 사라지고, 탭 이름의 건수도 함께 줄어듭니다.
- **후기 작성 버튼은 마이페이지로 이동합니다.** 후기는 **구매한 주문에 붙기 때문에**, 상품 화면에서 바로 쓰지 않고 작성 가능한 주문을 골라서 씁니다.
- **문의하기 버튼을 누르면 문의 제목이 `[상품문의] 상품명` 으로 미리 채워집니다.** 손님이 상품명을 다시 적지 않아도 됩니다.
- **원산지·품질 탭에서 값이 없는 항목은 `상세설명 참조` 로 표시됩니다.** 빈칸으로 두지 않기 위한 처리입니다.

> **상품별 원산지를 직접 입력하는 칸은 아직 없습니다.** 그래서 원산지는 현재 모든 상품에서 `상세설명 참조` 로 나갑니다. 브랜드·제조사·상품코드는 상품 관리에 입력한 값이 그대로 나옵니다.

## 안 보이거나 비어 있을 때

**Q. 등록한 상품이 목록에 없습니다.**
① 판매 상태가 **판매중**인가, ② 노출 설정이 **전체 공개**인가(회원 전용이면 비로그인 고객에게 안 보입니다), ③ **편집 중인 몰**이 맞는가를 확인하세요.

**Q. 상품 상세가 404 로 뜹니다.**
노출 설정이 **숨김**이면 주소로 직접 들어와도 열리지 않습니다.

**Q. 할인 전 가격(취소선)이 안 보입니다.**
상품 관리에서 **정가**를 입력하지 않았습니다. 판매가만 있으면 취소선이 생기지 않습니다.

**Q. 배너에 적은 가격과 실제 결제 금액이 다릅니다.**
그 상품에 쇼핑특가가 걸려 있습니다. 특가 관리에서 진행 중인 특가를 확인하세요.

**Q. 함께 보면 좋은 상품이 비어 있습니다.**
수동으로 등록한 추천 상품만 노출됩니다. 자동 추천은 하지 않습니다.

**Q. 옵션 선택 상자가 안 나옵니다.**
그 상품이 옵션 상품으로 등록되어 있지 않습니다.

**Q. 옵션 이름 옆에 `(품절)`이 붙었습니다.**
그 옵션의 **재고가 0이거나 옵션 상태가 `중지`** 입니다(상품 전체의 판매 상태와는 다른 값입니다). 옵션이 두 종류 이상이면(예: 색상+사이즈) **이미 고른 옵션에 맞춰 다시 계산**되므로, 색상을 바꾸면 품절 표시가 달라질 수 있습니다. 재고와 옵션 상태는 <span class="manual-path">상품 관리 &gt; 옵션·SKU 관리</span> 에서 확인·수정합니다.

**Q. `B2B 상품` 필터를 눌렀는데 아무것도 안 나옵니다.**
그 몰에 **`B2B(사업자) 판매`를 켠 상품이 하나도 없습니다.** 상품 관리에서 상품을 수정해 `B2B(사업자) 판매` 를 체크하세요.

**Q. `B2B` 뱃지가 일반 고객에게도 보입니다.**
정상입니다. "사업자 구매가 가능한 상품"이라는 표시이며, **기업 전용가 금액은 승인된 사업자에게만** 보입니다.
