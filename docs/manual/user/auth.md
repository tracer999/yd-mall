# 로그인 / 회원가입

**쇼핑몰에 로그인**하거나 **새로 회원가입**하는 방법입니다. 로그인하면 장바구니·주문·1:1 문의·마이페이지 등을 이용할 수 있습니다.

---

## 로그인하기

1. **상단의 "로그인"** 링크를 클릭합니다.
2. **이메일로 가입**한 회원은 **이메일과 비밀번호**를 입력하고 **로그인** 버튼을 누릅니다.
3. **간편 로그인**을 쓰려면 아래의 **카카오 / 네이버 / 구글** 버튼을 누릅니다.
   - 이미 해당 계정으로 가입한 적이 있으면, 로그인만 하고 **쇼핑몰 메인** 또는 **이전에 보던 페이지**로 이동합니다.
   - 쇼핑몰 설정에 등록된 소셜 서비스만 버튼으로 표시됩니다.
4. 로그인에 성공하면 상단에 **"환영합니다, ○○○님"** 과 **마이페이지·로그아웃** 링크가 보입니다.

상품이나 쿠폰 화면에서 로그인 화면으로 넘어왔다면, 로그인 후 **원래 보던 화면으로 되돌아갑니다.**

---

## 회원가입하기

가입 방법은 두 가지이고, 어느 쪽이든 **주문·배송에 필요한 정보**를 한 번 입력받습니다.

### 방법 1. 간편 가입 (카카오·네이버·구글)

1. **상단의 "회원가입"** 을 클릭합니다.
2. 맨 위의 **카카오 / 네이버 / 구글** 버튼 중 하나를 누릅니다.
3. 해당 소셜 로그인 화면에서 계정을 선택하고 **동의·로그인**을 진행합니다.
4. 쇼핑몰로 돌아오면 **추가 정보 입력** 화면이 나옵니다. (이름·이메일은 가져오고, 나머지를 입력)
5. **이용약관·개인정보 처리방침**에 동의한 뒤 **가입 완료** 버튼을 누릅니다.

### 방법 2. 이메일로 가입

1. **회원가입** 화면 아래쪽의 **이메일 가입폼**에 **비밀번호**를 입력합니다.
   - 비밀번호는 **8자 이상**이어야 하고 **영문과 숫자를 모두 포함**해야 합니다.
   - **비밀번호 확인**란에 같은 비밀번호를 한 번 더 입력합니다.
2. 이어지는 **기본 정보 · 연락처 · 기본 배송지**를 입력합니다.
3. 약관에 동의하고 **가입하기** 를 누르면 가입과 동시에 로그인됩니다.

### 가입 축하 쿠폰

- 가입이 완료되면 **신규 가입 쿠폰**이 자동으로 지급될 수 있습니다.
- 받은 쿠폰은 **마이페이지 > 쿠폰함**에서 확인하세요. ([마이페이지](mypage.md))

---

## 추가 정보 입력 / 약관 동의

간편 가입으로 처음 로그인하면 **"추가 정보 입력"** 화면에서 아래를 입력합니다. (이메일 가입폼과 동일한 항목입니다.)

| 구분 | 입력 항목 |
|------|-----------|
| 기본 정보 | 성함, 이메일, 생년월일, 성별(선택) |
| 연락처 | 휴대폰 번호, 보조 연락처(선택) |
| 기본 배송지 | 수령인명, 우편번호·기본주소·상세주소, 배송 요청사항(선택) |
| 동의 | 이용약관·개인정보 처리방침(필수), 마케팅 수신(선택) |

- **기본 배송지**로 입력한 정보는 주문할 때 자동으로 채워지며, 주문 화면에서 변경할 수 있습니다.
- 이미 가입된 **휴대폰 번호·이메일**은 사용할 수 없습니다(입력 후 다른 곳을 클릭하면 바로 확인해 줍니다).
- **휴대폰 번호를 입력하기 전까지는** 로그인할 때마다 이 화면이 다시 나옵니다. 먼저 입력을 끝내 주세요.

## 약관 재동의

- **약관이 개정**되면, 기존 회원이 다시 로그인할 때 **"약관 재동의"** 화면이 나옵니다.
- 새 약관과 개인정보 처리방침을 확인하고 **동의**하면 계속 이용할 수 있습니다.

---

## 가입한 뒤 정보를 고치려면

- **마이페이지 > 회원정보 수정**에서 고칩니다.
- **휴대폰 번호·주소**처럼 중요한 정보를 바꿀 때는 **카카오 계정으로 본인 확인**을 한 번 거칩니다. ([마이페이지](mypage.md))

---

## 로그아웃하기

- **상단의 "로그아웃"** 링크를 클릭하면 로그아웃됩니다.
- 로그아웃 후에는 **로그인·회원가입** 링크만 보이고, 장바구니·주문·문의 등 회원 전용 메뉴는 로그인해야 다시 이용할 수 있습니다.

---

## 비회원 주문

- **비회원 주문**을 허용한 쇼핑몰은 **주문/결제** 단계에서 "비회원으로 주문하기"를 선택할 수 있습니다.
- 이 경우 **주문자 이름·연락처·이메일·배송지** 등을 직접 입력하고 결제합니다. 회원가입은 하지 않아도 됩니다.
