# 로그인 / 회원가입

**쇼핑몰에 로그인**하거나 **새로 회원가입**하는 방법입니다. 로그인하면 장바구니·주문·1:1 문의·마이페이지 등을 이용할 수 있습니다.

---

## 로그인하기

1. **상단의 "로그인"** 링크를 클릭합니다.
2. **로그인 페이지**가 나오면, **Google로 로그인** 또는 **Kakao로 로그인** 버튼이 있는 경우 해당 버튼을 누르면 됩니다.
   - 이미 해당 계정으로 가입한 적이 있으면, 로그인만 하고 **쇼핑몰 메인** 또는 **이전에 보던 페이지**로 이동합니다.
3. **일반 로그인**(아이디·비밀번호)이 있다면, 아이디와 비밀번호를 입력한 뒤 **로그인** 버튼을 누릅니다.
4. 로그인에 성공하면 상단에 **"환영합니다, ○○○님"** 과 **마이페이지·로그아웃** 링크가 보입니다.

**비밀번호를 잊었다면** "비밀번호 찾기" 링크가 있으면 이용하고, 없으면 쇼핑몰 고객센터나 1:1 문의로 문의하세요.

---

## 회원가입하기 (Google·Kakao)

1. **상단의 "회원가입"** 을 클릭합니다.
2. **Google로 회원가입** 또는 **Kakao로 회원가입** 버튼을 누릅니다.
3. **Google 또는 Kakao 로그인 화면**이 뜨면, 사용할 계정을 선택하고 **동의·로그인**을 진행합니다.
4. 쇼핑몰로 돌아오면 **추가 정보 입력** 화면이 나올 수 있습니다. (이름·이메일은 이미 가져오고, **전화번호·생년월일·주소** 등을 추가로 입력)
5. **이용약관·개인정보 처리방침**에 동의한 뒤 **가입 완료** 버튼을 누릅니다.
6. **가입 완료** 페이지가 나오면 회원가입이 끝납니다. 이제 로그인된 상태로 쇼핑몰을 이용할 수 있습니다.

---

## 추가 정보 입력 / 약관 동의

- **처음 소셜 로그인(Google·Kakao)으로 가입**할 때, **전화번호** 등 필수 정보가 없으면 **"추가 정보 입력"** 화면으로 이동합니다.
- 여기서 **전화번호, 생년월일, 주소** 등을 입력하고, **이용약관·개인정보 처리방침**에 동의해야 서비스를 이용할 수 있습니다.
- **약관이 개정**된 경우, 기존 회원이 다시 로그인하면 **"약관 재동의"** 화면이 나올 수 있습니다. 새 약관에 동의하면 계속 이용할 수 있습니다.

---

## 로그아웃하기

- **상단의 "로그아웃"** 링크를 클릭하면 로그아웃됩니다.
- 로그아웃 후에는 **로그인·회원가입** 링크만 보이고, 장바구니·주문·문의 등 회원 전용 메뉴는 로그인해야 다시 이용할 수 있습니다.

---

## 비회원 주문

- **비회원 주문**을 허용한 쇼핑몰은 **주문/결제** 단계에서 "비회원으로 주문하기"를 선택할 수 있습니다.
- 이 경우 **주문자 이름·연락처·이메일·배송지** 등을 직접 입력하고 결제합니다. 회원가입은 하지 않아도 됩니다.
