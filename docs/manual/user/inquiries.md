# 1:1 문의

**쇼핑몰에 질문이나 불만·제안을 보내는** 기능입니다. **로그인한 회원만** 문의를 작성하고, **내가 쓴 문의 목록**을 볼 수 있습니다.

---

## 문의 목록 보기 (내 문의)

1. **메뉴의 "1:1 문의"** 를 클릭합니다.
2. **로그인하지 않았다면** 먼저 **로그인** 화면으로 이동합니다.
3. 로그인 후 **1:1 문의** 메뉴에 들어가면 **내가 작성한 문의 목록**이 나옵니다.
4. **제목, 작성일, 답변 여부**(답변 완료 / 대기 중) 등이 표시됩니다.
5. **제목을 클릭**하면 **해당 문의의 상세 내용과 관리자 답변**을 볼 수 있습니다.

---

## 문의 작성하기

1. **1:1 문의** 메뉴에서 **문의 작성** 또는 **글쓰기** 버튼을 클릭합니다.
2. **제목**을 입력합니다. (예: "배송 문의", "상품 교환 요청")
3. **내용**을 입력합니다. 문의할 내용을 구체적으로 적어 주시면 답변을 받기 쉽습니다. (연락처, 주문 번호 등이 필요하면 함께 적어 주세요.)
4. **등록** 또는 **보내기** 버튼을 누릅니다.
5. 등록이 완료되면 **문의 목록**으로 돌아가거나 **완료 메시지**가 나옵니다.

---

## 답변 확인

- 관리자가 **답변을 등록**하면, 해당 문의를 열었을 때 **답변 내용**이 보입니다.
- **이메일·문자**로 답변 안내가 오는 쇼핑몰도 있습니다. 1:1 문의 목록에서 **"답변 완료"** 로 바뀌었는지 확인한 뒤, 문의를 클릭해 답변을 읽으면 됩니다.

추가로 궁금한 점이 있으면 **같은 문의에 이어서** 다시 문의를 작성할 수 있는지, 아니면 **새 문의**로 작성해야 하는지는 쇼핑몰마다 다를 수 있습니다.
