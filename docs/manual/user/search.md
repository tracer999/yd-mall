# 상품 검색

쇼핑몰에서 **원하는 상품을 키워드로 찾는** 기능입니다. 상단 메뉴의 **검색 아이콘(돋보기)** 을 클릭하면 검색 페이지로 이동합니다.

---

## 검색하는 방법

1. **검색 아이콘**을 클릭해 **검색 페이지**로 이동합니다.
2. **검색창**에 찾고 싶은 **단어**를 입력합니다. (상품명, 브랜드명, 재료명 등)
3. **검색** 버튼을 누르거나 **Enter** 키를 누릅니다.
4. **검색 결과 목록**이 나옵니다. 일치하는 상품이 없으면 "검색 결과가 없습니다" 등으로 표시됩니다.

---

## 검색 결과 화면

- **입력한 키워드**와 **결과 개수**가 표시됩니다.
- **상품 이미지, 이름, 가격** 등이 리스트로 나열됩니다.
- **상품을 클릭**하면 **상품 상세 페이지**로 이동해 자세히 보고 장바구니에 담거나 구매할 수 있습니다.

---

## 검색 시 유의사항

- **두 글자 이상** 입력해야 검색이 되는 경우가 많습니다.
- 띄어쓰기나 맞춤법이 다르면 결과가 달라질 수 있으므로, **다양한 키워드**로 한 번씩 검색해 보는 것도 좋습니다.
- 결과가 0건이면 비슷한 단어로 다시 검색하거나, **카테고리**에서 둘러보세요.

검색한 기록은 관리자 쪽 **검색 로그**에 남을 수 있어, 쇼핑몰 운영자가 인기 검색어·결과 없음 키워드를 파악하는 데 쓰입니다.
