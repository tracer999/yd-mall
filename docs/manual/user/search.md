# 검색

고객이 **키워드로 상품을 찾는** 화면입니다. 헤더의 돋보기 아이콘을 누르면 검색 화면(`/search`)이 열립니다.

검색은 **관리자에서 만들어 두는 화면이 아닙니다.** 상품 관리에 등록된 내용을 그대로 훑기 때문에, 검색이 잘 되게 하려면 **검색 화면이 아니라 상품 데이터를 손봐야** 합니다.

대신 고객이 무엇을 검색했는지는 **검색 로그**에 쌓이며, 여기가 상품 이름·설명을 다듬을 근거가 됩니다.

## 이 화면에 무엇이 보이나

- 검색어 입력창
- 검색 결과 상품 카드 (최대 50개)
- 결과가 없을 때의 안내

검색어는 **두 글자 이상**부터 동작합니다. 결과는 판매 상태 순(판매중 → 재입고 → 입고예정 → 품절)으로 먼저 정렬되고, 그 안에서 최근 등록순으로 나열됩니다.

---

## 이 화면은 관리자에서 이렇게 만들어집니다

### 검색이 훑는 항목

고객이 입력한 단어는 상품의 **여섯 항목**과 대조됩니다. 모두 상품 관리에서 입력하는 값입니다.

| 검색에 잡히는 항목 | 관리자 어느 기능으로 만드나 |
|---|---|
| 상품명 | 상품 관리 |
| 주소용 이름(slug) | 상품 관리 |
| 공급사·제조사 이름 | 상품 관리 |
| 브랜드 이름 | 카테고리 관리(브랜드) — 상품에 지정 |
| 상세 설명 | 상품 관리 |
| AI 추천 문구 | 상품 관리 |

### 화면 · 결과 관련

| 고객 화면의 이 부분 | 관리자 어느 기능으로 만드나 | 비고 |
|---|---|---|
| 검색 결과에 나오는 상품 | 상품 관리 | 판매중·재입고·입고예정·품절만 노출. 판매중지는 제외 |
| 회원 전용 상품 | 상품 관리 (노출 설정) | 로그인한 고객의 검색 결과에만 포함 |
| 결과 카드에 표시되는 할인가 | 쇼핑특가 관리 | 특가가 걸린 상품은 특가가로 표시 |
| 헤더의 검색 아이콘 | 메뉴 관리 (검색 메뉴) | 끄면 아이콘이 사라짐 |
| 고객이 검색한 키워드 기록 | **검색 로그** (자동 수집) | 검색할 때마다 키워드와 결과 건수가 저장됨 |
| 검색 정렬 순서 · 인기 검색어 | **관리 화면 없음** | 정렬은 고정, 인기 검색어 기능은 미제공 |

> **검색 로그에는 결과 건수도 함께 저장됩니다.** 결과가 **0건인 검색어**가 곧 개선 목록입니다. 고객이 찾는 이름과 상품에 적힌 이름이 다르다는 뜻이므로, 그 표현을 상품명이나 상세 설명에 넣어 주면 다음부터 검색됩니다.

## 안 보이거나 비어 있을 때

**Q. 검색 아이콘이 없습니다.**
메뉴 관리에서 검색 메뉴가 꺼져 있습니다.

**Q. 분명히 있는 상품이 검색되지 않습니다.**
① 검색어가 **두 글자 이상**인지, ② 그 상품의 판매 상태가 **판매중지**는 아닌지, ③ 노출 설정이 **숨김**은 아닌지, ④ **회원 전용** 상품을 비로그인 상태로 찾고 있지 않은지, ⑤ **편집 중인 몰**의 상품이 맞는지 확인하세요.

**Q. 검색어와 상품 이름이 다릅니다.**
검색은 입력한 글자가 그대로 포함된 상품만 찾습니다. 동의어·오타 보정은 하지 않습니다. 고객이 쓰는 표현을 상품명이나 상세 설명에 함께 적어 두세요.

**Q. 검색 결과가 50개에서 끊깁니다.**
검색 결과는 50개까지만 표시합니다. 더 넓게 보려면 카테고리나 브랜드로 찾도록 안내하세요.

**Q. 검색 로그가 비어 있습니다.**
아직 고객 검색이 없었거나, 편집 중인 몰이 다릅니다.

## 아직 지원하지 않는 것

- 인기 검색어 · 추천 검색어 노출
- 동의어 사전 · 오타 보정
- 검색 결과 정렬 변경 (정렬은 고정입니다)
- 검색 결과에 대한 필터
