# 마이페이지

**내 주문·쿠폰·포인트·찜한 상품**을 확인하고, **회원 정보를 고치거나 취소·반품을 신청**하는 곳입니다. 마이페이지는 **로그인해야** 들어갈 수 있습니다.

- **PC:** 화면 위 오른쪽의 **마이쇼핑**
- **휴대폰:** 화면 아래 고정 바의 **마이**

## 마이페이지 첫 화면

들어가면 아래 내용이 한눈에 보입니다.

- **최근 주문 5건**과 주문 상태별 건수
- **보유 쿠폰 수 · 포인트 잔액 · 찜한 상품 수 · 최근 본 상품 수**
- 왼쪽 메뉴: **장바구니 · 주문내역 · 취소/반품 내역 · 쿠폰함 · 포인트 · 1:1 문의 · 회원정보 수정**

## 주문 내역 보기

1. 왼쪽 메뉴에서 **주문내역** 을 누릅니다.
2. **주문일, 상품명(○○ 외 N건), 결제금액, 주문 상태**가 목록으로 나옵니다.
3. 주문을 클릭하면 **주문 상세**로 들어갑니다.

### 주문 상세에서 볼 수 있는 것

- **배송지 정보**와 배송 메모
- **결제 정보:** 상품금액 / 쿠폰할인 / 포인트사용 / **배송비** / 배송비할인 / **총 결제금액**
- **주문한 상품 목록**
- **배송 조회:** 송장이 등록되면 버튼이 나타납니다. (CJ대한통운·우체국·한진·롯데·로젠 택배를 조회할 수 있습니다)
- **리뷰 작성:** 배송이 완료된 상품에 후기를 남길 수 있습니다.

## 주문 취소 · 반품 신청

**주문 상세 화면**에서 신청합니다. **주문이 지금 어떤 상태인지에 따라** 취소가 되기도 하고 반품이 되기도 합니다.

| 주문 상태 | 신청할 수 있는 것 |
|-----------|------------------|
| 결제 전 · 결제 완료 · 상품 준비중 | **주문 취소** |
| 배송중 · 배송완료 | **반품** (배송완료 후 **7일 이내**) |

> **교환은 아직 지원하지 않습니다.** 반품하신 뒤 다시 주문해 주세요.

### 신청 방법

1. **주문 상세**에서 **취소/반품 신청** 버튼을 누릅니다.
2. **사유**를 고릅니다.
   - **취소 사유:** 단순변심 / 주문실수 / 배송지연 / 기타
   - **반품 사유:** 단순변심 / 상품불량 / 오배송 / 기타
   - **기타**를 고르면 자세한 사유를 직접 적어야 합니다.
3. **신청**을 누르면 접수됩니다.

- **결제 전·결제 완료** 상태의 취소는 **바로 처리**되어 환불이 시작됩니다.
- **상품 준비중**의 취소와 **모든 반품**은 **담당자 확인 후 처리**됩니다.
- 배송이 완료된 지 **7일이 지나면 반품 신청을 받을 수 없습니다.**

### 반품 배송비

- **단순 변심**으로 반품하면 **왕복 배송비가 청구**되어, **환불 금액에서 차감**됩니다.
- **상품 불량 · 오배송**은 **배송비를 받지 않습니다.**

### 진행 상태 확인 · 신청 철회

1. 왼쪽 메뉴의 **취소/반품 내역** 에서 진행 상황을 볼 수 있습니다.
2. 상태는 **처리 중 / 승인 / 완료 / 반려 / 철회** 로 표시됩니다.
3. **처리 중** 상태(담당자가 아직 승인하기 전)라면 **철회** 버튼을 눌러 신청을 취소할 수 있습니다. **승인·완료된 건은 철회할 수 없습니다.**
4. 처리가 끝나면 **환불 금액**이 표시되고, 반품 배송비를 뺐다면 그 내용도 함께 나옵니다.

## 쿠폰함

**내가 받은 쿠폰을 보관하는 곳**입니다. 왼쪽 메뉴의 **쿠폰함** 을 누르세요.

> 쿠폰을 **받는 곳**은 **쿠폰존**입니다. 화면 위의 **쿠폰 받으러 가기 →** 를 누르면 이동합니다. ([프로모션](promotions.md))

쿠폰은 **4가지 상태**로 나뉩니다.

| 상태 | 뜻 |
|------|-----|
| **사용가능** | 지금 주문에 쓸 수 있는 쿠폰 |
| **주문진행중** | 다른 주문에서 사용 중인 쿠폰. 그 주문을 끝내지 않으면 **30분 뒤 자동으로 풀립니다.** |
| **사용완료** | 이미 사용한 쿠폰. 어떤 주문에 썼는지 표시됩니다. |
| **기간만료** | 유효기간이 지나 쓸 수 없는 쿠폰 |

- 쿠폰마다 **할인 혜택**(정액·정률·무료배송), **최소 구매 금액**, **유효기간**이 적혀 있습니다.
- **만료까지 3일 이하**로 남으면 유효기간이 **빨간색으로 강조**되고 **"(곧 만료)"** 라고 표시됩니다. 잊지 말고 사용하세요.

## 포인트(적립금)

- 왼쪽 메뉴의 **포인트** 에서 **사용 가능한 포인트**와 **전체 변동 내역**을 볼 수 있습니다.
- 내역에는 **일시 · 내용 · 변동량**이 나오며, **적립은 파란색(+)**, **사용은 빨간색(−)** 으로 표시됩니다.
- 주문과 연결된 내역은 **(주문보기)** 를 눌러 해당 주문으로 갈 수 있습니다.
- 포인트는 주문할 때 사용할 수 있습니다. ([주문/결제](checkout.md))

## 회원정보 수정

왼쪽 메뉴의 **회원정보 수정** 을 누릅니다. 항목에 따라 **본인 확인**이 필요합니다.

### 바로 고칠 수 있는 항목

- **이메일, 성함** → 입력하고 저장하면 끝입니다.

### 본인 확인이 필요한 항목

**휴대폰 번호, 주소(우편번호·기본주소·상세주소)** 는 중요한 정보라 본인 확인을 거칩니다.

1. 화면에는 휴대폰 번호가 **010-****-1234** 처럼 가려져 보입니다.
2. **변경** 버튼을 누르면 **본인 확인 안내**가 뜹니다.
3. **카카오 계정으로 본인 확인**을 진행합니다.
4. 확인이 끝나면 정보를 수정할 수 있습니다. **확인 후 15분이 지나면 다시 인증**해야 합니다.

- 주소는 **우편번호 찾기** 버튼으로 검색해 입력합니다.

## 회원 탈퇴

1. **회원정보 수정** 화면 아래쪽의 **회원 탈퇴** 를 누릅니다.
2. **카카오 계정으로 본인 확인**을 진행합니다. (본인 확인 없이는 탈퇴할 수 없습니다)
3. 탈퇴 **사유**를 고릅니다.
4. **안내 사항에 동의** 를 체크하면 탈퇴 버튼이 활성화됩니다.
5. **탈퇴하기** 를 누르고 다시 한 번 확인하면 탈퇴가 완료됩니다.

> **탈퇴 전에 꼭 확인하세요.**
> - 보유한 **포인트와 쿠폰이 모두 사라집니다.**
> - **주문 이력을 더 이상 조회할 수 없습니다.**
> - 작성한 **리뷰·문의가 삭제**됩니다.
> - 다시 가입해도 **혜택은 복구되지 않으며**, **같은 계정으로는 재가입할 수 없습니다.**
