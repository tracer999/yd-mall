# 브랜드관

**브랜드를 찾아보고, 브랜드별 전용관에서 상품과 혜택을 한번에 보는 곳**입니다. 상단 메뉴의 **브랜드** 를 누르면 열립니다. 브랜드 보기는 **로그인 없이** 가능합니다.

---

## 브랜드 찾기

### 검색으로 찾기

1. 화면 맨 위의 **검색창**에 브랜드 이름을 입력합니다.
2. 입력하는 동안 **후보 브랜드가 바로 아래에 나타납니다.**
3. **한글 이름, 영문 이름, 초성** 어느 쪽으로도 찾을 수 있습니다. (예: `나이키`, `NIKE`, `ㄴㅇㅋ`)
4. 후보에 **"진행 중 혜택 N개"** 가 함께 표시되니 참고하세요.

### 목록에서 찾기

브랜드 홈은 아래 순서로 되어 있습니다.

| 구역 | 내용 |
|------|------|
| **이번 주 브랜드 혜택** | 지금 혜택(쿠폰·기획전·특가·공동구매)이 진행 중인 브랜드를 슬라이드로 |
| **인기 브랜드** | 인기 브랜드와 그 브랜드의 상품을 가로 슬라이드로 |
| **신규 입점 브랜드** | 새로 들어온 브랜드 |
| **카테고리별 브랜드** | 카테고리를 누르면 그 카테고리의 브랜드만 보여 줍니다 |
| **전체 브랜드** | 모든 브랜드를 한 곳에 |

### 전체 브랜드에서 찾기

- **ㄱㄴㄷ · ABC 색인**을 눌러 그 글자로 시작하는 브랜드만 볼 수 있습니다. (브랜드가 없는 글자는 눌러지지 않습니다)
- **정렬:** 상품많은순(기본) / 인기순 / 가나다순 / 신규순
- 아래쪽 번호를 눌러 다음 페이지로 넘어갑니다.

---

## 브랜드 상세관

브랜드를 클릭하면 그 브랜드만의 전용관이 열립니다.

### 맨 위에 보이는 것

- **브랜드 로고**와 이름, **공식 브랜드관** 표시
- **상품 수 · 대표 카테고리 · 가격대 · 입점일**
- **관심 브랜드(찜) 버튼**
- 받을 수 있는 **쿠폰**이 있으면 **빨간 안내 띠**가 나옵니다. **받기** 를 누르면 쿠폰 화면으로 이동합니다. ([프로모션](promotions.md))

### 탭으로 둘러보기

| 탭 | 내용 |
|------|------|
| **홈** | 브랜드 소개, 진행 중인 혜택 요약, 베스트·신상품 미리보기 |
| **베스트** | 이 브랜드에서 가장 잘 팔리는 상품 |
| **신상품** | 이 브랜드의 최근 상품 |
| **전체 상품** | 모든 상품. **카테고리 필터**와 **정렬**(신상품순 / 인기순 / 낮은가격 / 높은가격)을 쓸 수 있습니다 |
| **혜택** | 이 브랜드의 기획전·쿠폰·쇼핑특가·공동구매를 모아서 |

- **상품이 적은 브랜드**는 베스트·신상품 탭이 나오지 않고, 홈에서 바로 전체 상품을 보여 줍니다.
- **진행 중인 혜택이 없으면** 혜택 탭이 나오지 않습니다.
- 화면 맨 아래에 **비슷한 브랜드** 추천이 나옵니다.

---

## 브랜드 찜하기

1. 브랜드 타일이나 브랜드 상세관의 **하트 버튼**을 누릅니다. (**로그인 필요**)
2. 하트가 채워지면 찜한 것입니다. 다시 누르면 해제됩니다.
3. 로그인하지 않고 누르면 **로그인 화면**으로 이동합니다.
4. 찜한 브랜드는 **마이페이지 > 찜한 브랜드**에서 모아 볼 수 있습니다. ([마이페이지](mypage.md))
