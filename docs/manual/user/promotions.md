# 혜택 · 프로모션 메뉴

상단 메뉴줄에서 상품 분류가 아닌 **혜택·전시 성격의 메뉴**를 한데 모아 설명합니다. 쇼핑특가 · 베스트 · 신상품 · 추천 · 기획전 · 전문관 · 이벤트 · 공동구매 · 아울렛 · 쇼핑라이브 · 쿠폰존 · 멤버십이 여기에 해당합니다.

이 메뉴들은 **주소가 고정**되어 있습니다. 운영자는 메뉴 관리에서 켜고 끄기 · 이름 · 순서만 바꿀 수 있고, 주소는 바꿀 수 없습니다. 각 메뉴의 내용은 **저마다 다른 관리 화면**에서 만들어집니다 — 이 문서의 핵심이 아래 매핑표입니다.

> **내용이 0건이면 메뉴를 눌러도 "준비 중" 화면이 나옵니다.** 아울렛 · 공동구매 · 쇼핑라이브는 한발 더 나아가 **메뉴 자체가 GNB 에서 자동으로 사라집니다.** 고객에게 빈 화면을 보여주지 않기 위한 동작이며, 내용을 채우면 다시 나타납니다.

---

## 이 화면에 무엇이 보이나

각 메뉴는 대체로 **모아 보여주는 목록 화면**이고, 실제 결제는 상품 상세에서 이뤄집니다. 쿠폰존만 예외로 화면에서 직접 쿠폰을 받습니다.

### 메뉴별 대응표

| 고객 메뉴 | 주소 | 관리자 어느 기능으로 만드나 | 비고 |
|---|---|---|---|
| **쇼핑특가** | `/deals` | 쇼핑특가 관리 + 특가 분류 | 화면 상단 탭이 곧 특가 분류. 특가가는 실제 결제 금액에 반영 |
| **베스트/랭킹** | `/best` | 베스트/랭킹 관리 | 순위는 판매·좋아요를 합산해 자동 산출. **BEST 10 → 카테고리별 → 브랜드별** 순서 |
| **신상품** | `/new` | 상품 관리 (판매 시작일 · NEW 뱃지) | **카테고리별 → 브랜드별** 신상품. 별도 설정 없이 자동으로 채워집니다 |
| **추천** | `/recommend` | 상품 추천관리 | 최근 본 상품 기반 · MD 픽 · 많이 보는 상품 3개 섹션 |
| **기획전** | `/exhibition` | 기획전·전문관 관리 | 유형이 `전문관`이 아닌 것만 여기에 나옴 |
| **전문관** | `/specialty` | 기획전·전문관 관리 | **같은 관리 화면**에서 유형을 `전문관`으로 만든 것 |
| **이벤트 & 혜택** | `/event` | 이벤트 관리 (+ 멤버십 관리) | 멤버십 등급 안내가 하위 섹션으로 함께 노출 |
| **공동구매** | `/group-buy` | 공동구매 관리 | 1건이라도 있으면 메뉴가 살아납니다 |
| **아울렛** | `/outlet` | 아울렛 관리 | 판매중 상품이 **설정한 최소 개수** 미만이면 메뉴가 숨겨짐 |
| **쇼핑라이브** | `/live` | 쇼핑라이브 관리 | 방송은 직접 송출하지 않고 YouTube·Vimeo 영상을 연결 |

| **멤버십** | `/membership` | 멤버십 관리 | 등급이 없으면 기본 안내로 표시. GNB 기본 노출은 꺼져 있고 이벤트 화면 안에서 연결됨 |

### 메뉴 맨 위의 배너와 추천 캐러셀

위 메뉴 페이지들은 **본문 위에** 두 가지가 더 붙을 수 있습니다. 둘 다 선택이고, 등록한 것만 나옵니다.

| 순서 | 무엇 | 관리자 어디서 |
|---|---|---|
| ① 위 | **이미지 배너** | 배너 관리 — 메뉴 배너 |
| ② 아래 | **추천 상품 캐러셀** | 상품 그룹 관리 — `메뉴 상단 노출` 지정 |

캐러셀 제목은 운영자가 정하며, 기본 예시는 쇼핑특가 = `추천 특가`, 베스트 = `추천 베스트`, 신상품 = `주목할 신상품` 입니다.

**캐러셀에 어떤 상품이 나오는지**는 다음 순서로 정해집니다.

1. 상품 그룹에 **담긴 상품이 하나라도 있으면** 그 상품이 담은 순서 그대로
2. 하나도 담지 않았으면 그 그룹의 **수집 조건**으로 자동 수집한 결과
3. 담긴 상품도 수집 조건도 없으면 **캐러셀 자체가 나오지 않습니다**

> **쇼핑특가의 추천 캐러셀에는 특가 상품만 담을 수 있습니다.** 쇼핑특가 관리의 특가 상세에 등록한 상품 중 **아직 종료되지 않은 특가**의 상품만 후보로 나옵니다. 베스트는 랭킹 상위 50위, 신상품은 신상품만 후보입니다.

> **①과 ②는 서로를 대체하지 않습니다.** 배너만 걸 수도, 캐러셀만 둘 수도, 둘 다 둘 수도 있습니다.

### 헷갈리기 쉬운 두 쌍

**기획전과 전문관** — 관리 화면이 **하나**입니다. 등록할 때 고른 **유형**이 어느 메뉴에 나갈지를 정합니다.

| | 기획전 | 전문관 |
|---|---|---|
| 유형 | 테마 · 브랜드 · 시즌 등 | **전문관** |
| 종료일 | 보통 지정 | 비움 (상시 운영) |
| `/exhibition` 목록 | 나옴 | **안 나옴** |
| `/specialty` 목록 | 안 나옴 | **나옴** |

두 목록은 서로 배타적입니다. "전문관을 만들었는데 기획전 목록에 없다"는 정상입니다.

**베스트와 랭킹** — 별개 메뉴가 아닙니다. 랭킹은 베스트에 흡수되어 `/ranking` 은 `/best` 로 넘어갑니다. 관리도 베스트/랭킹 관리 한 곳에서 합니다. 메뉴 관리 목록에 있던 **'랭킹' 항목도 없어졌습니다**(2026-07-22).

### 베스트/랭킹 · 신상품 화면은 이렇게 구성됩니다

| 화면 | 구성 | 무엇으로 채우나 |
|---|---|---|
| **베스트/랭킹** (`/best`) | **배너** → **BEST 10** → **카테고리별 BEST** → **브랜드별 BEST** | 배너는 배너 관리 &gt; 메뉴별 배너. 나머지는 판매·좋아요 순위로 **자동**입니다 |
| **전체 랭킹** (`/best/all`) | BEST 10 의 [전체 랭킹 더보기] 착지점. 100위까지 | 같은 순위를 길게 펼친 화면입니다 |
| **신상품** (`/new`) | **배너** → **카테고리별 신상품** → **브랜드별 신상품** | 최근 등록된 순서로 **자동**입니다 |

- 각 줄은 **최대 10개**까지 보여주고, 더 있으면 오른쪽에 **[더보기]** 가 나타납니다. 10개가 전부면 [더보기]는 나오지 않습니다.
- 상품이 하나도 없는 카테고리·브랜드는 **줄 자체가 만들어지지 않습니다.** 빈 줄이 보이는 일은 없습니다.
- **신상품으로 보이는 기간**은 기본 100일입니다. 판매 시작일을 입력하지 않은 상품은 **등록일**을 기준으로 잡습니다 — 상품을 막 올린 새 몰에서도 신상품 화면이 비지 않습니다.

### 기획전 화면은 이렇게 구성됩니다

| 화면 | 구성 | 무엇으로 채우나 |
|---|---|---|
| **기획전 목록** (`/exhibition`) | 제목 → **배너** → 기획전 카드 목록 | 배너는 **배너 관리 &gt; 메뉴별 배너 &gt; 기획전**. 카드는 발행된 기획전이 자동으로 채웁니다. |
| **기획전 상세** (`/exhibition/{주소}`) | 제목·기간·요약 → **추천 상품** → 탭 → 상품 | 추천 상품은 상품 연결에서 `상단 추천`을 켠 것(최대 8개) |
| **전문관 상세** (`/specialty/{주소}`) | **대표 이미지** → 제목 → 추천 상품 → 탭 → 상품 | 전문관만 상단 대표 이미지를 씁니다 |

- **기획전 목록 맨 위에 자동으로 걸리던 "진행 중 기획전 한 건"은 없어졌습니다.** 무엇을 크게 보여줄지는 배너로 직접 정합니다. 배너를 등록하지 않으면 제목 아래에서 곧바로 목록이 시작됩니다.
- **기획전 상세에는 상단 이미지 배너가 없습니다.** 대신 그 자리에 **추천 상품**이 들어갑니다 — 기획전에 담은 상품 중 `상단 추천`을 켠 것만 올라옵니다. 하나도 켜지 않으면 영역이 나오지 않습니다.
- 목록에 건 배너는 **상세 화면까지 따라가지 않습니다.**

## 안 보이거나 비어 있을 때

**Q. 메뉴를 켰는데 GNB 에 안 보입니다.**
아울렛 · 공동구매 · 쇼핑라이브는 **내용이 없으면 자동으로 숨겨집니다.** 아울렛은 설정한 최소 상품 수를 넘겨야 하고, 공동구매·라이브는 발행된 것이 1건이라도 있으면 나타납니다. 메뉴 미리보기에서 숨김 사유를 볼 수 있습니다.

**Q. 메뉴는 있는데 눌러보니 "준비 중" 화면입니다.**
그 메뉴에 발행된 내용이 0건입니다. 해당 관리 화면에서 등록하고 **발행**하세요. 기획전 · 전문관 · 이벤트 · 공동구매 · 라이브 · 쿠폰 · 추천 · 특가 · 아울렛이 이 방식으로 동작합니다.

**Q. 만든 기획전이 목록에 없습니다.**
① 상태가 `발행`인가, ② 목록 노출이 켜져 있는가, ③ 기간이 오늘을 포함하는가, ④ **유형이 전문관은 아닌가**, ⑤ 편집 중인 몰이 맞는가를 확인하세요.

**Q. 쿠폰존에 쿠폰이 없습니다.**
쿠폰 관리에서 **다운로드형**으로 만든 쿠폰만 쿠폰존에 노출됩니다. 코드 입력형 쿠폰은 소개만 되고, 실제 사용은 주문서에서 코드를 입력합니다.

**Q. 특가를 걸었는데 특가 화면에 안 나옵니다.**
특가는 기간뿐 아니라 **시간대 · 요일 · 선착순 수량** 조건이 모두 맞아야 열립니다. 조건이 어긋나면 해당 상품은 정상가로 나옵니다.

**Q. 멤버십 메뉴가 GNB 에 없습니다.**
기본 구성에서 멤버십은 GNB 에서 내려와 있고, **이벤트 & 혜택 화면 안의 섹션**으로 노출됩니다. GNB 에 올리려면 메뉴 관리에서 켜세요.
