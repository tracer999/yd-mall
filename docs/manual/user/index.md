# 쇼핑몰 사용 매뉴얼

이 매뉴얼은 **쇼핑몰(고객용 사이트)** 를 이용하는 방법을 안내합니다. 상품 보기, 장바구니, 주문·결제, 로그인·회원가입, 문의 등 각 메뉴가 무엇인지·어떻게 사용하는지 설명합니다.

---

## 쇼핑몰 접속

- 웹 브라우저에서 **쇼핑몰 주소**로 접속합니다. (예: `https://쇼핑몰주소`)
- 접속하면 **메인(홈) 페이지**가 보입니다.

---

## 화면 구성 (공통)

쇼핑몰의 거의 모든 페이지에서 아래 구조가 같습니다.

| 영역 | 내용 |
|------|------|
| **상단 바** | 로고, 슬로건, **로그인/회원가입** 또는 **환영 메시지·마이페이지·로그아웃** |
| **메인 메뉴** | 홈, 전체상품, 카테고리(테마별), 공지사항, **검색 아이콘**, **장바구니 아이콘** (장바구니 개수 표시) |
| **가운데 본문** | 해당 페이지 내용 (메인 배너, 상품 목록, 상세 등) |
| **하단 푸터** | 회사명, 고객센터 링크(공지·문의), 연락처, 이용약관·개인정보처리방침 링크 |

휴대폰에서는 메뉴가 **햄버거 버튼(≡)** 으로 접을 수 있습니다.

---

## 이 매뉴얼에서 다루는 내용

| 항목 | 설명 |
|------|------|
| **홈(메인)** | 메인 페이지에 무엇이 보이는지, 배너·신상품·베스트 등 |
| **검색** | 상품 검색하는 방법 |
| **상품** | 상품 목록 보기, 카테고리 선택, 상품 상세, 장바구니 담기·찜 |
| **약관/정책/소개** | 이용약관, 개인정보 처리방침, 회사 소개 페이지 |
| **공지사항** | 공지 목록·상세 보기 |
| **1:1 문의** | 문의 목록·문의 작성·내 문의 보기 (로그인 필요) |
| **로그인/회원가입** | 로그인, Google·Kakao 회원가입, 추가 정보 입력, 로그아웃 |
| **장바구니** | 담은 상품 보기, 수량 변경·삭제, 주문하기 |
| **주문/결제** | 구매 방법 선택, 주문서 작성, 쿠폰·포인트, 결제, 주문 완료 |

왼쪽 목차에서 원하는 항목을 클릭하면 자세한 내용을 볼 수 있습니다.
