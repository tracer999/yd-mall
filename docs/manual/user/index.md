# 쇼핑몰 사용 매뉴얼

이 매뉴얼은 **쇼핑몰(고객용 사이트)** 를 이용하는 방법을 안내합니다. 상품 보기, 장바구니, 주문·결제, 로그인·회원가입, 혜택 받기, 문의 등 각 메뉴가 무엇인지·어떻게 사용하는지 설명합니다.

---

## 쇼핑몰 접속

- 웹 브라우저에서 **쇼핑몰 주소**로 접속합니다. (예: `https://쇼핑몰주소`)
- 접속하면 **메인(홈) 페이지**가 보입니다.

---

## 화면 구성 (공통)

쇼핑몰의 거의 모든 페이지에서 아래 구조가 같습니다.

| 영역 | 내용 |
|------|------|
| **상단 바** | 로고, **로그인/회원가입** 또는 **환영 메시지·마이페이지·로그아웃**, 고객센터 |
| **메인 메뉴(상단 메뉴)** | 카테고리, 쇼핑특가, 베스트, 신상품, 추천, 기획전, 이벤트, 브랜드, 전문관, 공동구매, 아울렛, 쇼핑라이브 등 + **검색**, **장바구니**(담은 개수 표시) |
| **가운데 본문** | 해당 페이지 내용 (메인 배너, 상품 목록, 상세 등) |
| **하단 푸터** | 회사명, 고객센터 링크(공지·문의), 연락처, 이용약관·개인정보처리방침 링크 |

- **PC 넓은 화면**에서는 오른쪽에 **바로접속 레일**(장바구니·최근 본 상품·맨 위로 등)이 보입니다. 접었다 펼 수 있습니다.
- **휴대폰**에서는 화면 **아래 고정 바**(홈·카테고리·장바구니·마이)가 있고, 카테고리를 누르면 전체 화면 카테고리가 열립니다.

> 상단 메뉴는 쇼핑몰 설정에 따라 **일부 메뉴가 없을 수 있습니다.** 아직 준비 중인 메뉴를 누르면 "준비중" 안내 화면이 보입니다.

## 알아 두면 좋은 것

- **쿠폰은 "받는 곳"과 "보관함"이 다릅니다.** 쿠폰존에서 **받고**, 마이페이지 쿠폰함에서 **확인**합니다.
- **공동구매·쇼핑라이브 상품은 장바구니에 담을 수 없습니다.** 바로구매만 가능합니다.
- **쇼핑특가는 시간·수량이 정해져 있습니다.** 시간이 끝나거나 수량이 소진되면 정상가로 돌아갑니다.
- **아울렛 상품은 구매 전 상품 상세의 할인 사유·상품 등급·하자 안내를 꼭 확인**하세요.
