# 쇼핑몰 사용 매뉴얼

이 매뉴얼은 **쇼핑몰(고객용 사이트)** 이용 방법을 **고객이 보는 화면의 메뉴 순서 그대로** 안내합니다.

---

## 쇼핑몰 접속

- 웹 브라우저에서 **쇼핑몰 주소**로 접속합니다. (예: `https://쇼핑몰주소`)
- 접속하면 **메인 화면**이 보입니다.

---

## 화면 구성 (공통)

쇼핑몰의 거의 모든 페이지에서 아래 구조가 같습니다.

| 영역 | 내용 |
|------|------|
| **상단 1줄 — 로고·검색·유틸** | 로고, **검색창**, 그리고 오른쪽에 **로그인**(또는 로그아웃) · **마이쇼핑** · **장바구니**(담은 개수 표시) · **고객센터** |
| **상단 2줄 — 메인 메뉴(GNB)** | 맨 왼쪽 **[☰ 카테고리]** 버튼 + 그 옆에 일반 메뉴들이 한 줄로 |
| **가운데 본문** | 해당 페이지 내용 (메인 배너, 상품 목록, 상세 등) |
| **하단 푸터** | 회사명, 고객센터 링크, 연락처, 이용약관·개인정보처리방침 링크 |

- **PC 아주 넓은 화면**(가로 1600px 이상)에서는 오른쪽에 **바로접속 레일**이 보입니다 — 장바구니 · 찜 · 찜한 브랜드 · 최근본상품 · 멤버십 · 앱다운 · TOP. 헤더의 **바로접속 ON/OFF** 로 접었다 펼 수 있습니다.
- **휴대폰**에서는 화면 **아래 고정 바**(홈 · 카테고리 · 장바구니 · 마이)가 있고, **카테고리**를 누르면 전체 화면 카테고리가 열립니다. 상단 메뉴는 **좌우로 밀어서** 볼 수 있습니다.

> 상단 메뉴는 쇼핑몰 설정에 따라 **일부 메뉴가 없을 수 있습니다.** 또 **내용이 없는 메뉴는 자동으로 숨겨집니다** — 예를 들어 진행 중인 공동구매나 쇼핑라이브가 하나도 없으면 그 메뉴는 아예 나오지 않습니다.

## 알아 두면 좋은 것

- **쿠폰은 "받는 곳"과 "보관함"이 다릅니다.** 쿠폰존에서 **받고**, 마이페이지 쿠폰함에서 **확인**합니다.
- **공동구매·쇼핑라이브 상품은 장바구니에 담을 수 없습니다.** 바로구매만 가능합니다.
- **쇼핑특가는 시간·수량이 정해져 있습니다.** 시간이 끝나거나 수량이 소진되면 정상가로 돌아갑니다.
- **아울렛 상품은 구매 전 상품 상세의 할인 사유·상품 등급·하자 안내를 꼭 확인**하세요.
- **메뉴가 어제와 다를 수 있습니다.** 내용이 없는 메뉴는 자동으로 숨겨지고, 기간이 정해진 메뉴는 기간이 끝나면 사라집니다.
