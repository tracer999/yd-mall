# 고객센터

**궁금한 점을 해결하는 곳**입니다. 자주 묻는 질문(FAQ)을 찾아보고, 답을 못 찾으면 **1:1 문의**나 **카카오톡 상담**으로 물어보실 수 있습니다.

- 화면 위 오른쪽의 **고객센터** 아이콘, 또는 화면 맨 아래 **고객센터** 링크로 들어갑니다.

---

## 화면 구성

| 영역 | 내용 |
|------|------|
| **왼쪽 메뉴** | 1:1 문의하기 · 1:1 문의내역 · 공지사항 전체보기 · 자주묻는질문 분류 · **대표번호** |
| **가운데** | FAQ 검색창 → 자주 묻는 질문 → 최근 공지사항 5건 |

## 자주 묻는 질문(FAQ) 찾아보기

### 그냥 둘러보기

- 고객센터에 들어가면 **"자주묻는 질문 BEST 10"** 이 먼저 나옵니다. 많이 찾는 질문들입니다.
- 질문을 클릭하면 **아래로 펼쳐지며 답변**이 보입니다. 다시 누르면 접힙니다.

### 분류로 찾기

- 왼쪽 메뉴의 **자주묻는질문** 아래에 **분류 목록**이 있습니다. 각 분류에 **질문이 몇 개 있는지** 함께 표시됩니다.
- 분류를 클릭하면 그 분류의 질문만 모아 보여 줍니다.

### 검색으로 찾기

1. 가운데의 **검색창**에 궁금한 낱말을 입력합니다. (예: "배송", "환불")
2. **검색** 버튼을 누릅니다.
3. **질문과 답변 내용을 함께 찾아** 결과를 보여 줍니다.
4. 결과가 없으면 **"검색 결과가 없습니다"** 안내와 함께 **1:1 문의** 안내가 나옵니다.

---

## 1:1 문의

**FAQ에서 답을 찾지 못했을 때** 직접 물어보는 방법입니다. **로그인이 필요**합니다.

### 문의 남기기

1. 왼쪽 메뉴의 **1:1 문의하기** 를 누릅니다.
2. **제목**과 **내용**을 입력합니다.
3. **등록** 버튼을 누르면 접수됩니다.

### 답변 확인하기

1. 왼쪽 메뉴의 **1:1 문의내역** 을 누르면 **마이페이지**의 문의 목록으로 이동합니다.
2. 각 문의에 **답변완료 / 답변대기** 상태가 표시됩니다.
3. 문의를 클릭하면 내용과 답변을 볼 수 있습니다. **내가 쓴 문의만** 볼 수 있습니다.

## 공지사항

- 고객센터 화면 아래쪽에 **최근 공지 5건**이 나옵니다.
- **전체보기** 를 누르면 공지 목록 전체를 볼 수 있습니다.
- 중요한 공지는 목록 맨 위에 표시됩니다. 제목을 클릭하면 내용을 볼 수 있습니다.
- 배송 지연·휴무일·이벤트 안내 등이 올라오니 가끔 확인해 주세요. ([공지사항](notices.md))

## 카카오톡으로 문의하기

- 어느 화면에 있든 **오른쪽 아래에 노란 카카오 버튼**이 떠 있습니다.
- 마우스를 올리면 **"카카오톡으로 문의하세요."** 라는 안내가 펼쳐집니다.
- 버튼을 누르면 **카카오톡 상담 채널**이 새 탭으로 열립니다. 상품 문의·1:1 상담을 바로 하실 수 있습니다.
- 홈 화면 중간에도 **"카카오채널로 문의하기"** 안내가 있을 수 있습니다.

> 카카오 상담 버튼은 쇼핑몰이 카카오 채널을 운영할 때만 보입니다.

---

## 전화로 문의하기

- 왼쪽 메뉴 아래쪽에 **대표번호**가 표시됩니다. (등록되어 있을 때만)
- **운영 시간은 공지사항**에서 확인해 주세요.
