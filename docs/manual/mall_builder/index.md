# 몰 빌더 가이드

이 문서는 **새 쇼핑몰을 처음부터 만들고 싶은 사람**을 위한 안내서입니다. 순서대로 따라 하면 빈 몰을 하나 만들고, 카테고리·상품을 채우고, 필요하면 지우고 다시 만드는 전체 과정을 익힐 수 있습니다.

## 이 솔루션은 무엇인가

이 프로그램은 **특정 쇼핑몰을 실제로 운영하기 위한 것이 아닙니다.** 몰이 필요한 사용자에게 **몰을 만들어(빌드해) 포팅해 주는 "몰 빌더" 솔루션**입니다. 즉,

- 목표는 "운영"이 아니라 **"쉽게 몰을 찍어내는 것"** 입니다.
- 그래서 몰을 **만들고 → 확인하고 → 지우고 → 다시 만드는** 반복 작업이 자연스럽습니다.
- 화면에 보이는 건강식품관·종합관 같은 몰도 운영 데이터가 아니라 **빌더가 만들어 낸 예시 몰**입니다.

> 자세한 배경은 [솔루션 개요](overview) 문서를 참고하세요.

## 따라 하기 순서

1. [솔루션 개요](overview) — 이 프로젝트가 어떤 물건인지, 무엇을 할 수 있는지
2. [몰 만들기](create_mall) — 새 몰을 만들고 초기 골격(메뉴·테마·홈)을 채우기
3. [몰 채우기](fill_content) — 카테고리·상품·메뉴·테마·페이지 등 실제 내용 넣기
4. [몰 지우고 다시 만들기](delete_rebuild) — 만든 몰을 강제 삭제하고 처음부터 다시 만들기

## 어디서 작업하나

모든 몰 빌드 작업은 **관리자 페이지**에서 합니다.

| 화면 | 경로 | 하는 일 |
|------|------|---------|
| 관리자 로그인 | `/admin/login` | 관리자 계정으로 로그인 |
| 몰 관리 | `/admin/malls` | 몰 목록 · 만들기 · 선택(전환) · 삭제 |
| 카테고리 관리 | `/admin/categories` | 선택한 몰의 카테고리 편집 |
| 상품 관리 | `/admin/products` | 선택한 몰의 상품 편집 |

> **핵심 개념 — "선택(전환)":** 관리자에서 카테고리·상품·메뉴·테마·페이지 빌더는 **"지금 선택된 몰"** 기준으로 동작합니다. 여러 몰이 있을 때는 몰 관리 화면(또는 우측 상단 몰 선택기)에서 **먼저 대상 몰을 선택**한 뒤 편집하세요.
