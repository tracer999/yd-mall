# 몰 빌더 가이드 (퀵 스타트)

이 문서는 **새 쇼핑몰을 처음부터 만들고 싶은 사람**을 위한 **가장 빠른 안내서**입니다. 몰 하나를 만드는 데 **꼭 필요한 최소한의 작업만** 순서대로 담았습니다.

> **가장 빠른 길:** [몰 만들기](create_mall) 문서 하나만 따라 해도 **볼 수 있는 몰이 하나 완성**됩니다. 몰 등록 폼에서 **테마**를 고르고 **샘플 데이터 포함**을 켠 채로 저장하면, 디자인·헤더·메뉴·메인 화면·샘플 상품까지 자동으로 채워집니다.
>
> 나머지 문서는 그렇게 만든 몰을 **내 것으로 바꾸는** 과정입니다.

> 몰을 만든 뒤의 실제 운영 작업(주문·회원·프로모션 등)은 **[관리자 매뉴얼](/manual/admin)** 에서 관리자 메뉴 순서대로 설명합니다.

## 이 솔루션은 무엇인가

이 프로그램은 **특정 쇼핑몰을 실제로 운영하기 위한 것이 아닙니다.** 몰이 필요한 사용자에게 **몰을 만들어(빌드해) 포팅해 주는 "몰 빌더" 솔루션**입니다. 즉,

- 목표는 "운영"이 아니라 **"쉽게 몰을 찍어내는 것"** 입니다.
- 그래서 몰을 **만들고 → 확인하고 → 지우고 → 다시 만드는** 반복 작업이 자연스럽습니다.
- 화면에 보이는 건강식품관·종합관 같은 몰도 운영 데이터가 아니라 **빌더가 만들어 낸 예시 몰**입니다.

> 자세한 배경은 [솔루션 개요](overview) 문서를 참고하세요.

## 따라 하기 순서

| 단계 | 문서 | 하는 일 | 꼭 해야 하나 |
|------|------|---------|-------------|
| 0 | [솔루션 개요](overview) | 이 프로젝트가 어떤 물건인지 | 읽기만 |
| **1** | **[몰 만들기](create_mall)** | 몰 등록 → 테마 선택 → 초기 골격 자동 생성 | **필수** |
| **2** | **[테마·디자인 다듬기](theme_design)** | 색상·로고·모서리·헤더·메인 히어로 | **필수** |
| **3** | **[몰 채우기](fill_content)** | 카테고리 → 상품 → 메뉴 → 홈 발행 | **필수** |
| 4 | [몰 지우고 다시 만들기](delete_rebuild) | 만든 몰을 강제 삭제하고 다시 시작 | 필요할 때만 |

## 어디서 작업하나

모든 몰 빌드 작업은 **관리자 페이지**에서 합니다.

| 화면 | 경로 | 하는 일 |
|------|------|---------|
| 관리자 로그인 | `/admin/login` | 관리자 계정으로 로그인 |
| 몰 리스트 관리 | `/admin/malls` | 몰 목록 · 만들기 · 선택(전환) · 삭제 |
| 사이트 설정 | `/admin/site-settings` | 색상 · 로고 · 상호 |
| 테마 설정 | `/admin/theme-settings` | 모서리 · 글꼴 · 간격 |
| Header 설정 | `/admin/header-settings` | 헤더 레이아웃 · 메뉴 개수 |
| 카테고리 관리 | `/admin/categories` | 선택한 몰의 카테고리 편집 |
| 상품 관리 | `/admin/products` | 선택한 몰의 상품 편집 |
| 페이지 빌더 | `/admin/page-builder` | 메인 화면 조립 · **발행** |

> **핵심 개념 — "선택(전환)":** 관리자에서 상품·메뉴·테마·페이지 빌더는 **"지금 선택된 몰"** 기준으로 동작합니다. 여러 몰이 있을 때는 몰 관리 화면(또는 우측 상단 몰 선택기)에서 **먼저 대상 몰을 선택**한 뒤 편집하세요. (단 **카테고리·브랜드는 전 몰 공용**이라 어느 몰을 선택하든 같은 목록을 편집합니다.)
