# 몰 채우기

몰의 뼈대를 만들었으면, 이제 실제 내용을 넣습니다. **작업 전에 반드시 대상 몰이 "선택"되어 있는지** 확인하세요(몰 관리 화면 또는 우측 상단 몰 선택기).

## 순서: 카테고리 → 상품 → 꾸미기

카테고리를 먼저 만들어야 상품을 그 안에 넣을 수 있으므로 순서가 중요합니다.

### 1. 카테고리 만들기 (`/admin/categories`)

- 상품을 담을 분류를 먼저 만듭니다.
- 카테고리는 **최대 3단계**까지 계층(부모-자식)을 가질 수 있습니다.
- 규칙:
  - 4단계 이상으로는 만들 수 없습니다.
  - 자식이 달린 카테고리는 삭제할 수 없습니다(먼저 자식을 정리).

### 2. 상품 등록 (`/admin/products`)

- 각 상품은 하나의 카테고리에 속합니다.
- 이름·가격·이미지·설명 등을 입력합니다. 이미지는 업로드하면 `public/uploads/` 에 저장됩니다.
- 상품에도 몰이 자동으로 묶입니다(현재 선택된 몰).

### 3. 몰 꾸미기

내용이 어느 정도 채워지면 겉모습을 다듬습니다.

| 화면 | 하는 일 |
|------|---------|
| 헤더 설정 (`/admin/header-settings`) | 헤더/GNB 스킨, 카테고리 노출 방식 |
| 메뉴 관리 | GNB·상단·커스텀 메뉴 구성 |
| 테마 | 색상 등 디자인 토큰 |
| 페이지 빌더 | 메인(홈) 화면의 섹션(배너·상품 그리드·랭킹 등) 조립 |
| 배너 관리 (`/admin/banners`) | 메인/카테고리 배너 이미지 |

## 확인: 스토어프론트 미리보기

몰 관리 화면의 **스토어 링크**(`?mall=<코드> ↗`)를 누르거나, 브라우저에서 직접 엽니다.

```
/?mall=<몰코드>
```

예를 들어 코드가 `myshop` 이면 `/?mall=myshop` 으로 접속해 실제 손님이 보는 화면을 확인합니다. 카테고리·상품이 잘 보이면 몰 하나가 완성된 것입니다.
