# 테마·디자인 다듬기

몰을 만들면 고른 **테마**대로 디자인이 이미 적용되어 있습니다. 이 문서는 그 디자인을 **몰에 맞게 다듬는 최소한의 작업**을 설명합니다.

## 어디서 무엇을 바꾸나 — 이것만 기억하세요

디자인 설정이 네 군데로 나뉘어 있어서 가장 많이 헤매는 부분입니다.

| 바꾸고 싶은 것 | 가야 할 메뉴 | 경로 |
|---------------|-------------|------|
| **색상 · 로고 · 상호** | 사이트 설정 | `/admin/site-settings` |
| **모서리 · 글꼴 · 여백** | 테마 설정 | `/admin/theme-settings` |
| **헤더 모양 · 메뉴 개수** | Header 설정 | `/admin/header-settings` |
| **메인 최상단 큰 이미지** | 배너 관리 → 메인 슬라이더 | `/admin/banners/hero-slides` |

> **테마 설정에는 색상이 없습니다.** 색을 바꾸러 테마 설정에 들어가면 원하는 항목을 찾을 수 없습니다. 색상·로고는 **사이트 설정**입니다.

> 네 가지 모두 **몰마다 따로** 저장됩니다. 손대기 전에 상단에서 **편집 중인 몰**을 반드시 확인하세요.

---

## 1. 색상·로고 넣기 (사이트 설정)

몰의 **상호·로고·브랜드 색상**을 넣습니다. 몰을 납품할 때 가장 먼저 바뀌어야 하는 값들입니다.

> 사이트 설정은 **프리셋을 다시 적용해도 덮어쓰지 않습니다.** 로고·상호는 운영자의 자산이라 보존됩니다. 안심하고 채워 두세요.

## 2. 모양 다듬기 (테마 설정)

**모서리 둥글기 · 글꼴 · 간격**을 조정합니다.

| 항목 | 예 |
|------|-----|
| 폰트 패밀리 | `'Pretendard', sans-serif` |
| 버튼 / 카드 / 입력창 모서리 | `0.5rem`, `8px`, `0` |
| 섹션 간격 | `3rem` |
| 본문 최대 너비 | `72rem` |
| 상품 카드 스타일 | shadow(그림자) · border(테두리) · flat(민무늬) |

- **비워 두면 기본값으로 되돌아갑니다.** 되돌리는 가장 쉬운 방법입니다.
- **입력 형식이 엄격합니다.** 길이는 `0.5rem`·`12px`·`50%`·`0` 형식만 됩니다. 틀리면 저장이 거부되고 이유를 알려 줍니다.
- 저장하면 **고객 화면에 즉시 반영**됩니다.

자세한 내용은 관리자 매뉴얼 [테마 설정](/manual/admin/theme)을 보세요.

> **테마 자체(테마 1/2/3)를 바꾸려면** 여기가 아니라 **몰 관리의 프리셋 재적용** 또는 **페이지 빌더 → 테마 설정 탭**을 씁니다. 메인 구성만 손쉽게 바꾸고 싶다면 **페이지 빌더 → 페이지 설정 → 이지모드**에서 미리 만든 3종 구성 중 하나를 고르면 히어로 아래 캐러셀이 통째로 교체됩니다.
>
> ⚠️ 테마를 다시 적용하면 위에서 다듬은 **모서리·글꼴·간격 값이 그 테마 기본값으로 초기화**됩니다. 세부 커스터마이즈는 테마 재적용을 끝낸 뒤에 하세요.

## 3. 헤더 모양 고르기 (Header 설정)

헤더 레이아웃을 셋 중에 고릅니다.

| 레이아웃 | 어울리는 몰 |
|---------|-------------|
| **기본형** — 카테고리 버튼 + 평면 GNB | 대부분의 몰 |
| **드로어형** — 햄버거 전체메뉴 | 메뉴·카테고리가 많은 종합몰 |
| **에디토리얼형** — 투명 오버레이 헤더 | 화보 느낌 (테마 3 과 짝) |

여기서 **GNB 최대 노출 수**(상단 메뉴 최대 개수)도 정합니다. **메뉴를 켰는데 상단에 안 보인다면 이 값에 걸려 뒤에서 잘린 것**일 수 있습니다.

## 4. 메인 히어로 채우기 (배너 관리 → 메인 슬라이더)

메인 화면 **최상단의 큰 이미지 영역**입니다. 테마 1·2·3 모두 이 영역을 씁니다.

> ⚠️ **슬라이드가 하나도 없으면 히어로 영역이 통째로 사라집니다.** 빈 칸을 보여 주지 않으려고 일부러 그렇게 만든 것입니다. **"메인 상단에 큰 이미지가 안 보여요"** 의 대부분이 이 경우입니다 — 배너 관리 → **메인 슬라이더** 탭에서 슬라이드를 등록하세요.
>
> 몰을 만들 때 **샘플 데이터 포함**을 켰다면 이미 채워져 있습니다.
