# 몰 만들기

새 몰을 하나 등록하고, 초기 골격(GNB 메뉴·헤더 정책·테마·메인 화면)을 자동으로 채우는 단계입니다.

## 1. 관리자 로그인

`/admin/login` 으로 접속해 관리자 계정으로 로그인합니다. (몰 관리는 `super_admin`·`admin` 권한만 접근할 수 있습니다.)

## 2. 몰 관리 화면 열기

관리자 좌측 메뉴에서 **몰 관리**(`/admin/malls`)로 이동합니다. 지금까지 만든 몰들이 표로 보입니다.

| 열 | 뜻 |
|----|-----|
| 코드 | 스토어프론트 주소에 쓰는 몰 식별자 (`?mall=코드`) |
| 데이터 | 이 몰에 등록된 카테고리·상품 수 |
| 기본/활성 | 기본몰 여부, 그리고 노출(ON/OFF) 상태 |
| 관리 | 선택(전환) · 수정 · 삭제 |

## 3. "몰 등록" 클릭

우측 위 **몰 등록** 버튼을 누르면 등록 폼이 열립니다. 입력 항목:

| 항목 | 설명 | 규칙 |
|------|------|------|
| 코드 | 몰 식별자. 주소에 그대로 쓰입니다. | 소문자·숫자·`-`·`_` 2~50자, 몰끼리 중복 불가 |
| 이름 | 사람이 읽는 몰 이름 | 필수 |
| 도메인 | (선택) 이 몰 전용 도메인 | 비워도 됨 |
| 헤더 스킨 | 헤더/GNB 모양(프리셋) | 아래 참고 |

### 헤더 스킨(프리셋) 고르기

몰을 만들 때 **헤더·GNB 스킨**을 고르면, 그에 맞는 내비게이션·메뉴·테마·홈 섹션이 한 번에 채워집니다.

| 스킨 | 모양 | 어울리는 몰 |
|------|------|-------------|
| **기본형** | 카테고리 버튼 + 평면 GNB (3단 헤더) | 카테고리가 몇 개 안 되는 일반 몰 |
| **드로어형** | 햄버거 버튼으로 전체메뉴 + 아코디언 카테고리 | 카테고리가 많은 종합몰 |

> 스킨은 몰 규모와 무관하게 자유롭게 고르면 되고, 나중에 **헤더 설정**(`/admin/header-settings`)에서 언제든 바꿀 수 있습니다.

## 4. 저장 → 초기 구성 자동 생성

저장하면 몰이 만들어지고, 고른 프리셋대로 **초기 골격이 자동으로 채워집니다**:

- GNB 메뉴 세트
- 헤더 정책(내비 레이아웃)
- 기본 테마
- 메인(홈) 화면 골격

"**〇〇 몰을 만들고 초기 구성을 채웠습니다**" 안내가 뜨면 성공입니다. 이제 몰의 뼈대는 있지만 **카테고리·상품은 비어 있는** 상태입니다.

> 혹시 "몰은 만들어졌지만 초기 구성에 실패했습니다" 라고 나오면, 몰 자체는 남아 있는 것입니다. 몰 수정 화면에서 **프리셋 적용**을 다시 실행하면 골격을 다시 채울 수 있습니다.

## 5. 이 몰을 편집 대상으로 선택

몰 관리 표에서 방금 만든 몰의 **선택** 버튼을 누릅니다. 그러면 카테고리·상품·메뉴·테마·페이지 빌더가 **모두 이 몰 기준**으로 동작합니다. (우측 상단 몰 선택기도 같은 몰로 바뀝니다.)

> 비활성(OFF) 몰은 선택할 수 없습니다. 먼저 몰 수정에서 활성화하세요.
