# 몰 만들기

새 몰을 하나 등록하고, 초기 골격(테마·헤더·GNB 메뉴·메인 화면)을 자동으로 채우는 단계입니다. **여기까지만 하면 볼 수 있는 몰이 하나 생깁니다.**

## 1. 관리자 로그인

`/admin/login` 으로 접속해 관리자 계정으로 로그인합니다. (몰 관리는 `super_admin` 권한만 접근할 수 있습니다.)

## 2. 몰 관리 화면 열기

관리자 좌측 메뉴에서 **쇼핑몰 관리 → 몰 리스트 관리**(`/admin/malls`)로 이동합니다. 지금까지 만든 몰들이 표로 보입니다.

| 열 | 뜻 |
|----|-----|
| 코드 | 스토어프론트 주소에 쓰는 몰 식별자 (`?mall=코드`) |
| 데이터 | 이 몰에 등록된 카테고리·상품 수 |
| 기본/활성 | 기본몰 여부, 그리고 노출(ON/OFF) 상태 |
| 관리 | 선택(전환) · 수정 · 삭제 |

## 3. "몰 등록" 클릭

우측 위 **몰 등록** 버튼을 누르면 등록 폼이 열립니다.

### 기본 정보

| 항목 | 설명 | 규칙 |
|------|------|------|
| **코드** (고정 식별자) | 몰 식별자. 주소에 그대로 쓰입니다. | 소문자·숫자·`-`·`_` 2~50자, 몰끼리 중복 불가 |
| **몰 이름** | 사람이 읽는 몰 이름 | 필수 |
| **도메인** (선택) | 이 몰 전용 도메인 | 비워도 됨. **지금은 저장만 되고 동작하지 않습니다.** |
| **활성** | 끄면 이 몰로 접속할 수 없습니다. | |
| **기본몰** | 주소에 `?mall=` 이 없을 때 열리는 몰. 항상 하나입니다. | |

> **몰 하나만 납품한다면 기본몰로 만드세요.** 기본몰은 **코드를 입력하지 않아도 됩니다** — 저장할 때 자동으로 붙습니다(예: `main`). 그러면 고객은 `?mall=` 없이 `/` 로 바로 접속합니다.

### 테마 선택 (이지 모드)

**몰의 첫인상을 정하는 가장 중요한 선택**입니다. 테마를 고르면 그에 맞는 **디자인 · 헤더 · 메인 화면**이 한 번에 구성됩니다.

| 테마 | 메인 최상단 모양 | 어울리는 몰 |
|------|-----------------|-------------|
| **테마 1 — 상품 배너 슬라이드쇼** (상품형) | 상품을 크게 돌리는 **상품 쇼케이스** | 상품을 바로 보여 주고 싶은 몰 |
| **테마 2 — 일반 배너 슬라이드쇼** (배너형) | 전체폭 **이미지 배너 슬라이드** | 일반적인 몰 (**기본 선택**) |
| **테마 3 — 에디토리얼(풀블리드)** (에디토리얼) | 화면을 꽉 채우는 **풀블리드 히어로** + 투명 헤더 + 세리프 글꼴 | 화보 느낌의 라이프스타일 몰 |

셋 중 무엇을 골라도 그 아래에는 **베스트 · 쇼핑특가 · 베스트 카테고리 · 베스트 브랜드** 캐러셀이 자동으로 놓입니다.

> **몰 규모와 상관없습니다.** 대형몰이든 소형몰이든 세 테마 중 하나를 쓰면 됩니다. 나중에 **페이지 빌더**에서 얼마든지 더하거나 바꿀 수 있습니다.

### 샘플 데이터 포함 ← 체크한 채로 두세요

기본으로 **체크되어 있습니다.** 갓 만든 몰은 상품이 하나도 없어서 **메인 화면이 텅 비어 보입니다.** 이 옵션은 **카테고리 3 · 브랜드 2 · 상품 6 · 특가 1** 을 함께 만들어 **생성 직후 메인 화면이 바로 뜨게** 해 줍니다.

최소한만 넣기 때문에 나중에 상품 관리에서 쉽게 지우거나 바꿀 수 있습니다. **처음 몰을 만들어 보는 중이라면 반드시 켜 두세요.**

## 4. "저장하고 구성 채우기" → 초기 구성 자동 생성

버튼을 누르면 몰이 만들어지고, 고른 테마대로 **초기 골격이 자동으로 채워집니다**:

- 테마(디자인 토큰)
- 헤더 정책(레이아웃·메뉴 슬롯)
- GNB 메뉴 세트 (카테고리·쇼핑특가·베스트·신상품·이벤트·기획전·브랜드·전문관)
- 메인(홈) 화면 골격
- (체크했다면) 샘플 카테고리·브랜드·상품·특가

"**〇〇 몰을 만들고 초기 구성을 채웠습니다**" 안내가 뜨면 성공입니다.

> 혹시 "몰은 만들어졌지만 초기 구성에 실패했습니다" 라고 나오면, 몰 자체는 남아 있는 것입니다. 몰 수정 화면의 **프리셋 재적용**을 실행하면 골격을 다시 채울 수 있습니다.

## 5. 이 몰을 편집 대상으로 선택

몰 관리 표에서 방금 만든 몰의 **선택** 버튼을 누릅니다. 그러면 카테고리·상품·메뉴·테마·페이지 빌더가 **모두 이 몰 기준**으로 동작합니다. (우측 상단 몰 선택기도 같은 몰로 바뀝니다.)

> 비활성(OFF) 몰은 선택할 수 없습니다. 먼저 몰 수정에서 활성화하세요.

## 6. 바로 확인

`/?mall=<몰코드>` 로 접속해 보세요. (기본몰이면 그냥 `/`) 샘플 데이터를 포함했다면 **메인 화면이 이미 그럴듯하게 떠 있을 것**입니다. 여기까지가 몰 하나를 만드는 최소 과정입니다.

---

## 나중에 테마를 바꾸고 싶다면 — 프리셋 재적용

몰 **수정** 화면 아래쪽에 **프리셋 재적용** 영역이 있습니다. 여기서 다른 테마를 골라 다시 적용할 수 있습니다.

- 현재 상태(**내비 설정 · 테마 · 사이트 설정 · 홈 섹션 개수**)가 먼저 표시됩니다.
- **프리셋 적용**을 누르면 **내비 정책 · GNB 메뉴 ON/OFF · 테마 토큰**이 프리셋 값으로 되돌아갑니다.
- **로고·상호·연락처(사이트 설정)는 덮어쓰지 않습니다.** 운영자가 채운 자산이라 보존됩니다.

> ⚠️ **"메인 화면(홈 섹션)까지 교체" 체크박스는 조심하세요.** 체크하면 **현재 홈 섹션이 전부 삭제되고** 프리셋 골격으로 교체된 뒤 발행됩니다. **되돌릴 수 없습니다.** 체크하지 않으면 홈은 그대로 두고 내비·메뉴·테마만 되돌립니다.
