# 1. 몰 만들기

몰을 하나 등록하면 고른 테마대로 **테마·헤더·GNB 메뉴·메인 화면 골격**과 **멤버십 기본 등급(베이직·실버·골드·VIP)**이 자동으로 채워집니다. **여기까지만 하면 볼 수 있는 몰이 하나 생깁니다.**

화면 위치: <span class="manual-path">관리자 &gt; 쇼핑몰 관리 &gt; 몰 리스트 관리</span> (`/admin/malls`) — **몰 등록** 버튼

> 몰 관리는 `super_admin` · `admin` 권한만 접근할 수 있습니다.

---

## 1) 기본 정보 입력

| 항목 | 필수 | 설명 |
|------|------|------|
| **코드** (고정 식별자) | 필수 | 소문자·숫자·`-`·`_` 2~50자. `?mall=코드` 주소에 그대로 쓰입니다. 중복 불가 |
| **몰 이름** | 필수 | 사람이 읽는 이름 (최대 100자) |
| **도메인** | 선택 | 향후 도메인 라우팅용. **지금은 저장만 되고 동작하지 않습니다** |
| **활성** | — | 끄면 이 몰로 접속할 수 없고 편집 대상으로 **선택도 안 됩니다** |
| **기본몰** | — | `?mall=` 이 없을 때 열리는 몰. 항상 하나 |

> **몰 하나만 납품한다면 기본몰로 만드세요.** 기본몰을 체크하면 코드 입력칸이 사라지고 저장할 때 자동으로 부여됩니다(예: `main`). 고객은 `/` 로 바로 접속합니다.

## 2) 테마 선택

메인 최상단 **히어로 배치**를 정합니다. 셋 중 무엇을 골라도 아래에는 베스트·쇼핑특가·베스트 카테고리·베스트 브랜드 캐러셀이 자동으로 놓입니다.

| 테마 | 배치 |
|------|------|
| **테마 1 — 좌 히어로 + 우 상품 카드** (2단형) | 왼쪽 큰 슬라이드 + 오른쪽 추천 상품 카드 |
| **테마 2 — 전체폭 히어로** (전체폭) | 가로 전체폭 슬라이드 한 칸. **기본 선택** |
| **테마 3 — 풀블리드(에디토리얼)** (풀블리드) | 투명 오버레이 헤더 + 화면을 꽉 채우는 히어로, 세리프 폰트 |

## 3) 메뉴 구성 방식 선택

테마와 **무관한 별개 축**입니다.

| 방식 | 헤더 모양 |
|------|-----------|
| **카테고리 분리형** (기본) | 상단에 `☰ 카테고리` 버튼 + 그 옆 일반 메뉴 한 줄. 카테고리는 3단 드롭다운 |
| **통합형 (드로어)** | 헤더는 `☰`·로고·장바구니만. 메뉴 전체가 좌측 슬라이드 드로어에 |

## 4) 샘플 데이터 포함 — 켠 채로 두세요

기본으로 **체크되어 있습니다.** 갓 만든 몰은 상품이 없어 메인이 텅 비므로, **상품 6 · 특가 1 · 배너 3(영상 2 포함)** 을 함께 심어 생성 직후 화면이 바로 뜨게 합니다.

> **카테고리·브랜드는 새로 만들지 않고 기존 공용 카테고리에 연결합니다.** 어느 공용 카테고리에 붙일지는 <span class="manual-path">서비스 관리 &gt; 샘플 데이터 관리</span>에서 지정합니다.

## 5) 저장하고 구성 채우기

**저장하고 구성 채우기** 버튼을 누르면 테마·헤더 정책·GNB 메뉴 세트·메인 화면 골격(+샘플)이 채워집니다.

- **완료 판단:** 목록 상단에 "**〇〇 몰을 만들고 초기 구성을 채웠습니다**" 안내가 뜬다.
- "몰은 만들어졌지만 초기 구성에 실패했습니다" 가 뜨면 몰 자체는 남아 있습니다. 페이지 빌더 → **테마 설정** 탭에서 테마를 다시 적용하세요.

## 6) 이 몰을 편집 대상으로 선택 → 확인

몰 목록에서 방금 만든 몰의 **선택** 버튼을 누릅니다. 이후 상품·메뉴·테마·페이지 빌더가 모두 이 몰 기준으로 동작합니다.

그다음 **스토어 링크**(`?mall=<코드> ↗`)를 눌러 실제 화면을 봅니다. 메인이 뜨면 1단계 완료입니다.

> 비활성(OFF) 몰은 선택할 수 없습니다. 먼저 몰 수정에서 활성화하세요.

> 몰 수정 화면에는 테마 선택이 없습니다. 만든 뒤 테마·메뉴 방식을 바꾸려면 **페이지 빌더 → 테마 설정 탭**으로 갑니다.
