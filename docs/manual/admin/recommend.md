# 상품 추천관리

고객 화면의 **추천 페이지(`/recommend`)에 들어갈 섹션**을 만드는 메뉴입니다. "면역력 관리", "선물 추천", "유산균 모음"처럼 **주제를 정하고 상품을 손으로 골라 담으면**, 그 주제가 추천 페이지의 섹션(한 줄) 하나가 되어 고객에게 노출됩니다.

여기서 만드는 것은 **추천 그룹** 한 가지뿐입니다. **그룹 하나 = 섹션 한 줄**이라고 생각하시면 정확합니다.

> **추천 그룹은 "사람이 고른 큐레이션"입니다.** 할인율·카테고리 같은 조건을 걸어 상품이 자동으로 들어오는 기능은 이 화면에 없습니다. 담는 방법은 **수동 선택 하나뿐**입니다. 자동으로 채워지는 줄이 필요하면 베스트/랭킹 관리나 상품 그룹을 쓰세요.

---

## 1. 시작하기 전에 — 준비물 2가지

| 준비물 | 왜 필요한가 | 어디서 만드나 |
|---|---|---|
| **담을 상품** | 추천 그룹은 이미 등록된 상품을 골라 담는 화면입니다. 여기서 상품을 새로 만들 수는 없습니다. | 상품 관리 |
| **섹션 제목이 될 문구** | 그룹명이 고객 화면에 **섹션 제목 그대로** 노출됩니다. 관리용 메모 이름을 쓰면 고객이 그대로 봅니다. | 미리 정해 두세요 |

> **지금 편집 중인 몰을 반드시 확인하세요.** 추천 그룹은 **몰별로 따로 저장**됩니다. 관리자 화면 상단의 편집 몰이 A몰인데 B몰에 만들 생각이었다면, 저장해도 B몰의 추천 페이지에는 나타나지 않습니다.

## 2. 전체 흐름 한눈에 보기

```
① 그룹 만들기 (그룹명·근거문구·순서·사용) → 저장
        ↓                       (여기까지 해야 상품 영역이 열립니다)
② 상품 담기  (조회 팝업에서 여러 개 한 번에)
        ↓
③ 순서 정리  (위/아래 화살표 — 누르는 즉시 저장)
        ↓
④ 고객 추천 페이지에서 확인
```

**①번 저장을 먼저 해야 ②가 가능합니다.** 새 등록 화면에는 상품 영역이 아직 없고, *"그룹을 먼저 저장하면 상품을 담을 수 있습니다."* 라는 안내만 보입니다.

## 3. 1단계 — 추천 그룹 만들기

<div class="manual-goto-bar">
  <span class="manual-goto-label">이 단계 화면</span>
  <a class="manual-goto" href="/admin/recommend-groups/new" target="_blank">새 추천 그룹 등록 화면 열기</a>
</div>

목록 화면 오른쪽 위의 **그룹 등록** 버튼을 누르면 나오는 화면입니다. 입력란은 **4개**뿐입니다.

| 입력란 | 필수 | 설명 | 예시 |
|---|---|---|---|
| **그룹명** | 필수 | 추천 페이지에 **섹션 제목으로 그대로** 노출됩니다. 최대 100자. | `면역력 관리` |
| **근거 문구** | 선택 | 섹션 제목 아래 작은 글씨로 노출됩니다. 비우면 제목만 나옵니다. 최대 200자. | `환절기에 챙기면 좋은 상품을 모았습니다` |
| **노출 순서** | 선택 | 숫자가 **작을수록 위**에 나옵니다. 0~9999. 새 그룹은 자동으로 맨 뒤 번호가 채워집니다. | `1` |
| **사용** | 체크박스 | 끄면 추천 페이지에서 **이 섹션이 통째로 사라집니다.** 담긴 내용은 그대로 남습니다. | 켜 둠 |

입력을 마치면 **저장**을 누릅니다. 저장하면 수정 화면으로 넘어가고, 이제 상품을 담을 수 있습니다.

> **그룹명은 고객이 그대로 봅니다.** `테스트`, `임시`, `홍길동요청건` 같은 관리용 이름을 넣으면 그 문구가 그대로 고객 화면의 섹션 제목이 됩니다. 반드시 고객에게 보여도 되는 문구로 지으세요.

### 근거 문구를 왜 쓰나

추천 페이지의 핵심은 "왜 이 상품을 당신에게 보여 주는가"입니다. 근거 문구가 있으면 고객이 그 줄의 의도를 이해하고 클릭합니다.

- 그룹명 `선물 추천` + 근거 문구 `받는 분 부담 없는 3만원대 구성`
- 그룹명 `유산균 모음` + 근거 문구 `장 건강을 처음 챙기신다면`

## 4. 2단계 — 상품 담기

그룹을 저장하면 화면 아래에 **담긴 상품** 영역이 열립니다. **상품 추가** 버튼을 누르면 **상품 조회** 팝업이 뜹니다.

### 4-1. 조회 팝업의 검색 조건

| 조건 | 설명 |
|---|---|
| **검색어** | 상품명 또는 상품코드로 찾습니다. **비워도 됩니다.** |
| **카테고리** | 일반 카테고리로 범위를 좁힙니다. |
| **브랜드** | 브랜드로 범위를 좁힙니다. |
| **재고** | `전체` / `재고 있음` / `재고 없음` |
| **노출** | `전체` / `노출` / `숨김` / `회원전용` |

조건을 지정하고 **검색**을 누릅니다. **초기화**를 누르면 조건이 모두 비워집니다.

> **검색어는 필수가 아닙니다.** 카테고리나 브랜드만 골라도 조회됩니다. "무엇을 담을지 아직 못 정했다"면 카테고리만 걸고 검색해 후보를 훑어보세요.

### 4-2. 여러 개를 한 번에 담기

1. 검색 결과 각 줄 왼쪽의 **체크박스**를 켭니다. 아래 **N개 선택됨** 숫자가 올라갑니다.
2. 오른쪽 위 **결과 전체 선택** 을 켜면 검색 결과 전부가 한 번에 선택됩니다.
3. **선택 담기** 를 누르면 담기고 화면이 새로고침됩니다.

> **조회 결과는 최대 100개까지만 나옵니다.** 100개에 도달하면 *"결과가 100개에 도달했습니다. 필터를 더 좁혀 주세요."* 라는 안내가 뜹니다. 이때는 카테고리·브랜드로 조건을 더 좁혀 다시 검색하세요. 그대로 두면 못 본 상품이 생깁니다.

**이미 이 그룹에 담긴 상품은 조회 결과에 아예 나오지 않습니다.** 중복해서 담을 수 없기 때문입니다. "분명 있는 상품인데 검색에 안 잡힌다" 싶으면 이미 담겨 있는지부터 확인하세요.

## 5. 3단계 — 순서 정리하고 빼기

담긴 상품은 목록에 보이는 **위에서 아래 순서 그대로** 고객 화면에 노출됩니다.

| 버튼 | 동작 |
|---|---|
| **↑ (위로)** | 한 칸 위로 올립니다. **누르는 즉시 저장**됩니다. |
| **↓ (아래로)** | 한 칸 아래로 내립니다. **누르는 즉시 저장**됩니다. |
| **제거** | 이 그룹에서 상품을 빼냅니다. **상품 자체는 삭제되지 않습니다.** |

> **순서 변경에는 저장 버튼이 없습니다.** 화살표를 누르는 순간 바로 반영됩니다. 반대로 위쪽 **기본 정보**(그룹명·근거문구·순서·사용)를 고쳤을 때는 반드시 **저장** 버튼을 눌러야 합니다. 이 둘은 서로 다른 영역이니 혼동하지 마세요.

### 담긴 상품 줄에 붙는 경고 배지

상품이 고객 화면에 뜰 수 없는 상태면 노란 배지가 붙습니다.

| 배지 | 뜻 | 해결 |
|---|---|---|
| **판매중지 — 고객 화면에 뜨지 않음** | 상품 관리에서 판매를 중지한 상품입니다. | 상품 관리에서 판매 상태를 되돌리거나 이 그룹에서 제거 |
| **숨김 — 고객 화면에 뜨지 않음** | 상품 노출이 `숨김`입니다. | 상품 관리에서 노출을 `노출`로 변경 |
| **회원전용 — 고객 화면에 뜨지 않음** | 상품 노출이 `회원전용`입니다. | 그대로 두어도 되지만 비회원에게는 보이지 않습니다 |

| 열 | 의미 |
|---|---|
| **순서** | 노출 순서 값. 작을수록 위. |
| **그룹명** | 클릭하면 수정 화면으로 갑니다. 아래 회색 글씨는 근거 문구입니다. |
| **담긴 상품** | 담은 개수. 옆에 `(노출 N)` 이 붙으면 **그중 N개만 고객에게 보인다**는 뜻입니다. |
| **사용** | 아래 표 참고 |

**사용 열에 뜨는 3가지 상태**

| 표시 | 의미 |
|---|---|
| **노출** (초록) | 사용이 켜져 있고 보일 상품도 있습니다. 정상입니다. |
| **상품 없음** (노랑) | 사용은 켜져 있지만 **보일 상품이 하나도 없어** 섹션이 그려지지 않습니다. |
| **미사용** (회색) | 사용 체크가 꺼져 있습니다. 추천 페이지에 나오지 않습니다. |

> **`상품 없음`은 "아무것도 안 담았다"만 뜻하지 않습니다.** 상품을 10개 담았어도 전부 숨김·판매중지면 똑같이 `상품 없음`이 됩니다. 담긴 상품 열의 `(노출 N)` 표시를 함께 보세요.

**삭제** 버튼을 누르면 확인창이 뜨고, 그룹과 "무엇이 담겼는지"라는 목록이 함께 지워집니다. **상품 자체는 지워지지 않습니다.**

## 7. 고객 추천 페이지에서 어디에 나오나

추천 페이지는 여러 섹션이 **정해진 순서로** 쌓입니다.

```
① 회원님을 위한 추천     (로그인 고객의 최근 본 상품 기반 — 자동)
        ↓
② 추천 그룹              ← 여기서 만든 것. 노출 순서대로 여러 줄
        ↓
③ MD 추천                (상품에 '추천' 배지를 단 것 — 상품 관리에서 지정)
        ↓
④ 지금 많이 보는 상품    (조회수 기준 — 자동)
```

즉 **여기서 만든 그룹은 ②번 자리에 들어갑니다.** ①과 ④는 관리자가 만드는 것이 아니라 데이터에서 자동으로 조립됩니다.

### 알아 두면 좋은 3가지 규칙

**① 한 섹션에는 최대 12개까지만 보입니다.**
그룹에 30개를 담아도 고객 화면에는 **위에서 12개**만 나옵니다. 그래서 순서 정리가 중요합니다.

**② 같은 상품이 화면에 두 번 나오지 않습니다.**
추천 그룹에 담긴 상품은 ①·③·④ 자동 섹션에서 자동으로 빠집니다. **운영자가 손으로 담은 큐레이션이 우선**이고, 자동 섹션이 그 상품을 피해 갑니다.

**③ 다만 추천 그룹끼리는 중복이 허용됩니다.**
`선물 추천`과 `유산균 모음`에 같은 상품을 담으면 **양쪽에 다 나옵니다.** 이건 운영자의 의도로 봅니다.

> **회원 맞춤 추천은 로그인한 고객에게만, 그것도 최근 본 상품이 있을 때만 나옵니다.** 비회원이나 처음 온 고객은 곧바로 추천 그룹부터 보게 됩니다. 그래서 **첫 번째 추천 그룹이 사실상 추천 페이지의 첫인상**입니다. 가장 자신 있는 큐레이션을 순서 1번에 두세요.

### MD 추천은 여기가 아니라 상품 화면에서 지정합니다

③ MD 추천 섹션은 이 화면에서 만들지 않습니다. **상품 관리**에서 상품을 수정하고 **상품 구분(배지)** 에 **추천**을 체크하면 그 상품이 MD 추천 섹션에 나타납니다.

## 8. 자주 막히는 상황

**Q. 그룹을 만들었는데 상품을 담을 곳이 없습니다.**
아직 저장 전입니다. 기본 정보를 입력하고 **저장**을 먼저 누르면 수정 화면 아래에 담긴 상품 영역이 나타납니다.

**Q. 상품을 담았는데 추천 페이지에 섹션이 안 보입니다.**
아래를 순서대로 확인하세요.
1. **사용** 체크박스가 켜져 있는가?
2. 목록에서 이 그룹이 **`상품 없음`** 으로 표시되어 있지 않은가? (담긴 상품이 전부 숨김·판매중지면 빈 섹션은 그려지지 않습니다)
3. **편집 중인 몰**이 맞는가? (다른 몰에 만들었을 수 있습니다)

**Q. 담긴 상품 중 일부만 고객 화면에 보입니다.**
정상입니다. 원인은 둘 중 하나입니다. ① 상품이 **숨김·회원전용·판매중지** 상태다 — 관리자 화면에서 노란 배지가 붙은 줄을 확인하세요. ② 담긴 상품이 **12개를 넘었다** — 13번째부터는 나오지 않습니다.

**Q. 조회 팝업에서 특정 상품이 검색되지 않습니다.**
① **이미 이 그룹에 담겨 있는** 상품은 후보에서 제외됩니다. ② 카테고리·브랜드·재고·노출 필터가 걸려 있는지 확인하세요. ③ 다른 몰의 상품은 조회되지 않습니다.

**Q. 순서를 바꿨는데 저장 버튼이 없습니다.**
순서는 화살표를 누르는 **즉시 저장**됩니다. 별도 저장이 필요 없습니다.

**Q. 상품을 "제거"하면 상품이 삭제되나요?**
아닙니다. 이 추천 그룹에서만 빠집니다. 상품 자체와 다른 그룹·기획전의 편성은 그대로입니다.

**Q. 그룹을 삭제하면 담긴 상품도 삭제되나요?**
아닙니다. 그룹과 "이 그룹에 무엇이 담겼는지"라는 목록만 지워집니다. 상품은 그대로 남습니다.

**Q. 같은 상품을 두 그룹에 담아도 되나요?**
됩니다. 추천 그룹끼리는 중복이 허용되어 양쪽에 다 나옵니다.

**Q. 할인율이 높은 상품을 자동으로 모으고 싶습니다.**
추천 그룹에는 자동 수집 기능이 없습니다. 조건으로 자동 구성이 필요하면 베스트/랭킹 관리나 상품 그룹을 사용하세요.

---

## 9. 아직 지원하지 않는 것

- 조건(카테고리·할인율·배지 등)으로 **상품 자동 담기** — 전부 수동 선택입니다
- 엑셀/CSV로 상품 일괄 담기
- 그룹별 클릭·매출 성과 집계
- 회원 등급·연령대별로 그룹을 다르게 보여 주기
