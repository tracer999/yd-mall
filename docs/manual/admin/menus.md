# 메뉴 관리

고객 화면 **상단에 보이는 메뉴(GNB)** 와 **헤더 유틸·우측 레일**을 관리하는 메뉴입니다.

> **먼저 확인:** 메뉴 설정은 **몰마다 따로** 저장됩니다. 작업 전에 **편집 중인 몰**이 맞는지 확인하세요.

---

## 메뉴는 4개 화면으로 나뉩니다

| 화면 | 무엇을 하나요 |
|------|----------------|
| **기능 메뉴** | 베스트·아울렛·이벤트처럼 **시스템이 제공하는 메뉴**를 켜고 끕니다. 이름·순서·PC/모바일 노출·배지(NEW 등)를 정합니다. |
| **시스템 메뉴** | 로그인·장바구니·마이페이지 같은 **헤더 유틸**과 **우측 레일**입니다. **추가·삭제는 할 수 없고**, 켜고 끄거나 순서만 바꿉니다. |
| **커스텀 메뉴** | 운영자가 **직접 만드는 메뉴**입니다. 원하는 이름과 링크로 추가합니다. |
| **메뉴 미리보기** | 지금 설정대로 고객 화면에 **어떻게 보일지 확인**합니다. **메뉴가 안 보이는 이유도 여기서 확인**합니다. |

---

## ⚠️ "메뉴를 켰는데 고객 화면에 안 보여요"

가장 많이 들어오는 문의입니다. 대부분 **고장이 아니라 정상 동작**입니다. 원인은 셋 중 하나입니다.

### ① 아울렛 — 상품이 모자라서 자동으로 숨겨짐

**판매 중인 아울렛 상품이 최소 개수(기본 30개) 미만**이면 아울렛 메뉴는 **자동으로 숨겨집니다.** 메뉴를 켜 두어도 보이지 않습니다.

→ 아울렛 상품을 더 담거나, **아울렛 설정에서 최소 노출 상품 수를 낮추세요.**

### ② 공동구매 · 쇼핑라이브 — 발행된 항목이 0건

**발행된 공동구매(또는 쇼핑라이브)가 하나도 없으면** 해당 메뉴는 **자동으로 숨겨집니다.**

→ 항목을 하나 이상 등록하고 **발행**하세요.

### ③ GNB 최대 개수를 넘김

상단 메뉴에 **보여 줄 수 있는 최대 개수**가 정해져 있습니다. 그 수를 넘으면 **뒤쪽 메뉴부터 잘려서 안 보입니다.**

→ 안 쓰는 메뉴를 끄거나, **헤더 설정에서 상단 메뉴 최대 개수를 늘리세요.**

### 확인 방법

**메뉴 미리보기** 화면을 열면 각 메뉴가 **왜 제외됐는지(제외 사유)** 가 표시됩니다. 원인을 추측하지 말고 여기부터 확인하세요.

## 기본으로 꺼져 있는 메뉴

아래 메뉴는 처음에 **꺼진 상태**입니다. 필요하면 기능 메뉴에서 켜세요.

| 메뉴 | 왜 꺼져 있나요 |
|------|----------------|
| **랭킹** | **베스트 메뉴로 통합**되었습니다. |
| **쿠폰** | 필요할 때만 켭니다. |
| **멤버십** | **이벤트 페이지 안에서 안내**하고 있습니다. |

## 메뉴 내용은 어디서 채우나요?

메뉴를 켜는 것과, 그 메뉴 안에 들어갈 **내용을 채우는 것**은 **다른 화면**입니다.

| 메뉴 | 내용을 관리하는 화면 |
|------|----------------------|
| 베스트 | **베스트 관리** |
| 쇼핑특가 | **특가 카테고리** + **쇼핑특가 관리** |
| 아울렛 | **아울렛 관리** |
| 이벤트 | **이벤트 관리** |
| 쇼핑라이브 | **쇼핑라이브 관리** |
| 쿠폰 | **쿠폰 관리** |
| 추천 | **상품 추천관리** |
| 기획전 · 전문관 | **기획전 관리** |

## 헤더 스킨 2종

**헤더 설정**에서 고릅니다. (사이트 설정 참고)

| 스킨 | 차이 |
|------|------|
| **기본형** | **카테고리 버튼이 따로** 상단에 있습니다. |
| **드로어형** | **카테고리가 메뉴 안에 합쳐져** 있습니다. |

## 관리자 메뉴 설정과 헷갈리지 마세요

**관리자 메뉴 설정(`/admin/menus`)** 은 **관리자 사이트의 왼쪽 사이드바** 를 설정하는 화면입니다. **고객 화면 메뉴와는 아무 관계가 없습니다.**

관리자 메뉴 설정에서 할 수 있는 일:

- **메뉴 순서 변경** — 위/아래 버튼이나 숫자로 순서를 바꿉니다.
- **메뉴 표시/숨김** — 사이드바에서 특정 메뉴를 숨깁니다.
- **권한 설정** — 어떤 권한을 가진 운영자에게 그 메뉴를 보일지 지정합니다.

**주의:** **자신이 쓰는 메뉴를 숨기거나 권한에서 제외하면 다음부터 그 메뉴가 안 보입니다.** 최고 관리자는 모든 메뉴가 보이도록 두는 것이 안전합니다.
