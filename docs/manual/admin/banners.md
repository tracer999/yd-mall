# 배너 관리

쇼핑몰 화면 곳곳에 들어가는 **배너 이미지**를 등록·수정·삭제하는 메뉴입니다. 홈 최상단의 큰 슬라이더부터 헤더 맨 위 띠 배너, 카테고리·브랜드 화면 상단 배너, 팝업, 메뉴별 배너, 페이지 빌더가 불러다 쓰는 프로모션 배너까지 **7개 탭**으로 나뉘어 있습니다.

> **지금 편집 중인 몰을 반드시 확인하세요.** 배너는 **몰별로 따로 저장**됩니다. 편집 몰이 A몰인데 B몰 화면을 보며 "안 나온다"고 하는 경우가 많습니다.

## 1. 7개 탭 한눈에 보기

| 탭 | 어디에 나오나 | 특징 |
|---|---|---|
| **메인 슬라이더** | 홈 최상단의 큰 영역 | 두 가지 방식 중 **하나만** 노출됩니다. 아래 2장 참고. |
| **톱바 배너·알림** | 헤더 맨 위 얇은 띠 | 알림 문구 1개 + 배너 **최대 3개**. 등록 화면 없이 한 장에서 편집. |
| **카테고리 배너** | 카테고리 목록 화면 상단 | 카테고리별 개별 배너 + 전체 공통 배너 |
| **브랜드 배너** | 브랜드 상세 화면 최상단 | 브랜드별 개별 배너 + 전체 공통 배너 |
| **팝업 배너** | 화면 위에 뜨는 팝업 | 정사각형 이미지 |
| **메뉴별 배너** | 베스트·신상품 등 메뉴 목록 화면 상단 | 켜져 있는 GNB 메뉴에 각각 걸 수 있음 |
| **프로모션 배너** | 페이지 빌더가 불러다 쓰는 자리 | **그룹 키**로 묶어 캐러셀로 노출 |

## 2. 메인 슬라이더 (홈 최상단)

<div class="manual-goto-bar">
  <span class="manual-goto-label">이 단계 화면</span>
  <a class="manual-goto" href="/admin/banners/hero-slides" target="_blank">메인 슬라이더 관리 열기</a>
</div>

홈 최상단 영역은 **두 가지 방식 중 하나로만** 채워집니다. 화면 중간의 **히어로 방식**에서 고릅니다.

| 방식 | 무엇인가 |
|---|---|
| **상품 쇼케이스** | 상품을 골라 슬라이드로 돌립니다. 고르지 않으면 **베스트 상위 7개가 자동으로 노출**됩니다. |
| **이미지 배너 슬라이드** | 직접 만든 이미지 배너를 넘기는 방식. PC·모바일 이미지를 따로 등록할 수 있습니다. |

고른 방식에는 **적용 중** 배지가 붙습니다.

> **고른 방식만 홈에 노출됩니다. 다른 방식의 콘텐츠는 지워지지 않고 그대로 보존됩니다.** 그래서 미리 만들어 두고 나중에 전환할 수 있습니다.

> **적용 중이 아닌 방식을 편집하고 있으면 노란 안내 띠가 뜹니다.** "여기서 등록한 내용은 적용 전까지 노출되지 않습니다"라는 뜻이며, 그 옆의 **이 방식을 홈에 적용** 버튼으로 바로 전환할 수 있습니다.

> **이 몰의 사이트 설정이 아직 없으면 방식을 바꿀 수 없습니다.** "사이트 설정에서 먼저 저장한 뒤 다시 시도해 주세요"라는 안내가 나오면 아래 버튼으로 이동해 한 번 저장하고 돌아오세요.

### 2-1. 상품 쇼케이스 방식

상품을 골라 슬라이드로 만듭니다. 슬라이드가 들어가는 자리(**슬롯**)는 두 가지입니다.

| 슬롯 | 자리 | 최대 개수 |
|---|---|---|
| **중앙 슬라이더 (MAIN)** | 화면 가운데에서 돌아가는 슬라이드 | **7개** |
| **우측 추천 카드 (FEATURE)** | 오른쪽 고정 카드 | **1개** |

**상품 담는 순서**

1. 각 영역의 **상품 고르기** 버튼을 누릅니다.
2. **상품명 또는 상품코드**로 검색합니다.
3. 담을 상품을 체크하고 **선택한 상품 추가**를 누릅니다.

담긴 슬라이드는 카드로 나열되며, 각 카드의 **수정**·**삭제** 버튼으로 관리합니다.

> **최대 개수를 넘겨 담으면 넘친 만큼 자동으로 빠집니다.** 몇 건이 빠졌는지 화면에 알려 줍니다. **꺼 둔(비노출) 슬라이드도 자리를 차지**한다는 점에 주의하세요.

> **이미 슬라이드로 등록한 상품은 검색 결과에 나오지 않습니다.** 중복 등록을 막기 위한 동작입니다.

> **중앙 슬라이더에 켜져 있는 슬라이드가 하나도 없으면 베스트 상위 7개 상품이 자동으로 들어갑니다.** 베스트 상품도 0건이면 홈 히어로 영역 자체가 나오지 않습니다.

**슬라이드 등록·수정 화면의 입력란**

| 입력란 | 필수 | 설명 |
|---|---|---|
| **노출 위치(슬롯)** | 필수 | `중앙 슬라이더 (MAIN)` 또는 `우측 추천 카드 (FEATURE)` |
| **슬라이드 종류** | 필수 | `상품배너` = 상품을 연결. `일반배너` = 상품 없이 이미지·문구만. |
| **연결 상품** | 상품배너일 때 | 검색해서 고릅니다. 가격·할인율·대표이미지를 이 상품에서 가져옵니다. |
| **라벨** | 선택 | 슬라이드 상단/썸네일의 작은 태그. 예: `[베스트]`. 상품배너에서 비우면 상품 공급사명이 대신 쓰입니다. |
| **헤드라인** | 선택 | 큰 문구. 상품배너에서 비우면 상품명이 표시됩니다. |
| **미디어 유형** | 필수 | `이미지` 또는 `영상` |
| **이미지** | 선택 | 상품배너에서 비우면 상품 대표이미지를 씁니다. **일반배너는 이미지가 없으면 슬라이드가 비어 보입니다.** 중앙 슬라이더는 **정사각형(1:1) 권장**. |
| **연결 링크** | 선택 | 상품배너에서 비우면 연결 상품 상세로 이동합니다. |
| **노출 순서** | — | 숫자가 작을수록 앞에 나옵니다. |
| **활성화** | — | 꺼 두면 노출되지 않습니다(자리는 차지합니다). |

**영상 슬라이드를 쓸 때** — 미디어 유형을 `영상`으로 바꾸면 아래 칸이 열립니다.

| 칸 | 필수 | 설명 |
|---|---|---|
| **PC 영상 · MP4** | **필수** | 가장 호환성이 넓습니다. |
| **PC 영상 · WebM** | 선택 | 있으면 더 가볍게 재생됩니다. |
| **모바일 영상 · MP4** | 선택 | 비우면 PC 영상을 그대로 씁니다. |
| **모바일 영상 · WebM** | 선택 | — |
| **포스터 이미지** | 선택 | 영상이 로드되기 전 보이는 정지 이미지. 넣어 두면 화면이 깜빡이지 않습니다. |

파일을 직접 올리거나 **영상 주소를 붙여넣어 가져올** 수 있습니다. 가져온 영상은 우리 서버에 저장됩니다.

> **영상은 중앙 슬라이더(MAIN)에서만 재생됩니다.** 우측 추천 카드(FEATURE)는 항상 이미지로 표시됩니다.

> **미디어 유형을 `영상`에서 `이미지`로 되돌리면 등록해 둔 영상 경로가 모두 지워집니다.**

각 영상 칸에는 **이 항목 비우기** 체크박스가 있습니다. 체크하고 저장하면 그 파일이 지워집니다.

### 2-2. 이미지 배너 슬라이드 방식

직접 만든 가로 이미지 배너를 넘겨 보여 줍니다. 홈 상단에 **가로 전체 폭**으로 나옵니다.

- 배너는 화면 우측 상단의 **배너 등록** 링크로 만들고, 등록 시 **배너 타입 = `메인 슬라이더 (이미지 배너)`** 를 고릅니다.
- 목록에서 각 배너의 **순서**와 **노출** 체크를 바로 고친 뒤 **순서·노출 저장** 버튼을 누릅니다.

> **순서가 빠른 것부터 최대 6개까지만 홈에 나옵니다.** 7번째부터는 목록에 `노출 제외` 배지가 붙습니다.

> **순서·노출을 고친 뒤 반드시 `순서·노출 저장` 버튼을 누르세요.** 누르지 않으면 변경이 사라집니다.

이 방식에서만 **배너 위에 문구를 얹을 수 있습니다** — 아래 8-2 "배너 내 문구" 참고.

### 2-3. 하단 흐름문구 (마퀴)

메인 슬라이더 화면 맨 위에 있는 영역입니다. 히어로 아래로 **문구가 옆으로 흐르는 띠**를 만듭니다.

| 입력란 | 설명 |
|---|---|
| **문구 내용** | **한 줄에 항목 하나씩** 적습니다. 비우면 기본 문구가 나옵니다. |
| **흐름 속도 (초)** | 작을수록 빠릅니다. **5~120** 범위. 기본 28. |
| (사용 체크) | 끄면 띠가 나오지 않습니다. |

입력 후 **흐름문구 저장** 버튼을 누릅니다. 저장하면 고객 화면에 **즉시 반영**됩니다.

> **이 흐름문구는 에디토리얼 히어로 전용입니다.** 테마 배치가 다르면 화면에 나타나지 않습니다.

## 3. 톱바 배너·알림 (헤더 맨 위 띠)

<div class="manual-goto-bar">
  <span class="manual-goto-label">이 단계 화면</span>
  <a class="manual-goto" href="/admin/banners/topbar" target="_blank">톱바 배너·알림 열기</a>
</div>

헤더 최상단의 얇은 띠입니다. 슬롯이 고정이라 **목록·등록 화면 없이 이 한 장에서 모두 편집**합니다.

### 3-1. 알림 메시지 (1개)

| 입력란 | 설명 |
|---|---|
| **문구** | 최대 200자. 예: `첫 구매 10% 쿠폰 증정 — 지금 확인하세요` |
| **링크 (선택)** | 문구를 눌렀을 때 이동할 주소. 예: `/events/1` |
| **노출 시작일 / 종료일 (선택)** | 비우면 기간 제한 없이 나옵니다. |
| (사용 체크) | 끄면 나오지 않습니다. |

> **문구를 비우고 저장하면 알림이 완전히 삭제됩니다.** 잠시 내리고 싶다면 문구는 두고 사용 체크만 끄세요.

### 3-2. 배너 (최대 3개)

슬롯 1·2·3이 있고, **슬롯 순서대로 왼쪽부터** 놓입니다. 슬롯마다 다음을 넣습니다.

| 입력란 | 설명 |
|---|---|
| **이미지 업로드 / 교체** | **이미지가 없으면 그 슬롯은 배너가 아닙니다.** |
| **링크 (선택)** | 배너를 눌렀을 때 이동할 주소 |
| **대체 텍스트 (선택)** | 이미지 로드 실패·스크린리더용 문구 |
| **시작일 / 종료일** | 노출 기간 |
| (새 창 / 사용 체크) | 새 창으로 열지, 지금 노출할지 |

> **이미지가 없는 슬롯은 저장할 때 비워집니다.** 실수로 이미지를 지우고 저장하면 그 슬롯 설정이 통째로 사라집니다.

> **배너는 4개째를 넣을 자리가 없습니다.** 최대 3개입니다.

편집을 마치면 화면 아래 **저장** 버튼을 누릅니다.

## 4. 카테고리 배너 · 브랜드 배너

두 탭은 동작이 같습니다. **대상(카테고리 또는 브랜드) 목록**을 보여 주고, 각 대상에 어떤 배너가 걸려 있는지 한눈에 확인합니다.

화면 구성은 이렇습니다.

1. **맨 위: 전체 공통 배너** — 개별 설정이 없는 대상에 접근했을 때 나오는 기본 배너입니다.
2. **아래: 대상 목록** — 대상마다 걸린 배너가 카드로 붙고, `+ 배너` 버튼으로 그 대상에 배너를 바로 등록합니다.
3. 상단 필터로 **전체 / 설정됨 / 미설정** 을 거를 수 있습니다. 대상이 많으면 페이지로 나뉩니다(한 쪽 40개).

> **개별 설정이 있으면 개별 배너가 우선합니다.** 전체 공통 배너는 개별 배너가 없는 대상에만 나오는 폴백입니다.

**전체 공통 배너 만들기** — 등록 화면에서 배너 타입을 `카테고리 배너`(또는 `브랜드 배너`)로 고른 뒤, **전체 공통 배너로 설정** 체크박스를 켭니다. 켜면 대상 선택 칸이 사라집니다.

> **전체 공통을 켜지 않았는데 대상을 안 고르면 저장되지 않습니다.** `대상 카테고리/브랜드를 선택하거나 전체 공통으로 설정하세요.` 라는 오류가 뜹니다.

목록에 나오는 대상은 **이 몰에 실제로 노출되는(상품이 있는) 1단계 대상** 과 **이미 배너가 걸린 대상** 입니다. 지금 상품이 없어 노출되지 않는 대상이라도 배너가 걸려 있으면 관리할 수 있도록 목록에 남습니다.

## 5. 팝업 배너

화면 위에 겹쳐 뜨는 팝업입니다. 등록 화면에서 배너 타입을 `팝업 배너`로 고릅니다. 대상 선택은 없고, **이미지·링크·기간·활성화**만 정합니다. **정사각형(1:1) 이미지**를 씁니다.

## 6. 메뉴별 배너

**베스트·신상품·쇼핑특가처럼 켜져 있는 GNB 메뉴 목록 화면 상단**에 배너를 겁니다.

탭 안에서 메뉴를 하나 골라(서브탭) 그 메뉴의 배너만 봅니다. 서브탭에는 각 메뉴에 몇 건이 등록됐는지 함께 표시됩니다.

등록 화면에서 배너 타입을 `메뉴별 배너`로 고르면 **노출 메뉴** 선택 칸이 열립니다.

> **상품 캐러셀이 걸린 메뉴에도 배너를 걸 수 있습니다.** 이 경우 **배너가 상품 캐러셀 위에** 표시된다는 안내가 뜹니다. 둘은 공존합니다.

> **목록에 없는 메뉴는 저장할 수 없습니다.** 노출될 곳이 없는 배너가 생기지 않도록 막습니다.

## 7. 프로모션 배너 (페이지 빌더 연동)

페이지 빌더의 **프로모션 배너 섹션**이 불러다 쓰는 배너 묶음입니다. **그룹 키**가 같은 배너들이 하나의 캐러셀로 묶여 노출됩니다.

**연결 방법**

```
① 배너 등록 → 배너 타입 '프로모션 배너' → 배너 그룹 키 입력 (예: home_promo)
        ↓
② 페이지 빌더 → 프로모션 배너 섹션 추가 → '배너 그룹 키'에 같은 값 입력
        ↓
③ 두 값이 같아야 화면에 나옵니다
```

**배너 그룹 키 규칙**

- **영문으로 시작**하고 **영문·숫자·`_`·`-`** 만 쓸 수 있습니다. 최대 50자.
- `:` 은 쓸 수 없습니다(메뉴별·공통 배너 전용 이름과 겹치기 때문).
- 기존 그룹은 입력칸에서 목록으로 제시되므로 골라 쓰면 됩니다.

탭 안에서 그룹을 하나 골라 그 그룹의 배너만 봅니다. **+ 새 그룹** 을 누르면 그룹 키를 직접 입력해 새 묶음을 만들 수 있습니다.

> **배너는 등록했는데 페이지 빌더 섹션의 그룹 키가 다르면 아무것도 나오지 않습니다.** 두 값의 철자를 반드시 맞추세요. 반대로 섹션만 걸고 배너가 0건인 그룹도 목록에 표시되므로, "왜 안 나오지"를 바로 확인할 수 있습니다.

> **한 그룹 안에서는 이미지 비율을 통일하세요.** 섞이면 캐러셀이 고르지 않게 보입니다.

## 8. 배너 등록·수정 화면 (공통)

<div class="manual-goto-bar">
  <span class="manual-goto-label">이 단계 화면</span>
  <a class="manual-goto" href="/admin/banners/add" target="_blank">배너 등록 화면 열기</a>
</div>

### 8-1. 기본 입력란

| 입력란 | 필수 | 설명 |
|---|---|---|
| **배너 타입** | 필수 | `메인 슬라이더 (이미지 배너)` / `카테고리 배너` / `팝업 배너` / `브랜드 배너` / `메뉴별 배너` / `프로모션 배너`. **고른 타입에 따라 아래 칸이 달라집니다.** |
| **배너 제목** | 선택 | 관리용 이름입니다. 예: `봄맞이 할인 이벤트` |
| **이미지 업로드** | 신규 등록 시 필수 | PC용 이미지. 타입을 고르면 **권장 사이즈 안내**가 자동으로 표시됩니다. |
| **모바일용 이미지 업로드** | 선택 | **미등록 시 PC 이미지를 모바일에서도 씁니다.** 권장 390 × 422px |
| **연결 링크 (URL)** | 선택 | 내부 경로(`/products/view/1`) 또는 외부 주소 |
| **노출 순서** | — | 숫자가 작을수록 앞에 나옵니다. |
| **시작일 / 종료일** | 선택 | 비우면 기간 제한 없이 나옵니다. |
| **활성화 (즉시 노출)** | — | 끄면 나오지 않습니다. 새로 만들면 켜져 있습니다. |

### 8-2. 배너 내 문구 (메인 슬라이더 전용)

배너 타입이 **`메인 슬라이더 (이미지 배너)`** 일 때만 나타나는 영역입니다. 이미지 위에 글자와 버튼을 얹습니다. **비워 두면 이미지만 노출**됩니다.

| 입력란 | 설명 |
|---|---|
| **배너 제목 (큰 글자)** | 최대 120자 |
| **추가 문구 (최대 2줄)** | 한 줄에 하나씩. **3줄째부터는 저장되지 않습니다.** |
| **이동 버튼 문구** | 최대 40자. **비우면 버튼이 나오지 않습니다.** 버튼은 위의 **연결 링크**로 이동합니다. |
| **버튼 색상** | 색상 선택기 또는 `#1a7f37` 형태로 직접 입력. 글자색(흰/검정)은 배경 밝기에 따라 자동 선택됩니다. |
| **문구 정렬** | `왼쪽` / `가운데` / `오른쪽` |

> **다른 타입에서는 이 영역이 숨겨지고 저장되지도 않습니다.** 메인 슬라이더로 만들어 문구를 넣어 둔 배너를 나중에 다른 타입으로 바꾸면 **문구가 지워집니다.**

### 8-3. 권장 이미지 사이즈

등록 화면에서 배너 타입을 고르면 아래 안내가 자동으로 표시됩니다.

| 배너 타입 | 권장 크기 | 비율 |
|---|---|---|
| 메인 슬라이더 | 1920 × 600px | 3.2 : 1 |
| 카테고리 배너 | 900 × 200px | 약 4.5 : 1 |
| 브랜드 배너 | 900 × 200px | 약 4.5 : 1 |
| 팝업 배너 | 960 × 960px | 1 : 1 |
| 메뉴별 배너 | 1200 × 260px | 약 4.6 : 1 |
| 프로모션 배너 (가로형) | 800 × 300px | 16 : 6 |
| 프로모션 배너 (세로형) | 600 × 800px | 3 : 4 |
| 모바일 이미지 (공통) | 390 × 422px | — |

2배 해상도로 만들려면 각 수치를 2배로 잡으면 됩니다.

> **업로드 용량 제한을 넘으면 저장되지 않습니다.** 화면에 표시된 최대 용량을 확인하세요. 이미지 칸에는 이미지만, 영상 칸에는 mp4·webm 만 올릴 수 있습니다.

## 9. 자주 막히는 상황

**Q. 홈 최상단에 배너가 안 나옵니다.**
1. 메인 슬라이더 화면에서 **히어로 방식**이 내가 편집한 방식으로 `적용 중`인가?
2. 배너/슬라이드의 **활성화**가 켜져 있는가?
3. **기간**(시작일·종료일)이 오늘을 포함하는가?
4. 이미지 배너 방식이라면 **순서가 7번째 이후**여서 `노출 제외`가 되지 않았는가?
5. **편집 중인 몰**이 맞는가?

**Q. 상품을 안 골랐는데 낯선 상품이 홈에 나옵니다.**
중앙 슬라이더에 켜진 슬라이드가 하나도 없으면 **베스트 상위 7개가 자동으로** 들어갑니다. 직접 고르면 자동 노출이 멈춥니다.

**Q. 상품을 검색해도 결과에 안 나옵니다.**
이미 히어로 슬라이드로 등록된 상품은 후보에서 빠집니다. 기존 슬라이드를 확인하세요.

**Q. 상품을 여러 개 골랐는데 일부만 담겼습니다.**
슬롯 최대 개수(중앙 7개 / 우측 1개)를 넘겼습니다. **꺼 둔 슬라이드도 자리를 차지**하니 필요 없는 것은 삭제하세요.

**Q. 배너 순서를 바꿨는데 그대로입니다.**
목록의 **순서·노출 저장** 버튼을 눌러야 반영됩니다.

**Q. "이 몰의 사이트 설정이 아직 없습니다" 라고 나옵니다.**
히어로 방식·흐름문구는 사이트 설정 정보에 저장됩니다. 사이트 설정 화면에서 한 번 저장한 뒤 다시 시도하세요.

**Q. 카테고리 배너를 저장하려는데 오류가 납니다.**
대상 카테고리/브랜드를 고르지 않았습니다. 개별 대상을 고르거나 **전체 공통 배너로 설정**을 켜세요.

**Q. 프로모션 배너를 등록했는데 화면에 안 나옵니다.**
페이지 빌더의 프로모션 배너 섹션에 적힌 **배너 그룹 키**와 배너의 그룹 키가 정확히 같은지 확인하세요. 섹션 자체가 걸려 있는지도 봐야 합니다.

**Q. 배너에 넣은 제목·버튼이 화면에 안 보입니다.**
배너 내 문구는 **메인 슬라이더(이미지 배너) 타입에서만** 나옵니다. 다른 타입에서는 표시되지 않습니다.

**Q. 파일 업로드가 실패합니다.**
용량 초과이거나 형식이 맞지 않습니다. 이미지 칸에는 이미지, 영상 칸에는 mp4·webm 만 올릴 수 있습니다.

**Q. 메뉴별 배너 탭에 원하는 메뉴가 없습니다.**
그 메뉴가 꺼져 있습니다. 기능 메뉴 관리에서 켜세요.

## 10. 알아 두면 좋은 것

- **메인 슬라이더는 두 방식이 서로를 지우지 않습니다.** 새 시즌 배너를 미리 만들어 두고 오픈 날 방식만 전환하는 운영이 가능합니다.
- **톱바는 몰별로 저장됩니다.** 다른 배너와 저장되는 곳이 달라, 몰마다 다른 띠를 띄울 수 있습니다.
- **모바일 이미지는 선택입니다.** 등록하지 않으면 PC 이미지가 그대로 쓰이며, 가로형 이미지는 모바일에서 작게 보일 수 있습니다.
