# 배너 관리

쇼핑몰 **메인 화면, 카테고리 상단, 팝업** 등에 들어가는 **배너 이미지**를 등록·수정·삭제하는 메뉴입니다.

---

## 배너 종류

| 종류 | 어디에 보이나요 |
|------|------------------|
| **메인 배너** | 쇼핑몰 메인 페이지 상단의 넓은 이미지 영역 (슬라이드로 여러 장 노출될 수 있음) |
| **카테고리 배너** | 특정 카테고리 상품 목록 페이지 상단 |
| **브랜드 배너** | 특정 브랜드 상세·목록 페이지 상단 |
| **메뉴별 배너** | 선택한 상단(GNB) 메뉴 — 베스트·신상품 등 — 목록 페이지 상단 |
| **팝업 배너** | 메인 접속 시 뜨는 팝업 창 (기간·노출 설정이 있으면 그에 따라 표시) |

---

## 배너 목록 보기

- **배너 관리** 메뉴에 들어가면 등록된 배너 목록이 나옵니다.
- **타입(메인/카테고리/브랜드/메뉴/팝업)** 으로 필터를 걸어 볼 수 있습니다.
- 각 배너마다 **썸네일, 제목, 링크 주소, 사용 여부, 표시 순서** 등이 보입니다.
- **카테고리 배너·브랜드 배너** 탭은 대상(카테고리/브랜드) 목록 형태로 열려, 각 대상에 배너가 걸렸는지 한눈에 보여 줍니다. 목록 맨 위 **초록색 영역**에서 아래 설명하는 **전체 공통 배너**를 등록·확인할 수 있습니다.

---

## 배너 등록하기

1. **배너 추가** 또는 **등록** 버튼을 클릭합니다.
2. **배너 타입**을 선택합니다. (메인 / 카테고리 / 팝업)
3. **카테고리 배너**인 경우, 연결할 **카테고리**를 선택합니다.
4. **배너 제목**을 입력합니다. (관리용으로만 쓸 수도 있음)
5. **배너 이미지**를 업로드합니다.
   - **배너 이미지 (PC):** PC 화면에서 보일 기본 이미지입니다. (예: 1920×500)
   - **모바일용 배너 이미지:** 모바일 화면에서 보일 세로형 이미지입니다. (예: 800×1200) 등록하지 않으면 PC 이미지가 대신 보입니다.
6. **링크 URL:** 배너를 클릭했을 때 이동할 주소(상품 페이지, 이벤트 페이지 등)를 입력합니다. 없으면 비워 둘 수 있습니다.
7. **표시 순서:** 숫자가 작을수록 앞에 노출됩니다. (예: 1, 2, 3 순서)
8. **노출 기간:** 시작일과 종료일을 지정하면 해당 기간에만 배너가 자동으로 노출됩니다.
9. **저장** 버튼을 눌러 등록을 완료합니다.

---

## 전체 공통 카테고리/브랜드 배너

카테고리·브랜드 배너를 등록할 때 **"전체 공통 배너로 설정"** 을 체크하면, 그 배너는 **개별 배너를 따로 걸지 않은 모든 카테고리(또는 브랜드)** 페이지 상단에 **기본으로** 노출됩니다. 카테고리마다 배너를 일일이 만들지 않아도 되는 기본값입니다.

- 특정 카테고리/브랜드에 **개별 배너가 있으면 개별 배너가 우선**하고, 없을 때만 공통 배너가 대신 뜹니다.
- 체크하면 대상(카테고리/브랜드) 선택칸이 사라집니다 — 대상 하나가 아니라 몰 전체 기본값이기 때문입니다.
- 공통 배너는 **몰별로 따로** 관리됩니다(현재 선택한 몰에만 적용).
- 카테고리/브랜드 탭 목록 맨 위 초록색 영역의 **"+ 공통 배너 등록"** 으로도 바로 만들 수 있습니다.

---

## 배너 수정하기

1. 목록에서 해당 배너의 **수정** 버튼을 클릭합니다.
2. 이미지·링크·순서·기간 등을 바꾼 뒤 **저장**합니다.
3. 이미지를 바꾸지 않으려면 기존 이미지를 유지하면 됩니다.

---

## 배너 삭제하기

1. 목록에서 해당 배너의 **삭제** 버튼을 클릭합니다.
2. 확인 후 삭제하면 해당 배너는 더 이상 노출되지 않습니다.

**사용 중지만 하고 싶을 때:** 삭제 대신 "사용 중지" 또는 "비활성"으로 두면, 나중에 다시 켤 수 있습니다. (화면에 그런 옵션이 있는 경우)
