# 검색 로그

고객이 쇼핑몰 **검색창에 입력한 키워드**가 기록된 목록입니다. 어떤 단어를 많이 검색하는지, 결과가 없었던 검색은 무엇인지 확인할 수 있습니다.

---

## 이 메뉴로 무엇을 할 수 있나요?

- **검색어별 건수·마지막 검색 시각** 확인
- **기간을 정해서** 최근 24시간, 7일, 30일 등으로 필터링
- **검색했는데 결과가 0건**이었던 키워드 파악 → 상품 추가·상품명 수정 시 참고

---

## 화면 구성

- **기간 선택:** 조회할 기간(예: 최근 7일, 최근 30일)을 선택합니다.
- **목록:** 검색 키워드, 검색 횟수, 마지막 검색 일시, 결과가 0건인 횟수 등이 표시됩니다.

**활용 예**

- 인기 검색어 상위에 해당하는 상품이 없다면 → 그 키워드로 상품을 등록하거나 노출을 늘릴 수 있습니다.
- "검색 결과 없음"이 많은 키워드는 → 고객이 찾는 표현이므로 상품명·카테고리명에 반영하거나, 해당 상품을 추가하는 것을 검토할 수 있습니다.
