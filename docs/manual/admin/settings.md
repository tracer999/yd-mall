# 사이트 설정

쇼핑몰 **전체에 공통으로 적용되는 정보**와 **외부 서비스 연동**을 설정하는 메뉴입니다. **최초 설치 후 반드시 한 번씩 확인·수정**하는 것이 좋습니다.

---

## 설정이 두 가지로 나뉘는 경우

많은 시스템에서 **"회사 정보"** 와 **"시스템 설정"** 이 탭이나 메뉴로 나뉘어 있습니다.

| 구분 | 무엇을 설정하나요 |
|------|-------------------|
| **회사 정보** | 쇼핑몰에 보이는 회사명, 로고, 연락처, 주소, SNS 링크 등 |
| **시스템 설정** | 결제 키, Google/Kakao 로그인 키, OpenAI, 이메일 발송 등 (개발·연동용) |

---

## 회사 정보에서 설정하는 항목

- **회사명:** 쇼핑몰 상단·푸터에 표시되는 이름
- **로고:** 상단에 넣을 로고 이미지 (파일 업로드)
- **대표 전화번호, 이메일:** 고객센터·문의용
- **주소:** 회사 또는 사업장 주소
- **헤더 슬로건:** 상단 바에 작게 쓰는 문구 (예: "건강한 하루의 시작")
- **푸터 슬로건:** 하단에 쓰는 한 줄 문구
- **회사 소개:** "회사 소개" 페이지에 들어갈 본문
- **SNS 링크:** 인스타그램, 페이스북, 유튜브 등 사용 여부와 주소

저장하면 쇼핑몰 화면에 바로 반영됩니다.

---

## 시스템 설정에서 설정하는 항목 (개발자·관리자 협의)

다음은 **외부 서비스와 연동**할 때 필요한 값들입니다. 보통 개발자나 시스템 관리자가 알려 준 값을 입력합니다.

- **도메인:** 쇼핑몰 실제 주소 (예: https://myshop.com). 결제·로그인 콜백 주소에 쓰입니다.
- **토스페이먼츠(결제):** 클라이언트 키, 시크릿 키 (결제 연동용)
- **Google 로그인:** 클라이언트 ID, 시크릿 (소셜 로그인용)
- **Kakao 로그인:** REST API 키 등 (소셜 로그인용)
- **OpenAI:** API 키 (상품 AI 추천 문구 등 사용 시)
- **이메일(SMTP):** 메일 발송 서버 주소, 계정, 비밀번호 (관리자 인증 메일, 문의 답변 메일 등)

**주의:** 키·비밀번호는 외부에 노출되지 않도록 관리하고, 테스트 후 저장하세요. 잘못 입력하면 결제·로그인·메일 발송이 동작하지 않을 수 있습니다.

---

## 저장 방법

- 각 탭(회사 정보 / 시스템 설정)에서 **저장** 버튼을 누르면 해당 설정만 저장됩니다.
- 로고는 **파일 선택** 후 저장해야 반영됩니다.
