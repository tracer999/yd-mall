# 몰 관리

이 시스템은 **하나의 관리자에서 여러 개의 몰(쇼핑몰)** 을 운영할 수 있습니다. 몰 관리는 그 몰들을 만들고 고르는 메뉴입니다.

---

## 편집 중인 몰 고르기

관리자 화면 **상단(또는 설정)** 에서 **지금 편집 중인 몰**을 고릅니다. 메뉴·테마·홈 화면 같은 설정을 바꾸면 **선택된 몰에만 적용**됩니다.

**작업을 시작하기 전에 편집 중인 몰이 맞는지 반드시 확인하세요.** 다른 몰을 고른 채로 메뉴나 홈 화면을 고치면 엉뚱한 몰이 바뀝니다.

---

## 무엇이 몰마다 나뉘고, 무엇이 공용인가요?

| 몰마다 따로 관리 | 모든 몰이 함께 사용 |
|-------------------|----------------------|
| 상단 메뉴 구성 | 상품 |
| 테마(색상·디자인) | 카테고리 |
| 메인 화면(홈) 섹션 | 회원 |
| 배너 | 주문 |

**주의:** **주문·회원·상품은 몰별로 나뉘어 있지 않습니다.** 따라서 **대시보드와 매출 통계는 전체 기준**으로 나옵니다. "몰별 매출"을 보여 주는 화면이 아닙니다.

---

## 새 몰 만들기

1. **몰 관리** 화면에서 **몰 추가** 버튼을 클릭합니다.
2. **몰 이름**과 **몰 코드**를 입력합니다.
3. **디자인 프리셋**을 고릅니다. (아래 참고)
4. **저장**합니다.

저장하면 **바로 사용할 수 있는 상태의 몰**이 만들어집니다. 상단 메뉴, 테마, 메인 화면, 기본 데이터가 함께 생성되고 **홈 화면이 발행까지 완료**되므로, 만든 직후 고객 화면을 열어 보면 정상적으로 나옵니다.

---

## 디자인 프리셋 2종

| 프리셋 | 헤더 모양 |
|--------|-----------|
| **기본형** | 상단에 **카테고리 버튼이 따로** 있는 헤더 |
| **드로어형** | 카테고리가 **메뉴 안에 합쳐진** 헤더 |

---

## ⚠️ 이미 쓰던 몰에 프리셋을 다시 적용할 때

프리셋은 새 몰을 만들 때만 쓰는 것이 안전합니다. **이미 운영 중인 몰에 프리셋을 다시 적용하면 아래 일이 벌어집니다.**

- **메인 화면(홈)의 섹션이 지워질 수 있습니다.** 실행 전 확인 화면에 **삭제될 섹션 개수**가 표시됩니다. **반드시 읽고 진행하세요.**
- **아울렛 · 공동구매 · 쇼핑라이브 메뉴가 꺼집니다.** 재적용 후 **메뉴 관리에서 다시 켜야 합니다.**

되돌리기가 번거로우므로, 운영 중인 몰에는 프리셋 재적용 대신 **헤더 설정·테마 설정에서 개별 항목만 바꾸는 방법**을 권장합니다.
