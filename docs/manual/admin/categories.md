# 카테고리 관리

**카테고리**는 상품을 묶어 두는 **분류**입니다. 예: "건강식품", "선물세트", "간편식" 등. 쇼핑몰 메인·상품 목록에서 고객이 카테고리를 선택해 둘러볼 수 있도록 할 수 있습니다.

---

## 카테고리로 무엇을 하나요?

- 상품을 **분류**해서 고객이 찾기 쉽게 합니다.
- 카테고리 타입은 두 가지입니다 — **상품 목록·검색 필터**에 쓰이는 **일반(NORMAL)** 과, 브랜드관에 쓰이는 **브랜드(BRAND)**. (과거의 "테마" 타입은 폐기되어 더 이상 만들 수 없습니다.)

> ### 카테고리·브랜드는 전 몰이 공유하는 공용 분류입니다
>
> 어느 몰에서 만들든 **모든 몰이 함께 쓰는 한 벌**로 저장됩니다. 다른 몰에서도 같은 카테고리가 보이며, 각 몰 스토어프론트에는 **그 몰에 상품이 있는 카테고리만** 자동으로 노출됩니다(빈 카테고리는 숨김). 화면에 보이는 **상품 수와 "상품 있는 것만 노출" 필터는 현재 선택된 몰 기준**이고, 특정 몰에서만 카테고리를 숨기려면 목록의 **몰별 표시** 토글을 씁니다.

---

## 카테고리 목록 보기

- 관리자 메뉴에서 **카테고리 관리**를 클릭하면 **등록된 카테고리 목록**이 나옵니다.
- **표시 순서**대로 정렬되어 있으며, 순서를 바꾸면 쇼핑몰에 보이는 순서도 그에 맞게 바뀝니다.

---

## 카테고리 추가하기

1. **카테고리 관리** 화면에서 **추가** 또는 **등록** 버튼을 클릭합니다.
2. 다음 정보를 입력합니다.
   - **카테고리 이름:** 고객에게 보일 이름 (예: 건강식품)
   - **표시 순서:** 숫자가 작을수록 앞에 보입니다. (같으면 등록 순서)
   - **타입:** 일반(상품 분류용) / 브랜드 중 선택
3. **저장** 버튼을 눌러 등록합니다.

---

## 카테고리 수정하기

1. 목록에서 **수정** 버튼을 클릭합니다.
2. 이름·표시 순서·타입 등을 바꾼 뒤 **저장**합니다.

이미 상품이 연결된 카테고리도 이름·순서만 수정할 수 있습니다. 타입(일반↔브랜드)을 바꾸면 분류 성격이 바뀌므로 주의하세요.

---

## 카테고리 삭제하기

1. 목록에서 **삭제** 버튼을 클릭합니다.
2. 확인 메시지가 나오면 **확인**을 눌러 삭제합니다.

**주의:** 하위 카테고리가 있거나, **(다른 몰 것을 포함해) 이 카테고리에 연결된 상품이 있으면 삭제가 차단**됩니다. 카테고리·브랜드는 전 몰 공용이라, 한 몰에서 지워도 다른 몰의 상품이 조용히 끊기지 않도록 막는 것입니다. 먼저 하위 카테고리를 정리하고 상품을 다른 카테고리로 옮긴 뒤 삭제하세요.
