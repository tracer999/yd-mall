## 1. 시작하기 전에 — 꼭 알아야 할 3가지

### 1-1. 카테고리는 "모든 몰이 함께 쓰는 한 벌"입니다

이 앱은 몰을 여러 개 만드는 빌더입니다. 그런데 **카테고리(와 브랜드)만은 몰마다 따로 있지 않고, 전 몰이 공유하는 목록 하나**를 씁니다.

| 항목 | 적용 범위 |
|---|---|
| 카테고리 **자체** (이름 · 계층 · 이미지) | **전 몰 공통** |
| **상품** | 몰별로 따로 |
| 화면에 보이는 **상품수** | 지금 편집 중인 몰 기준 |
| **[사용 여부]** 체크 | **전 몰 공통** 스위치 |
| **[이 몰 표시]** 토글 | 지금 편집 중인 몰에서만 숨김/표시 |

> **이름을 고치면 다른 몰의 화면도 같이 바뀝니다.** "우리 몰에서만 이 분류를 감추고 싶다"면 이름을 바꾸거나 [사용 여부]를 끄지 말고, 반드시 **[이 몰 표시]를 끄세요.**

> **지금 편집 중인 몰을 확인하세요.** 상품 담기·상품수·[이 몰 표시]는 모두 **편집 몰 기준**입니다. 화면 상단의 편집 몰이 A몰인데 B몰 작업을 하고 있었다면 결과가 엉뚱한 곳에 남습니다.

### 1-2. 탭은 "무엇을 보여줄지"를 고르는 것입니다 (탭 2개)

| 탭 | 무엇이 보이나 | 화면 모양 |
|---|---|---|
| **사용중 카테고리** (기본) | **지금 편집 중인 몰에 상품이 담긴** 카테고리만 | 접었다 펴는 **나무 구조** |
| **미사용 포함 전체** | 상품이 하나도 없는 카테고리까지 전부 | **평평한 목록 + 경로** (한 페이지에 100건씩) |

탭 이름 옆 괄호 안 숫자가 각각의 건수입니다.

> **전체 탭이 평평한 목록인 이유.** 카테고리가 2천 개가 넘으면 나무 구조로 한 번에 그릴 때 화면이 몇십 초씩 멈춥니다. 그래서 전체 탭은 계층을 펼치지 않고 **경로(`건강식품 > 홍삼`)를 글자로** 보여주며 페이지를 나눕니다.

> **전체 탭에는 이미지·간략 설명·상위 카테고리 칸이 없습니다.** 그 칸들이 필요하면 **사용중** 탭이나 줄의 **[관리]** 화면에서 고치세요. 전체 탭에서 **[수정]**을 눌러도 그 값들은 그대로 보존됩니다.

> **"방금 만든 카테고리가 목록에 없어요"의 1순위 원인이 이것입니다.** 상품이 0개인 분류는 기본 탭(사용중)에 나오지 않습니다. **[미사용 포함 전체]** 탭에서 찾으세요. (자손에 상품이 있으면 경로 유지를 위해 조상은 사용중 탭에도 남습니다.)

### 1-3. 계층은 최대 3단계입니다

```
건강식품          ← 1단계 (최상위)
 └ 홍삼           ← 2단계
    └ 홍삼정      ← 3단계  (여기까지)
```

4단계를 만들려고 하면 `카테고리는 최대 3단계까지만 만들 수 있습니다.` 가 뜨고 저장되지 않습니다. 그래서 **상위 카테고리 선택 목록에는 3단계 항목이 아예 나오지 않습니다** — 그 아래에 하나 더 붙일 수 없기 때문입니다.

## 2. 화면 읽는 법

**[사용중 카테고리] 탭**의 목록은 **접었다 펴는 나무 구조**입니다. 처음 들어오면 **최상위(1단계)만 보이고**, 하위가 있는 줄 왼쪽에 `>` 화살표가 붙습니다. 화살표를 누르면 바로 아래 단계가 펼쳐지고, 다시 누르면 그 아래 전부가 함께 접힙니다.

이름 옆의 `D1` `D2` `D3` 은 **몇 단계인지**, `하위 2` 는 **지금 이 화면에서 펼쳐질 하위가 몇 개인지**를 뜻합니다.

**목록의 각 칸**

| 칸 | 뜻 |
|---|---|
| **순서** | 숫자가 작을수록 앞에 나옵니다. 같은 부모 아래에서만 비교됩니다. |
| **이미지** | **최상위 줄에만** 올릴 수 있습니다. (사용중 탭) |
| **카테고리명** | 고객에게 보이는 이름입니다. 칸에서 바로 고칠 수 있습니다. |
| **간략 설명** | **최상위 줄에만** 있습니다. 상단 대메뉴가 펼쳐질 때 이름 아래 한 줄로 나옵니다. (사용중 탭) |
| **상위 카테고리** | 이 항목을 누구 밑에 넣을지 고릅니다. `— 최상위 —` 를 고르면 맨 위 단계가 됩니다. (사용중 탭) |
| **경로** | 전체 탭에만 있습니다. 이 카테고리가 어디에 매달려 있는지를 `건강식품 > 홍삼` 처럼 보여줍니다. |
| **사용 여부** | **전 몰 공통** 스위치. 끄면 어디에서도 쓰이지 않습니다. |
| **이 몰 표시** | 지금 편집 중인 몰에서만 보이게/숨기게 합니다. **바꾸는 즉시 저장**됩니다. |
| **상품수** | 지금 편집 중인 몰에서 이 분류에 담긴 상품 수입니다. |
| **관리** | 상세 화면으로 이동합니다(상품 담기·기본 정보 편집). |

**줄마다 있는 버튼**

- **[관리]** — 상세 화면(상품 담기·기본 정보 편집)으로 이동합니다.
- **[수정]** — 그 줄에서 고친 값을 저장합니다. 저장되는 항목은 **탭에 따라 다릅니다**(아래).
- **[삭제]** — 하위가 있으면 회색으로 잠깁니다 (7장).

**[미사용 포함 전체] 탭은 조금 다릅니다**

이 탭은 나무 구조를 펼치지 않습니다. 화살표도 없고, 모든 줄이 처음부터 펼쳐진 채 **한 페이지에 100건씩** 나옵니다. 대신 **경로** 칸이 `건강식품 > 홍삼` 처럼 이 카테고리가 어디에 매달려 있는지 알려 줍니다.

목록 아래에 `카테고리 전체 2,348개 중 101–200번째` 처럼 지금 보고 있는 범위가 표시되고, 그 오른쪽 페이지 번호로 이동합니다. **탭을 옮겨도 검색·페이지는 각 탭이 따로 기억**합니다.

| 탭 | 그 줄에서 고칠 수 있는 것 |
|---|---|
| **사용중 카테고리** | 순서 · 이름 · 상위 카테고리 · 이미지 · 간략 설명 · 사용 여부 |
| **미사용 포함 전체** | 순서 · 이름 · 사용 여부 (상위·이미지·설명은 **그대로 보존**되며 칸이 없을 뿐입니다) |

**화면 위쪽 버튼**

- **[카테고리 등록]** — 새로 만들기(팝업).
- **[사용여부 저장]** — **지금 페이지에** 보이는 줄들의 **사용 여부만** 한 번에 저장합니다. 이름·순서·상위는 건드리지 않습니다.

> **[사용여부 저장]은 지금 보고 있는 페이지만 저장합니다.** 전체 탭에서 여러 페이지를 오가며 체크했다면, **페이지를 넘기기 전에** 저장하세요. 넘기는 순간 이전 페이지에서 바꾼 체크는 사라집니다.

**[등록]**을 누르면 목록으로 돌아옵니다. 상품이 아직 없으므로 **[미사용 포함 전체]** 탭에서 확인하세요.

## 4. 2단계 — 계층 만들기 (상위 지정)

이미 만들어진 항목을 다른 항목 밑으로 옮기려면, 그 줄의 **상위 카테고리** 칸에서 부모를 고르고 **그 줄의 [수정]**을 누릅니다.

**지켜지지 않으면 저장이 막히는 규칙 2가지**

| 규칙 | 어길 때 나오는 메시지 |
|---|---|
| **자기 자신이나 자기 하위**를 상위로 고를 수 없다 | `자기 자신이나 하위 카테고리를 상위로 지정할 수 없습니다.` |
| 옮긴 뒤 **3단계를 넘으면** 안 된다 | `카테고리는 최대 3단계까지만 만들 수 있습니다.` |

> 상위 후보 목록에는 **상품 카테고리만** 나옵니다. 브랜드는 이 화면에 없으므로 섞일 일이 없습니다.

> **두 번째 규칙은 자기 자신이 아니라 하위 전체에 적용됩니다.** 아래에 자식이 달린 카테고리를 2단계 자리로 옮기면 그 자식이 3단계, 손자가 4단계가 되어 **통째로 거부**됩니다. 먼저 자식을 정리한 뒤 옮기세요.

부모를 바꾸면 **하위 전체의 단계가 자동으로 다시 계산**됩니다. 하나씩 손보지 않아도 됩니다.

## 5. 3단계 — 상품 담기 (상세 화면)

목록에서 줄 오른쪽의 **[관리]**를 누르면 상세 화면이 열립니다. 여기서 **이 분류에 상품을 담고 빼냅니다.**

### 5-1. 왼쪽 — 기본 정보

이름 · 노출 순서 · 상위 카테고리 · 이미지 · 간략 설명 · 사용 여부를 고칠 수 있습니다.

> **고친 뒤 [기본 정보 저장]을 눌러야 반영됩니다.** 누르지 않고 화면을 벗어나면 변경이 사라집니다.

### 5-1-1. 네이버 스마트스토어 연결 (외부몰 연동을 쓰는 몰만)

기본 정보 카드 **바로 아래**에 **[네이버 스마트스토어 연결]** 카드가 있습니다. 이 카테고리의 상품을 네이버에 올리려면 **네이버 쪽 분류를 여기서 한 번 연결**해 두어야 합니다.

1. 검색창에 **두 글자 이상** 입력합니다 (예: `홍삼`, `티셔츠`).
2. 나오는 목록에서 고릅니다. **끝단 분류만 나옵니다.**
3. **[이 카테고리 상품에 바로 반영]** 을 체크하면 이 카테고리 상품에 즉시 채워집니다.
4. **[네이버 카테고리 저장]**

> **상품마다 지정할 필요가 없습니다.** 카테고리에 한 번 연결하면 소속 상품이 물려받습니다. 상품이 수천 건이어도 카테고리 수만큼만 하면 됩니다.

> **이미 지정된 상품은 덮어쓰지 않습니다.** 개별로 다르게 지정해 둔 상품은 그대로 둡니다.

> **끝단 분류가 아니면 저장되지 않습니다.** `네이버는 리프(최하위)만 받습니다` 가 뜨면 더 깊은 분류를 고르세요. 여기서 막지 않으면 나중에 **상품 수백 건이 등록 단계에서 한꺼번에 실패**합니다.

> **고시 유형은 고르지 않습니다.** 연결한 네이버 분류가 정하며, 카드에 무엇으로 정해졌는지(예: `의류`) 표시만 됩니다.

> **검색 결과가 없다면** `외부몰 연동 → 네이버 리소스 관리`에서 카테고리를 먼저 수집하세요.

### 5-2. 오른쪽 — 이 카테고리의 상품

지금 편집 중인 몰의 상품만 나옵니다. 몰 이름과 총 개수가 제목 옆에 함께 표시됩니다.

**상품을 담는 순서**

1. **[상품 추가]**를 누릅니다. (버튼 옆의 `미설정 N` 은 아직 이 분류가 지정되지 않은 상품 수입니다.)
2. 팝업에서 조건을 넣고 **[검색]** 합니다.

   | 검색 조건 | 고를 수 있는 값 |
   |---|---|
   | **검색어** | 상품명 또는 상품코드 |
   | **재고** | 전체 / 재고 있음 / 재고 없음 |
   | **노출** | 전체 / 노출 / 숨김 / 회원전용 |

3. 담을 상품을 체크합니다. **[결과 전체 선택]** 으로 한 번에 고를 수 있습니다.
4. **[선택 배정]**을 누릅니다.

빼낼 때는 상품 줄 오른쪽의 **[제거]**를 누릅니다. **연결만 끊길 뿐 상품 자체는 삭제되지 않습니다.**

> **팝업에는 "아직 이 분류가 비어 있는 상품"만 나옵니다.** 이미 다른 카테고리에 속한 상품은 검색해도 나오지 않습니다. 실수로 남의 분류에서 상품을 빼앗아 오는 사고를 막는 장치입니다. **옮기려면 원래 분류에서 [제거]한 뒤 다시 담으세요.**

> 검색 결과는 **최대 100건**까지만 나옵니다. `결과가 100개에 도달했습니다` 안내가 뜨면 검색어로 범위를 좁히세요.

  <span class="manual-goto-label">담을 상품이 없다면</span>
  <a class="manual-goto is-sub" href="/admin/products" target="_blank">상품 관리에서 먼저 등록</a>
</div>

## 6. 사용 여부 · 이 몰 표시 — 가장 헷갈리는 두 스위치

두 개는 **완전히 다른 스위치**입니다.

| | 사용 여부 | 이 몰 표시 |
|---|---|---|
| 영향 범위 | **전 몰** | **지금 편집 중인 몰만** |
| 끄면 | 이 분류를 어디에서도 쓰지 않습니다 | 이 몰의 고객 화면 메뉴에서만 빠집니다 |
| 저장 방법 | 줄의 **[수정]** 또는 **[사용여부 저장]** | 체크를 바꾸는 **즉시 저장** |
| 쓸 수 있는 조건 | 언제나 | **이 몰에 상품이 있는 항목만** (없으면 `—` 로 표시) |

> **"다른 몰에서 이 카테고리가 사라졌어요"** — [사용 여부]를 끈 것입니다. 특정 몰에서만 감추려면 반드시 **[이 몰 표시]**를 쓰세요.

**여러 건의 사용 여부를 한 번에 바꿀 때**는 체크만 바꾼 뒤 화면 위의 **[사용여부 저장]**을 누릅니다.

> **[사용여부 저장]은 사용 여부만 저장합니다.** 같이 고쳐 놓은 **이름·순서·상위는 저장되지 않습니다.** 그것들은 줄마다 **[수정]**을 눌러야 합니다.

---

## 7. 삭제하기 (막히는 경우가 많습니다)

줄 오른쪽의 **[삭제]**를 누릅니다. 다만 아래 두 경우에는 **삭제가 거부**됩니다.

| 상황 | 메시지 | 해결 |
|---|---|---|
| **하위가 있다** | `하위 카테고리 N개가 있어 삭제할 수 없습니다.` | 하위를 먼저 다른 곳으로 옮기거나 지웁니다. |
| **상품이 참조 중이다** | `이 카테고리를 참조하는 상품이 (다른 몰 포함) N개 있어 삭제할 수 없습니다.` | [관리] 화면에서 상품을 **[제거]**한 뒤 다시 시도합니다. |

하위가 있는 줄은 **[삭제] 버튼 자체가 회색으로 잠겨** 있습니다.

> **"다른 몰 포함"에 주의하세요.** 화면의 상품수가 `0` 이어도 **다른 몰의 상품이 이 분류를 쓰고 있으면** 삭제되지 않습니다. 상품수는 편집 중인 몰만 세기 때문입니다. 고장이 아니라 정상 동작입니다.

## 9. 자주 막히는 상황

**Q. 카테고리를 만들었는데 목록에 없습니다.**
상품이 0개라 기본 탭(**사용중 카테고리**)에서 숨겨진 것입니다. **[미사용 포함 전체]** 탭을 누르세요.

**Q. 하위를 만들려는데 상위 목록에 원하는 항목이 없습니다.**
그 항목이 **이미 3단계**라 아래에 더 붙일 수 없는 것입니다.

**Q. 전체 탭에서 [수정]을 눌러도 이미지·설명이 지워지지 않나요?**
지워지지 않습니다. 전체 탭에는 그 칸이 없을 뿐, 저장할 때 기존 값을 그대로 함께 보냅니다.

**Q. 브랜드는 어디서 관리하나요?**
**브랜드 관리**(`/admin/brands`) 화면입니다. 만들기·순서·로고·입점일·삭제 전부 그곳에서 합니다.

**Q. 이름을 고쳤는데 저장되지 않습니다.**
그 줄의 **[수정]**을 눌러야 저장됩니다. 화면 위의 **[사용여부 저장]**은 사용 여부만 저장합니다.

**Q. 삭제가 안 됩니다.**
하위가 있거나, 상품이 이 분류를 쓰고 있습니다. 위 7장을 확인하세요. 상품수가 0인데도 막히면 **다른 몰의 상품**이 쓰고 있는 것입니다.

**Q. 상품 추가 팝업에서 원하는 상품이 검색되지 않습니다.**
그 상품은 **이미 다른 카테고리에 속해 있습니다.** 팝업에는 미설정 상품만 나옵니다. 원래 분류에서 먼저 **[제거]**하세요.

**Q. 카테고리를 만들었는데 고객 상단 메뉴에 안 보입니다.**
① **사용 여부**가 켜져 있는지, ② **[이 몰 표시]**가 표시로 되어 있는지, ③ 이 몰에 **상품이 담겨 있는지**, ④ **메뉴 미리보기**에서 순서·노출이 켜져 있는지를 차례로 확인하세요.

**Q. 다른 몰의 카테고리 이름이 같이 바뀌었습니다.**
정상입니다. 카테고리는 전 몰 공통 한 벌입니다. 몰마다 다르게 하려면 **[이 몰 표시]**로 감추는 방법만 있습니다.

**Q. 이미지 올리는 칸이 안 보입니다.**
① **최상위(1단계)에만** 이미지와 간략 설명이 있습니다. ② **[미사용 포함 전체]** 탭에는 이미지 칸 자체가 없습니다 — 사용중 탭이나 [관리] 화면에서 올리세요.

**Q. [이 몰 표시] 칸이 `—` 로 나와 누를 수 없습니다.**
이 몰에 그 분류의 상품이 하나도 없다는 뜻입니다. 어차피 고객 화면에 안 나오므로 토글할 대상이 아닙니다. 상품을 먼저 담으세요.

## 10. 아직 지원하지 않는 것

- 엑셀/CSV로 카테고리 일괄 등록
- 하나의 상품을 여러 카테고리에 동시에 담기 (상품 하나당 **카테고리 1개 + 브랜드 1개**가 최대입니다)
- **미사용 포함 전체** 탭에서의 계층 편집 (사용중 탭이나 [관리] 화면에서 하세요)
- 몰마다 카테고리 이름을 다르게 쓰기 (감추는 것만 가능)
- 중복 카테고리 병합 (상품을 옮긴 뒤 한쪽을 지워야 합니다)
