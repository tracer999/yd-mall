# 카테고리 관리

**카테고리**는 상품을 묶어 두는 **분류**입니다. 예: "건강식품", "선물세트", "간편식" 등. 쇼핑몰 메인·상품 목록에서 고객이 카테고리를 선택해 둘러볼 수 있도록 할 수 있습니다.

---

## 카테고리로 무엇을 하나요?

- 상품을 **분류**해서 고객이 찾기 쉽게 합니다.
- **메인 화면·상단 메뉴**에 노출되는 "테마 카테고리"와, **상품 목록 필터**에만 쓰이는 "일반 카테고리"를 구분해서 쓸 수 있습니다.

---

## 카테고리 목록 보기

- 관리자 메뉴에서 **카테고리 관리**를 클릭하면 **등록된 카테고리 목록**이 나옵니다.
- **표시 순서**대로 정렬되어 있으며, 순서를 바꾸면 쇼핑몰에 보이는 순서도 그에 맞게 바뀝니다.

---

## 카테고리 추가하기

1. **카테고리 관리** 화면에서 **추가** 또는 **등록** 버튼을 클릭합니다.
2. 다음 정보를 입력합니다.
   - **카테고리 이름:** 고객에게 보일 이름 (예: 건강식품)
   - **표시 순서:** 숫자가 작을수록 앞에 보입니다. (같으면 등록 순서)
   - **타입:** 테마(메뉴에 노출) / 일반(상품 분류용) 중 선택
3. **저장** 버튼을 눌러 등록합니다.

---

## 카테고리 수정하기

1. 목록에서 **수정** 버튼을 클릭합니다.
2. 이름·표시 순서·타입 등을 바꾼 뒤 **저장**합니다.

이미 상품이 연결된 카테고리도 이름·순서만 수정할 수 있습니다. 타입을 바꾸면 메뉴 노출 여부가 바뀌므로 주의하세요.

---

## 카테고리 삭제하기

1. 목록에서 **삭제** 버튼을 클릭합니다.
2. 확인 메시지가 나오면 **확인**을 눌러 삭제합니다.

**주의:** 해당 카테고리에 연결된 상품이 있으면 삭제가 막기거나, 상품을 먼저 다른 카테고리로 옮겨야 할 수 있습니다. 시스템에 따라 다르므로 삭제 전에 한 번 확인하세요.
