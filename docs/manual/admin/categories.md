# 카테고리 관리

쇼핑몰의 **상품 분류 체계**를 만드는 곳입니다. 고객이 상단 메뉴에서 "건강식품 › 홍삼 › 홍삼정"처럼 타고 들어가는 그 분류를 여기서 만듭니다. 같은 화면에서 **브랜드**도 함께 만듭니다 — 이 몰에서 브랜드는 "브랜드 전용 카테고리"로 저장되기 때문입니다.

## 1. 시작하기 전에 — 꼭 알아야 할 3가지

### 1-1. 카테고리와 브랜드는 "모든 몰이 함께 쓰는 한 벌"입니다

이 앱은 몰을 여러 개 만드는 빌더입니다. 그런데 **카테고리와 브랜드만은 몰마다 따로 있지 않고, 전 몰이 공유하는 목록 하나**를 씁니다.

| 항목 | 적용 범위 |
|---|---|
| 카테고리·브랜드 **자체** (이름 · 계층 · 로고) | **전 몰 공통** |
| **상품** | 몰별로 따로 |
| 화면에 보이는 **상품수** | 지금 편집 중인 몰 기준 |
| **[사용 여부]** 체크 | **전 몰 공통** 스위치 |
| **[이 몰 표시]** 토글 | 지금 편집 중인 몰에서만 숨김/표시 |

> **이름을 고치면 다른 몰의 화면도 같이 바뀝니다.** "우리 몰에서만 이 분류를 감추고 싶다"면 이름을 바꾸거나 [사용 여부]를 끄지 말고, 반드시 **[이 몰 표시]를 끄세요.**

> **지금 편집 중인 몰을 확인하세요.** 상품 담기·상품수·[이 몰 표시]는 모두 **편집 몰 기준**입니다. 화면 상단의 편집 몰이 A몰인데 B몰 작업을 하고 있었다면 결과가 엉뚱한 곳에 남습니다.

### 1-2. 분류는 두 갈래입니다 (탭 2개)

| 탭 | 무엇을 담나 | 특징 |
|---|---|---|
| **상품 카테고리** | 건강식품·생활용품 같은 **상품 분류** | 최상위 단계에만 이미지·간략 설명을 넣을 수 있습니다 |
| **브랜드 카테고리** | 정관장·종근당 같은 **브랜드** | **입점일**을 관리합니다. [관리]를 누르면 브랜드 관리 화면으로 넘어갑니다 |

> **두 갈래는 서로 섞이지 않습니다.** 상품 카테고리를 브랜드의 상위로 지정할 수 없고 반대도 안 됩니다. 상위 선택 목록에는 **같은 탭의 항목만** 나옵니다.

### 1-3. 계층은 최대 3단계입니다

```
건강식품          ← 1단계 (최상위)
 └ 홍삼           ← 2단계
    └ 홍삼정      ← 3단계  (여기까지)
```

4단계를 만들려고 하면 `카테고리는 최대 3단계까지만 만들 수 있습니다.` 가 뜨고 저장되지 않습니다. 그래서 **상위 카테고리 선택 목록에는 3단계 항목이 아예 나오지 않습니다** — 그 아래에 하나 더 붙일 수 없기 때문입니다.

## 2. 화면 읽는 법

목록은 **접었다 펴는 나무 구조**입니다. 처음 들어오면 **최상위(1단계)만 보이고**, 하위가 있는 줄 왼쪽에 `>` 화살표가 붙습니다. 화살표를 누르면 바로 아래 단계가 펼쳐지고, 다시 누르면 그 아래 전부가 함께 접힙니다.

이름 옆의 `D1` `D2` `D3` 은 **몇 단계인지**, `하위 2` 는 **지금 이 화면에서 펼쳐질 하위가 몇 개인지**를 뜻합니다.

**목록의 각 칸**

| 칸 | 뜻 |
|---|---|
| **순서** | 숫자가 작을수록 앞에 나옵니다. 같은 부모 아래에서만 비교됩니다. |
| **이미지 / 로고** | 상품 카테고리는 최상위 줄에만, 브랜드는 모든 줄에 올릴 수 있습니다. |
| **카테고리명 / 브랜드명** | 고객에게 보이는 이름입니다. 칸에서 바로 고칠 수 있습니다. |
| **간략 설명** | 상품 카테고리 **최상위 줄에만** 있습니다. 상단 대메뉴가 펼쳐질 때 이름 아래 한 줄로 나옵니다. |
| **입점일** | 브랜드 탭에만 있습니다. 이 날짜부터 일정 기간 '신규 입점 브랜드'로 취급됩니다. |
| **상위 카테고리** | 이 항목을 누구 밑에 넣을지 고릅니다. `— 최상위 —` 를 고르면 맨 위 단계가 됩니다. |
| **사용 여부** | **전 몰 공통** 스위치. 끄면 어디에서도 쓰이지 않습니다. |
| **이 몰 표시** | 지금 편집 중인 몰에서만 보이게/숨기게 합니다. **바꾸는 즉시 저장**됩니다. |
| **상품수** | 지금 편집 중인 몰에서 이 분류에 담긴 상품 수입니다. |
| **관리** | 상세 화면으로 이동합니다(상품 담기·기본 정보 편집). |

**화면 위쪽 버튼**

- **[카테고리 등록]** — 새로 만들기(팝업). 지금 열려 있는 탭에 만들어집니다.
- **[사용여부 저장]** — 화면에 보이는 줄들의 **사용 여부만** 한 번에 저장합니다. 이름·순서·상위는 건드리지 않습니다.
- **[빈 항목 포함 보기] / [상품 있는 것만 보기]** — 기본은 **상품이 있는 것만** 보여 줍니다.

> **"방금 만든 카테고리가 목록에 없어요"의 1순위 원인이 이것입니다.** 상품이 0개인 분류는 기본 화면에서 숨겨집니다. 오른쪽 위 **[빈 항목 포함 보기]**를 누르세요. (자손에 상품이 있으면 경로 유지를 위해 조상은 남습니다.)

## 3. 1단계 — 카테고리 만들기

<div class="manual-goto-bar">
  <span class="manual-goto-label">이 단계 화면</span>
  <a class="manual-goto" href="/admin/categories" target="_blank">카테고리 관리 열기</a>

만들 탭(**상품 카테고리** 또는 **브랜드 카테고리**)을 먼저 고른 뒤 **[카테고리 등록]**을 누릅니다.

| 입력란 | 필수 | 설명 | 예시 |
|---|---|---|---|
| **카테고리명** | 필수 | 고객에게 보이는 이름입니다. | `건강식품` |
| **간략 설명** | 선택 | 상품 카테고리를 **최상위로 만들 때만** 나타납니다. 대메뉴에서 이름 아래 한 줄로 표시됩니다. 최대 255자. | `매일 챙기는 건강 습관` |
| **입점일** | 선택 | **브랜드 탭에서만** 나타납니다. 기본값은 오늘이며, 이 날짜부터 일정 기간 '신규 입점 브랜드'로 노출됩니다. | `2026-07-20` |
| **상위 카테고리** | 선택 | 비워 두면(`— 최상위 —`) 1단계가 됩니다. 3단계 항목은 목록에 나오지 않습니다. | `건강식품` |
| **이미지 / 로고 업로드** | 선택 | 정사각형 권장, 최대 2MB. jpg·png·webp. | — |
| **출력 순서** | 선택 | 비워 두면 맨 뒤로 붙습니다. | `3` |
| **활성 / PC 노출 / 모바일 노출** | — | 기본값은 셋 다 켜짐입니다. | — |

> **상품 카테고리에서 상위를 고르는 순간 이미지·간략 설명 칸이 사라집니다.** 이 둘은 대메뉴 안내 영역 전용이라 **최상위에서만** 쓰기 때문입니다. 브랜드 로고는 단계와 무관하게 언제나 올릴 수 있습니다.

**[등록]**을 누르면 목록으로 돌아옵니다. 상품이 아직 없으므로 **[빈 항목 포함 보기]**로 확인하세요.

## 4. 2단계 — 계층 만들기 (상위 지정)

이미 만들어진 항목을 다른 항목 밑으로 옮기려면, 그 줄의 **상위 카테고리** 칸에서 부모를 고르고 **그 줄의 [수정]**을 누릅니다.

**지켜지지 않으면 저장이 막히는 규칙 3가지**

| 규칙 | 어길 때 나오는 메시지 |
|---|---|
| **같은 탭 안에서만** 상위를 고를 수 있다 | `상위 카테고리는 같은 분류(일반/테마/브랜드) 안에서만 지정할 수 있습니다.` |
| **자기 자신이나 자기 하위**를 상위로 고를 수 없다 | `자기 자신이나 하위 카테고리를 상위로 지정할 수 없습니다.` |
| 옮긴 뒤 **3단계를 넘으면** 안 된다 | `카테고리는 최대 3단계까지만 만들 수 있습니다.` |

> **세 번째 규칙은 자기 자신이 아니라 하위 전체에 적용됩니다.** 아래에 자식이 달린 카테고리를 2단계 자리로 옮기면 그 자식이 3단계, 손자가 4단계가 되어 **통째로 거부**됩니다. 먼저 자식을 정리한 뒤 옮기세요.

부모를 바꾸면 **하위 전체의 단계가 자동으로 다시 계산**됩니다. 하나씩 손보지 않아도 됩니다.

## 5. 3단계 — 상품 담기 (상세 화면)

목록에서 줄 오른쪽의 **[관리]**를 누르면 상세 화면이 열립니다. 여기서 **이 분류에 상품을 담고 빼냅니다.**

> **브랜드 탭의 [관리]는 브랜드 관리 화면으로 갑니다.** 브랜드는 영문명·스토리 같은 전용 항목이 많아 화면을 따로 씁니다. 상품을 담고 빼는 방식은 똑같습니다.

### 5-1. 왼쪽 — 기본 정보

이름 · 노출 순서 · 상위 카테고리 · 로고 이미지 · 간략 설명 · 입점일 · 사용 여부를 고칠 수 있습니다.

> **고친 뒤 [기본 정보 저장]을 눌러야 반영됩니다.** 누르지 않고 화면을 벗어나면 변경이 사라집니다.

### 5-2. 오른쪽 — 이 카테고리의 상품

지금 편집 중인 몰의 상품만 나옵니다. 몰 이름과 총 개수가 제목 옆에 함께 표시됩니다.

**상품을 담는 순서**

1. **[상품 추가]**를 누릅니다. (버튼 옆의 `미설정 N` 은 아직 이 분류가 지정되지 않은 상품 수입니다.)
2. 팝업에서 조건을 넣고 **[검색]** 합니다.

   | 검색 조건 | 고를 수 있는 값 |
   |---|---|
   | **검색어** | 상품명 또는 상품코드 |
   | **재고** | 전체 / 재고 있음 / 재고 없음 |
   | **노출** | 전체 / 노출 / 숨김 / 회원전용 |

3. 담을 상품을 체크합니다. **[결과 전체 선택]** 으로 한 번에 고를 수 있습니다.
4. **[선택 배정]**을 누릅니다.

빼낼 때는 상품 줄 오른쪽의 **[제거]**를 누릅니다. **연결만 끊길 뿐 상품 자체는 삭제되지 않습니다.**

> **팝업에는 "아직 이 분류가 비어 있는 상품"만 나옵니다.** 이미 다른 카테고리에 속한 상품은 검색해도 나오지 않습니다. 실수로 남의 분류에서 상품을 빼앗아 오는 사고를 막는 장치입니다. **옮기려면 원래 분류에서 [제거]한 뒤 다시 담으세요.**

> 검색 결과는 **최대 100건**까지만 나옵니다. `결과가 100개에 도달했습니다` 안내가 뜨면 검색어로 범위를 좁히세요.

## 6. 사용 여부 · 이 몰 표시 — 가장 헷갈리는 두 스위치

두 개는 **완전히 다른 스위치**입니다.

| | 사용 여부 | 이 몰 표시 |
|---|---|---|
| 영향 범위 | **전 몰** | **지금 편집 중인 몰만** |
| 끄면 | 이 분류를 어디에서도 쓰지 않습니다 | 이 몰의 고객 화면 메뉴에서만 빠집니다 |
| 저장 방법 | 줄의 **[수정]** 또는 **[사용여부 저장]** | 체크를 바꾸는 **즉시 저장** |
| 쓸 수 있는 조건 | 언제나 | **이 몰에 상품이 있는 항목만** (없으면 `—` 로 표시) |

> **"다른 몰에서 이 카테고리가 사라졌어요"** — [사용 여부]를 끈 것입니다. 특정 몰에서만 감추려면 반드시 **[이 몰 표시]**를 쓰세요.

**여러 건의 사용 여부를 한 번에 바꿀 때**는 체크만 바꾼 뒤 화면 위의 **[사용여부 저장]**을 누릅니다.

> **[사용여부 저장]은 사용 여부만 저장합니다.** 같이 고쳐 놓은 **이름·순서·상위는 저장되지 않습니다.** 그것들은 줄마다 **[수정]**을 눌러야 합니다.

## 7. 삭제하기 (막히는 경우가 많습니다)

줄 오른쪽의 **[삭제]**를 누릅니다. 다만 아래 두 경우에는 **삭제가 거부**됩니다.

| 상황 | 메시지 | 해결 |
|---|---|---|
| **하위가 있다** | `하위 카테고리 N개가 있어 삭제할 수 없습니다.` | 하위를 먼저 다른 곳으로 옮기거나 지웁니다. |
| **상품이 참조 중이다** | `이 카테고리를 참조하는 상품이 (다른 몰 포함) N개 있어 삭제할 수 없습니다.` | 상세 화면에서 상품을 **[제거]**한 뒤 다시 시도합니다. |

하위가 있는 줄은 **[삭제] 버튼 자체가 회색으로 잠겨** 있습니다.

> **"다른 몰 포함"에 주의하세요.** 화면의 상품수가 `0` 이어도 **다른 몰의 상품이 이 분류를 쓰고 있으면** 삭제되지 않습니다. 상품수는 편집 중인 몰만 세기 때문입니다. 고장이 아니라 정상 동작입니다.

## 9. 자주 막히는 상황

**Q. 카테고리를 만들었는데 목록에 없습니다.**
상품이 0개라서 숨겨진 것입니다. 오른쪽 위 **[빈 항목 포함 보기]**를 누르세요. 그래도 없다면 **다른 탭**(상품 ↔ 브랜드)에 만들었을 수 있습니다.

**Q. 하위를 만들려는데 상위 목록에 원하는 항목이 없습니다.**
① 그 항목이 **이미 3단계**라 아래에 더 붙일 수 없거나, ② **다른 탭**의 항목이라 나오지 않는 것입니다.

**Q. 이름을 고쳤는데 저장되지 않습니다.**
그 줄의 **[수정]**을 눌러야 저장됩니다. 화면 위의 **[사용여부 저장]**은 사용 여부만 저장합니다.

**Q. 삭제가 안 됩니다.**
하위가 있거나, 상품이 이 분류를 쓰고 있습니다. 위 7장을 확인하세요. 상품수가 0인데도 막히면 **다른 몰의 상품**이 쓰고 있는 것입니다.

**Q. 상품 추가 팝업에서 원하는 상품이 검색되지 않습니다.**
그 상품은 **이미 다른 카테고리(또는 브랜드)에 속해 있습니다.** 팝업에는 미설정 상품만 나옵니다. 원래 분류에서 먼저 **[제거]**하세요.

**Q. 카테고리를 만들었는데 고객 상단 메뉴에 안 보입니다.**
① **사용 여부**가 켜져 있는지, ② **[이 몰 표시]**가 표시로 되어 있는지, ③ 이 몰에 **상품이 담겨 있는지**, ④ **메뉴 미리보기**에서 순서·노출이 켜져 있는지를 차례로 확인하세요.

**Q. 다른 몰의 카테고리 이름이 같이 바뀌었습니다.**
정상입니다. 카테고리·브랜드는 전 몰 공통 한 벌입니다. 몰마다 다르게 하려면 **[이 몰 표시]**로 감추는 방법만 있습니다.

**Q. 상품 카테고리인데 이미지 올리는 칸이 안 보입니다.**
상품 카테고리는 **최상위(1단계)에만** 이미지와 간략 설명이 있습니다. 브랜드는 단계와 무관하게 로고를 올릴 수 있습니다.

**Q. [이 몰 표시] 칸이 `—` 로 나와 누를 수 없습니다.**
이 몰에 그 분류의 상품이 하나도 없다는 뜻입니다. 어차피 고객 화면에 안 나오므로 토글할 대상이 아닙니다. 상품을 먼저 담으세요.

---

## 10. 아직 지원하지 않는 것

- 엑셀/CSV로 카테고리 일괄 등록
- 하나의 상품을 여러 카테고리에 동시에 담기 (상품 하나당 **카테고리 1개 + 브랜드 1개**가 최대입니다)
- 몰마다 카테고리 이름을 다르게 쓰기 (감추는 것만 가능)
- 중복 카테고리 병합 (상품을 옮긴 뒤 한쪽을 지워야 합니다)
