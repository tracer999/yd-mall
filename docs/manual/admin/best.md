# 베스트 · 랭킹 관리

고객 화면의 **베스트/랭킹** 페이지에 보이는 **인기 상품 순위**를 관리하는 곳입니다.

순위는 운영자가 하나하나 줄 세우는 것이 아니라 **판매·좋아요·조회를 점수로 합산해 시스템이 자동으로 매깁니다.** 운영자가 하는 일은 세 가지입니다.

1. 어떤 **탭**으로 순위를 보여 줄지 정한다 (전체 / 카테고리별 / 브랜드별)
2. 점수를 매기는 **가중치**와 **집계 주기**를 정한다
3. 꼭 밀고 싶은 상품을 **MD 픽**으로 위에 고정한다

> **자동 순위는 직접 고칠 수 없습니다.** 순위는 집계가 만든 결과라서, 화면에서 손으로 끌어 옮기는 기능이 없습니다. 순위를 바꾸고 싶으면 **MD 픽(수동 고정)** 을 쓰세요. (6장)

## 1. 시작하기 전에 — 준비물

| 준비물 | 왜 필요한가 | 어디서 만드나 |
|---|---|---|
| **판매 이력이 있는 상품** | 점수는 판매·좋아요·조회에서 나옵니다. 아무 데이터가 없으면 순위가 거의 무의미해집니다. | 상품 관리 · 판매 관리 |
| **카테고리** (카테고리 탭을 만들 때) | 카테고리 탭은 이미 있는 카테고리를 골라 만듭니다. | 카테고리 관리 |
| **브랜드** (브랜드 탭을 만들 때) | 브랜드 탭도 마찬가지입니다. | 브랜드 관리 |

> **지금 편집 중인 몰을 반드시 확인하세요.** 탭·가중치·MD 픽은 모두 **몰별로 따로 저장**됩니다. 다른 몰의 상품은 이 몰의 탭에 고정할 수 없습니다.
> 다만 **집계 스케줄(주기)만은 몰 공통**입니다. 여기서 바꾸면 다른 몰에도 함께 적용됩니다.

## 2. 전체 흐름 한눈에 보기

```
① 탭(그룹) 추가        (전체 / 카테고리 / 브랜드)
        ↓
② 점수 가중치 확인·저장
        ↓
③ [지금 집계] 실행     ← 이걸 해야 순위가 생깁니다
        ↓
④ 탭 상세에서 미리보기 확인
        ↓
⑤ (필요하면) MD 픽으로 상품 고정
        ↓
⑥ 집계 스케줄을 켜서 이후 자동 갱신
```

**탭을 만든 직후에는 순위가 비어 있습니다.** 목록에 `미집계` 로 표시되며, **[지금 집계]** 를 눌러야 채워집니다.

---

## 3. 순위가 매겨지는 방식

점수 계산식은 이렇습니다.

```
점수 = 판매수량 × 판매점수 + 좋아요 × 좋아요점수 + 조회수 × 조회점수
```

- 매출로 인정하는 주문 상태는 **결제완료 · 상품준비중 · 배송중 · 배송완료** 입니다. 미결제·취소·환불은 점수에 들어가지 않습니다.
- 점수가 같으면 **누적 조회수**가 많은 상품이 앞에 옵니다.
- 전시 가능한 상태(`판매중`·`품절`·`판매예정`·`재입고예정`)의 상품만 순위에 들어갑니다.

**기간별로 따로 집계됩니다.**

| 기간 탭 | 무엇을 보나 |
|---|---|
| **실시간** | 최근 1시간 |
| **일간** | 최근 24시간 |
| **주간** | 최근 7일 |
| **월간** | 최근 30일 |

## 4. 점수 가중치 설정

  <span class="manual-goto-label">이 단계 화면</span>
  <a class="manual-goto" href="/admin/best-groups" target="_blank">베스트/랭킹 관리 열기</a>

**점수 가중치** 영역에서 값을 넣고 **[가중치 저장]** 을 누릅니다.

| 입력란 | 설명 | 기본값 |
|---|---|---|
| **판매 1건** | 판매 1개당 몇 점을 줄지 | 5 |
| **좋아요 1건** | 좋아요 1개당 몇 점을 줄지 | 3 |
| **조회 1건** | 조회 1회당 몇 점을 줄지 | 0 |
| **노출 순위 수** | 탭마다 몇 위까지 만들지 (1~200) | 100 |

> **가중치를 바꿔도 순위는 곧바로 바뀌지 않습니다.** 저장한 뒤 **[지금 집계]** 를 다시 실행해야 반영됩니다. 저장 후 뜨는 안내 문구도 같은 이야기를 합니다.

> **조회 점수를 0 보다 크게 올릴 때는 주의하세요.** 조회수는 **기간과 무관한 누적값**입니다. 조회 점수를 올리면 **실시간·일간 랭킹에도 예전부터 쌓인 조회수가 섞여 들어와** 최근 인기와 동떨어진 순위가 나올 수 있습니다. 화면에도 같은 경고가 표시됩니다.

## 5. 탭(그룹) 관리

고객 화면 베스트 페이지의 **탭**이 곧 여기의 그룹입니다.

### 5-1. 탭 추가하기

**탭 추가** 영역에서 아래를 입력하고 **[추가]** 를 누릅니다.

| 입력란 | 필수 | 설명 |
|---|---|---|
| **유형** | 필수 | `전체 (몰 전체 상품)` / `카테고리` / `브랜드` |
| **카테고리** | 유형이 `카테고리` 일 때 필수 | 대상 카테고리를 고릅니다. |
| **브랜드** | 유형이 `브랜드` 일 때 필수 | 대상 브랜드를 고릅니다. |
| **탭 이름** | 필수 | 고객에게 보이는 탭 이름. **카테고리·브랜드를 고르면 자동으로 채워집니다.** |
| **순서** | 선택 | 숫자가 작을수록 앞에 나옵니다. |
| **하위 카테고리 포함** | 카테고리 유형만 | 켜면 그 카테고리의 **하위 분류 상품까지** 순위에 넣습니다. 기본으로 켜져 있습니다. |

유형을 바꾸면 필요한 입력칸만 화면에 나타납니다.

> **이미 탭이 있는 카테고리·브랜드는 선택 목록에 나오지 않습니다.** 같은 대상으로 탭을 두 개 만들지 못하게 막는 장치입니다. 찾는 항목이 목록에 없다면 **이미 탭이 만들어져 있는 것**이니 아래 탭 목록을 확인하세요.

### 5-2. 탭 목록에서 보이는 것

| 칸 | 설명 |
|---|---|
| **순서** | 정렬 순서 |
| **탭 이름** | 누르면 상세 화면으로 갑니다 |
| **유형** | 전체 / 카테고리 / 브랜드 (+ `하위 포함` 표시) |
| **집계된 순위** | 몇 건이 집계됐는지. **아직이면 `미집계`** 로 표시됩니다. |
| **MD 픽** | 고정한 상품 수 |
| **노출** | 켜져 있으면 체크 표시 |

### 5-3. 탭 수정 · 삭제

탭 이름을 눌러 상세로 들어가면 **탭 설정** 영역에서 이름·순서·노출·하위 카테고리 포함을 바꾸고 **[저장]** 할 수 있습니다.

목록의 **삭제**를 누르면 확인 창이 뜹니다.

> **탭을 삭제하면 그 탭의 집계된 순위와 MD 픽도 함께 사라집니다.** 잠깐 내리려는 것이라면 삭제하지 말고 상세 화면에서 **노출** 체크를 끄세요.

## 6. MD 픽 (수동 고정) — 밀고 싶은 상품 올리기

자동 순위와 별개로 **특정 상품을 원하는 자리에 강제로 올리는** 기능입니다. 탭 이름을 눌러 들어간 상세 화면에서 관리합니다.

### 6-1. 고정하는 방법

1. **상품 검색** 칸에 상품명이나 상품코드를 **2글자 이상** 입력합니다.
2. 아래 뜨는 목록에서 상품을 클릭합니다. (`선택됨: 상품명` 이 표시되고 [고정] 버튼이 활성화됩니다)
3. 아래 항목을 채우고 **[고정]** 을 누릅니다.

| 입력란 | 필수 | 설명 |
|---|---|---|
| **고정 순위** | 선택 | 몇 위에 놓을지 (1~200). **비우면 자동 1위 앞에 순서대로 붙습니다.** |
| **노출 시작** | 선택 | 이 시각부터 고정이 적용됩니다. 비우면 즉시. |
| **노출 종료** | 선택 | 이 시각에 고정이 끝납니다. 비우면 계속. |
| **메모** | 선택 | 왜 미는지 적어 둡니다. 최대 200자. 운영자용이며 고객에게 보이지 않습니다. |

> **MD 픽은 고객 화면에 즉시 반영됩니다.** 집계를 기다리지 않습니다. 잘못 고정하면 바로 고객이 보게 되므로 확인하고 누르세요.

> **MD 픽은 실시간·일간·주간·월간 모든 기간 탭에 함께 적용됩니다.** 기간별로 따로 고정할 수는 없습니다.

### 6-2. 고정한 상품 수정 · 해제

고정 목록의 각 줄에서 고정 순위·순서·노출 기간·메모·활성을 고친 뒤 **[저장]** 을 누릅니다. 줄마다 저장 버튼이 따로 있습니다.

**[해제]** 를 누르면 고정이 풀립니다. (상품 자체는 지워지지 않습니다.)

**활성** 체크를 끄면 고정을 지우지 않고 잠시 멈출 수 있습니다.

> **고정한 상품의 판매 상태가 `판매중` 이 아니면 목록에 빨간 글씨로 표시됩니다.** 품절·판매중지 상품을 1위에 고정해 두면 고객이 살 수 없는 상품이 맨 위에 놓입니다. 표시를 보면 바로 확인하세요.

### 6-3. 고객 화면 미리보기

상세 화면 아래에는 **고객이 실제로 보는 것과 같은 순위**가 표시됩니다.

- 오른쪽 위 버튼(`실시간` / `일간` / `주간` / `월간`)으로 기간을 바꿔 봅니다.
- **MD 픽으로 고정한 상품에는 `MD` 표시**와 테두리가 붙습니다.
- 자동 순위 상품에는 **점수 · 판매 수 · 찜 수**가 함께 표시됩니다.
- 위쪽에 **집계 기준 시각**이 나옵니다. 집계 전이면 `아직 집계되지 않았습니다` 로 표시됩니다.

## 7. 집계 실행과 자동 집계 스케줄

### 7-1. 지금 집계

**집계 상태** 영역 오른쪽의 **[지금 집계]** 를 누르면 **모든 탭 × 모든 기간**을 즉시 다시 계산합니다. 상품이 많으면 몇 초 걸릴 수 있으니 화면이 끝날 때까지 기다리세요. 끝나면 갱신 건수가 안내로 표시됩니다.

집계 상태 영역에는 기간별로 **마지막 집계가 언제였는지**와 **몇 건이었는지**가 표시되고, 실패했다면 빨간 글씨로 `(실패)` 가 붙습니다.

### 7-2. 집계 스케줄

**집계 스케줄** 표에서 기간별로 자동 집계를 켜고 주기를 정합니다.

| 칸 | 설명 |
|---|---|
| **기간** | 실시간(최근 1시간) / 일간(최근 24시간) / 주간(최근 7일) / 월간(최근 30일) |
| **자동 집계** | 체크하면 자동으로 돌립니다. |
| **주기(분)** | 몇 분마다 돌릴지. **최소 5분, 최대 1440분(1일)** |
| **마지막 성공** | 마지막으로 성공한 시점 |

값을 바꿨으면 **[스케줄 저장]** 을 눌러야 반영됩니다.

> **집계 스케줄은 몰 공통입니다.** 다른 몰과 함께 쓰는 설정이라, 여기서 바꾸면 **모든 몰의 집계 주기가 바뀝니다.** 탭·가중치·MD 픽과 달리 편집 몰별로 분리되지 않습니다.

> **자동 집계가 동작하려면 서버에 크론 등록이 한 번 되어 있어야 합니다.** 화면 아래에 등록할 명령 한 줄이 안내돼 있습니다. 이 등록은 **최초 1회만** 하면 되고, 이후 주기·on/off 는 전부 이 화면에서 관리합니다. 서버 작업이 필요하니 개발 담당자에게 요청하세요.

## 8. 자주 막히는 상황

**Q. 탭을 만들었는데 순위가 비어 있습니다.**
아직 집계하지 않았습니다. 목록에 `미집계` 로 표시됩니다. **[지금 집계]** 를 누르세요.

**Q. 가중치를 바꿨는데 순위가 그대로입니다.**
저장만으로는 반영되지 않습니다. **[지금 집계]** 를 다시 실행하세요.

**Q. 원하는 카테고리가 탭 추가 목록에 없습니다.**
그 카테고리로 이미 탭이 만들어져 있습니다. 아래 탭 목록에서 찾아보세요. 그래도 없다면 **편집 중인 몰**이 다를 수 있습니다.

**Q. MD 픽으로 고정했는데 고객 화면에 안 보입니다.**
아래를 확인하세요.
1. 줄의 **활성** 체크가 켜져 있는가?
2. **노출 시작·종료** 기간이 지금을 포함하는가?
3. 그 탭의 **노출** 체크가 켜져 있는가?
4. 상품의 판매 상태·노출 설정이 정상인가? (`노출중지` 면 어디에도 안 나옵니다)

**Q. 순위를 직접 끌어서 바꾸고 싶습니다.**
그런 기능은 없습니다. 자동 순위는 집계 결과라서 편집할 수 없고, **MD 픽**이 순위를 손대는 유일한 방법입니다.

**Q. 실시간 탭 순위가 자꾸 비거나 이상합니다.**
실시간은 **최근 1시간**만 봅니다. 주문이 드문 시간대에는 데이터가 거의 없어 순위가 빈약할 수 있습니다. 일간·주간을 기본 탭으로 쓰는 편이 안정적입니다.

**Q. 조회수를 반영했더니 예전 상품이 실시간 1위에 올라옵니다.**
조회수는 **누적값**이라 기간 계산이 되지 않습니다. 조회 가중치를 0 으로 되돌리고 다시 집계하세요.

**Q. 품절 상품이 베스트 1위에 있습니다.**
MD 픽으로 고정해 둔 상품일 수 있습니다. 고정 목록에서 빨간 글씨로 표시된 줄을 찾아 **[해제]** 하거나 **활성**을 끄세요.

## 9. 아직 지원하지 않는 것

- 자동 순위를 손으로 끌어 옮기기
- 기간(실시간/일간/주간/월간)별로 다른 MD 픽 지정
- 몰마다 다른 집계 주기 설정
- 조건(가격대·태그 등)을 직접 지정하는 커스텀 탭
