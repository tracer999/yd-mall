# 베스트 관리

고객 화면의 **베스트** 메뉴에 보이는 **인기 상품 순위**를 관리하는 메뉴입니다.

베스트 순위는 운영자가 직접 줄을 세우는 것이 아니라, **판매량과 좋아요를 점수로 합산해 자동으로 매겨집니다.** 운영자는 ① 점수를 매기는 기준(가중치), ② 어떤 묶음(그룹)으로 순위를 보여 줄지, ③ 맨 위에 고정할 상품(MD 픽)을 정합니다.

---

## 순위가 매겨지는 방식

- 상품마다 **판매 수량 · 좋아요 수 · 조회 수**를 모아 점수를 냅니다.
- 기본 가중치는 **판매 5점 · 좋아요 3점 · 조회 0점**입니다. 즉 지금은 **판매량이 가장 크게 반영되고, 조회 수는 순위에 영향을 주지 않습니다.**
- 가중치는 베스트 관리 화면에서 바꿀 수 있습니다. 조회 수도 반영하고 싶으면 조회 가중치를 0보다 크게 올리면 됩니다.
- 매출로 인정하는 주문 상태는 **결제완료 · 상품준비중 · 배송중 · 배송완료** 입니다. 취소·반품된 주문은 점수에 들어가지 않습니다.

---

## 기간별 순위 (실시간 / 일간 / 주간 / 월간)

순위는 **기간마다 따로** 계산됩니다.

| 기간 | 무엇을 보나요 |
|------|----------------|
| **실시간** | 아주 최근의 판매·좋아요 흐름 |
| **일간** | 하루 동안의 성적 |
| **주간** | 한 주 동안의 성적 |
| **월간** | 한 달 동안의 성적 |

고객은 베스트 화면 위쪽의 **기간 탭**에서 원하는 기간을 골라 봅니다. 따라서 같은 상품이라도 실시간 1위인데 월간에서는 5위일 수 있습니다. **정상입니다.**

---

## 베스트 그룹 만들기

**그룹**은 "어떤 범위의 베스트인가"를 정하는 묶음입니다.

- **전체 베스트:** 쇼핑몰 전체 상품을 대상으로 하는 순위
- **카테고리별 베스트:** 특정 카테고리 안에서의 순위 (예: 건강식품 베스트)
- **브랜드별 베스트:** 특정 브랜드 상품 안에서의 순위

### 등록 절차

1. **베스트 관리** 화면에서 **그룹 추가** 버튼을 클릭합니다.
2. **그룹 이름**을 입력합니다. 고객 화면에 보이는 이름이므로 알아보기 쉽게 씁니다. (예: 건강식품 베스트)
3. **범위**를 고릅니다. (전체 / 특정 카테고리 / 특정 브랜드)
4. **표시 순서**와 **사용 여부**를 정합니다.
5. **저장**합니다.

저장한 뒤 집계가 한 번 돌아야 순위가 채워집니다. 바로 보고 싶으면 아래 **"지금 집계"** 를 이용하세요.

---

## MD 픽 (수동으로 상품 고정하기)

밀고 싶은 상품이 있는데 아직 판매량이 적어 순위가 낮을 때, **맨 위에 고정**할 수 있습니다.

1. 그룹을 선택한 뒤 **MD 픽** 또는 **수동 고정** 영역을 엽니다.
2. **상품 검색**으로 고정할 상품을 찾아 담습니다.
3. 고정 상품끼리의 **순서**를 정합니다.
4. **저장**합니다.

- 고정한 상품은 **집계를 기다리지 않고 즉시 고객 화면에 반영**됩니다.
- 고정한 상품이 자동 순위에도 들어 있으면, **중복해서 두 번 나오지 않고** 위쪽 고정 자리로 올라갑니다.

---

## 집계 주기와 "지금 집계"

순위 계산은 **자동으로 주기적으로** 돌아갑니다.

| 기간 | 기본 집계 주기 |
|------|----------------|
| 실시간 | 10분마다 |
| 일간 | 1시간마다 |
| 주간 | 하루 1회 |
| 월간 | 하루 1회 |

- 주기는 베스트 관리 화면에서 바꿀 수 있습니다.
- 급하게 반영해야 할 때는 **"지금 집계"** 버튼을 누르면 그 자리에서 다시 계산합니다.

**주의:** 가중치를 바꾸거나 그룹을 새로 만든 직후에는 아직 예전 순위가 보입니다. **"지금 집계"** 를 눌러 다시 계산해야 바뀐 기준이 화면에 나타납니다.

---

## 고객 화면에서는 어디에 보이나요?

- 상단 메뉴의 **베스트**(주소 `/best`) 에서 그룹과 기간 탭으로 순위를 봅니다.
- 메인 화면(홈)에도 **베스트 랭킹 섹션**을 넣어 일부를 보여 줄 수 있습니다. (메인 화면 만들기 참고)
- 예전에 따로 있던 **"랭킹" 메뉴는 베스트로 합쳐졌습니다.** 랭킹 메뉴는 기본으로 꺼져 있습니다.

---

## 아직 안 되는 것

- **성별·연령별 순위**는 화면에 항목이 보이지만 **선택할 수 없습니다.** 아직 집계할 데이터가 모이지 않아 비활성 상태입니다.
