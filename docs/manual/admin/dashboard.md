# 대시보드

관리자로 로그인하면 **가장 먼저 보이는 요약 화면**입니다. 쇼핑몰의 현재 상태를 숫자로 한눈에 볼 수 있습니다.

---

## 화면에 무엇이 나오나요?

### 상단 통계 카드 (4개)

| 카드 | 의미 |
|------|------|
| **회원 수** | 지금까지 가입한 회원 전체 수입니다. |
| **총 상품** | 등록된 상품 전체 개수입니다. |
| **새 문의** | 아직 답변하지 않은 1:1 문의 건수입니다. 숫자가 있으면 **문의 관리** 메뉴에서 답변을 작성할 수 있습니다. |
| **오늘 방문** | 오늘 하루 동안 쇼핑몰에 방문한 사람 수(중복 제외)입니다. |

이 숫자들은 **실시간에 가깝게** 계산되므로, 새로고침하면 최신 값으로 바뀝니다.

### 최근 가입 회원

- **최근 5명**의 가입 회원 이름·이메일·가입일이 표시됩니다.
- 더 많은 회원을 보거나 관리하려면 **회원 관리** 메뉴를 이용하세요.

### 검색 통계

- 고객들이 **무엇을 검색했는지** 기간별로 볼 수 있습니다.
- **인기 검색어:** 많이 검색된 키워드 순서
- **검색 결과 없음 키워드:** 검색했는데 결과가 0건이었던 키워드 (상품 추가·이름 수정 시 참고하면 좋습니다)

기간은 **최근 24시간 / 7일 / 30일** 중에서 선택할 수 있습니다.  
자세한 검색 로그는 **검색 로그** 메뉴에서 확인할 수 있습니다.

---

## 어떻게 활용하면 좋을까요?

- **새 문의** 숫자가 0보다 크면 → 문의 관리에서 답변을 먼저 처리하세요.
- **오늘 방문** 수로 홍보·이벤트 효과를 대략 파악할 수 있습니다.
- **검색 결과 없음 키워드**를 보면, 고객이 찾는데 없는 상품·이름을 알 수 있어 상품 기획이나 이름 수정에 참고할 수 있습니다.
