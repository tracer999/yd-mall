# 로그인 / 로그아웃

관리자 사이트에 들어오고 나가는 방법을 안내합니다.

---

## 로그인하기

1. 브라우저에서 **관리자 주소**로 접속합니다. (예: `https://쇼핑몰주소/admin`)
2. **로그인 화면**이 나오면 다음을 입력합니다.
   - **아이디:** 관리자 계정 아이디
   - **비밀번호:** 해당 계정의 비밀번호
3. **로그인** 버튼을 누릅니다.
4. 입력이 맞으면 **대시보드** 화면으로 이동합니다.

**잘못 입력했을 때**

- "존재하지 않는 계정입니다." → 아이디를 다시 확인하세요.
- "비밀번호가 일치하지 않습니다." → 비밀번호를 다시 확인하세요.

비밀번호를 잊었다면, 다른 관리자(최고 관리자)에게 요청해 비밀번호를 변경하거나, 시스템 관리자에게 문의하세요.

---

## 로그아웃하기

- 관리자 화면 **우측 상단**의 **로그아웃** 링크를 클릭하면 로그아웃됩니다.
- 로그아웃 후에는 다시 로그인 화면으로 돌아갑니다.

다른 사람과 컴퓨터를 같이 쓰는 경우, 사용이 끝나면 반드시 로그아웃하는 것이 좋습니다.
