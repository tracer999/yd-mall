# 테마 설정

스토어프론트의 **모서리·간격·폰트** 같은 스타일을 조정하는 메뉴입니다. (`/admin/theme-settings`)

---

## ⚠️ 먼저 알아야 할 두 가지

### 1. 이 화면에는 색상이 없습니다

많이 헷갈리는 부분입니다. **브랜드 색상과 로고는 [사이트 설정](./settings.md)(`/admin/site-settings`)에서** 관리합니다. 테마 설정은 **모양(모서리·간격·폰트)** 만 다룹니다.

| 바꾸고 싶은 것 | 가야 할 메뉴 |
|---------------|-------------|
| 색상, 로고, 상호 | **사이트 설정** |
| 모서리 둥글기, 글꼴, 여백 | **테마 설정** (이 화면) |
| 헤더 모양, 메뉴 개수 | **Header 설정** |

### 2. "테마를 고르는 곳"이 아닙니다

테마 1·2·3 중 **어떤 테마를 쓸지 고르는 것은 [몰 관리](./malls.md)** 입니다. 몰을 만들 때 고르고, 나중에 바꾸려면 몰 수정 화면의 **프리셋 재적용**을 씁니다.

이 화면은 **이미 적용된 테마를 미세하게 다듬는** 곳입니다.

---

## 테마 설정은 몰마다 따로입니다

테마 값은 **지금 편집 중인 몰**에만 저장됩니다. 값을 바꾸기 전에 상단에서 **편집 중인 몰이 맞는지 반드시 확인**하세요. 엉뚱한 몰의 디자인을 바꿔 놓기 쉽습니다.

---

## 입력 항목

### 테마 이름

이 테마를 부르는 이름입니다. 몰을 만들 때 **"〇〇 몰 기본 테마"** 로 자동으로 붙습니다.

### 스타일 토큰

**비워 두면 기본값으로 되돌아갑니다.** 각 칸 옆에 **적용 중: 〇〇** 로 현재 값이 보입니다.

| 항목 | 무엇을 바꾸나 | 입력 예 | 기본값 |
|------|--------------|---------|--------|
| **폰트 패밀리** | 쇼핑몰 전체 글꼴 | `'Pretendard', sans-serif` | Pretendard |
| **버튼 모서리** | 버튼의 둥근 정도 | `0.5rem`, `8px`, `0` | `0.5rem` |
| **카드 모서리** | 상품 카드의 둥근 정도 | `0.5rem` | `0.5rem` |
| **알약형 모서리** | 태그·뱃지처럼 완전히 둥근 요소 | `9999px` | `9999px` |
| **입력창 모서리** | 검색창·입력칸의 둥근 정도 | `0.375rem` | `0.375rem` |
| **섹션 간격** | 메인 화면 섹션 사이 위아래 여백 | `3rem` | `3rem` |
| **본문 최대 너비** | 화면이 넓을 때 내용이 퍼지는 최대 폭 | `72rem` | `72rem` |

### 상품 카드 스타일

상품 카드의 테두리 처리를 고릅니다.

| 선택지 | 모양 |
|--------|------|
| **shadow** | 그림자 |
| **border** | 테두리 |
| **flat** | 민무늬 |

### 스토어프론트에 주입되는 CSS

읽기 전용입니다. 지금 설정이 실제로 어떤 값으로 들어가는지 그대로 보여 줍니다. **수정할 수 없고, 확인용**입니다.

---

## ⚠️ 입력 형식이 엄격합니다

값이 화면 디자인에 직접 들어가기 때문에, **정해진 형식이 아니면 저장이 거부**됩니다.

- **길이 값**은 `0.5rem` · `12px` · `50%` · `0` 같은 형식만 됩니다. (숫자 + 단위)
- **폰트**는 따옴표·쉼표·하이픈·점만 쓸 수 있고 200자를 넘을 수 없습니다.

형식이 틀리면 **"저장하지 않았습니다. 허용되지 않는 값이 있습니다."** 와 함께 어떤 항목이 왜 거부됐는지 알려 줍니다. 이때는 **아무것도 저장되지 않습니다.**

또, 예전에 저장된 값이 검증을 통과하지 못하면 그 항목에 **⚠ 기본값 대체** 뱃지가 붙습니다. 저장은 되어 있지만 화면에는 기본값이 쓰이고 있다는 뜻이므로, 값을 고쳐 주세요.

---

## 저장하면 바로 반영됩니다

**저장**을 누르면 **"저장되었습니다. 스토어프론트에 즉시 반영됩니다."** 가 뜹니다. 고객 화면을 새로고침하면 바로 보입니다. 잘못 건드렸다면 **되돌리기**로 저장 전 상태로 돌아갈 수 있습니다.

> 값을 지우고 저장하면 **기본값으로 복귀**합니다. 원래대로 돌리고 싶을 때 가장 쉬운 방법입니다.

---

## 아직 안 되는 것

- **색상은 이 화면에서 바꿀 수 없습니다.** 사이트 설정을 쓰세요.
- **테마를 여러 개 만들어 두고 전환하는 기능은 없습니다.** 몰마다 활성 테마는 항상 하나입니다.
- **테마 1·2·3 을 이 화면에서 고를 수 없습니다.** 몰 관리의 프리셋 재적용으로 바꿉니다.
- **미리보기가 없습니다.** 저장해야 결과를 볼 수 있으므로, 고객이 보는 몰이라면 값을 조금씩 바꿔 가며 확인하세요.
