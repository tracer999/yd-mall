# 디자인 스타일

고객 화면의 **모서리 둥글기 · 여백 · 글꼴 · 상품 카드 스타일** 같은 세부 디자인 값을 다듬는 메뉴입니다. 몰의 인상을 크게 좌우하지만, 값 하나하나는 아주 작은 조정입니다.

화면 위치: <span class="manual-path">관리자 &gt; 쇼핑몰 관리 &gt; 디자인 스타일</span> (`/admin/theme-settings`)

> **이 화면에는 색상이 없고, 테마 1·2·3을 고르는 기능도 없습니다.** 가장 많이 헷갈리는 부분이니 아래 1장을 먼저 읽으세요.

---

## 1. 먼저 알아야 할 두 가지

### 1-1. 색상과 로고는 여기가 아닙니다

**브랜드 색상·로고·상호는 사이트 설정**에서 관리합니다. 디자인 스타일은 **모양(모서리·간격·글꼴)** 만 다룹니다.

| 바꾸고 싶은 것 | 가야 할 화면 |
|---|---|
| 색상, 로고, 상호, 파비콘 | **사이트 설정** |
| 모서리 둥글기, 글꼴, 여백, 카드 스타일 | **디자인 스타일** (이 화면) |
| 헤더 모양, 메뉴 개수 | **Header 설정** |
| 히어로 배치(테마 1·2·3) | **페이지 빌더 → 테마 설정 탭** |

<div class="manual-goto-bar">
  <span class="manual-goto-label">색상을 바꾸려면</span>
  <a class="manual-goto" href="/admin/site-settings" target="_blank">사이트 설정 → 브랜드 키 컬러</a>
</div>

### 1-2. "테마를 고르는 곳"이 아닙니다

이름이 비슷한 화면이 하나 더 있습니다. **페이지 빌더의 [테마 설정] 탭**입니다. 둘은 하는 일이 다릅니다.

| 화면 | 하는 일 |
|---|---|
| **디자인 스타일** (이 화면) | 이미 적용된 테마의 **모서리·글꼴·간격·카드 스타일**을 다듬습니다 |
| **페이지 빌더 → [테마 설정] 탭** | 테마 1·2·3 중 **어떤 테마를 쓸지 고릅니다**(히어로 배치·메뉴 구성 방식) |

즉 테마 1·2·3 중 **어떤 테마를 쓸지 고르는 것**은 몰을 만들 때, 또는 나중에 **페이지 빌더의 [테마 설정] 탭**에서 합니다. 이 **디자인 스타일** 화면은 그렇게 **이미 적용된 테마를 미세하게 다듬는** 곳입니다.

<div class="manual-goto-bar">
  <span class="manual-goto-label">테마 1·2·3을 바꾸려면</span>
  <a class="manual-goto" href="/admin/page-builder" target="_blank">페이지 빌더 → [테마 설정] 탭</a>
</div>

> **페이지 빌더에서 테마를 다시 적용하면, 이 화면에서 손본 모서리·글꼴·간격 값이 그 테마의 기본값으로 초기화됩니다.** 순서를 지키세요. **① 테마 1·2·3 결정 → ② 이 화면에서 다듬기.** 반대로 하면 공들인 값이 날아갑니다.

---

## 2. 디자인 스타일은 몰마다 따로입니다

값은 **지금 편집 중인 몰**에만 저장됩니다. 오른쪽 위의 몰 이름을 확인한 뒤 시작하세요.

새로 만든 몰에 아직 테마가 없으면, 이 화면을 여는 순간 **기본 테마가 자동으로 만들어져** 편집할 수 있는 상태가 됩니다.

---

## 3. 입력 항목

### 3-1. 테마 이름

이 테마를 부르는 이름입니다. 최대 100자. 관리자에게만 보이는 이름이며 고객 화면에는 나오지 않습니다.

### 3-2. 스타일 토큰

**모두 목록에서 고르는 방식입니다.** 값을 직접 입력할 필요가 없고, 잘못된 값을 넣을 수도 없습니다. 각 항목의 선택지 끝에 **— 기본** 이라고 붙은 것이 처음 상태입니다.

| 선택 항목 | 무엇을 바꾸나 | 선택지 |
|---|---|---|
| **본문 폰트** | 쇼핑몰 전체 글꼴 | 프리텐다드(기본·고딕) · 나눔명조(명조) · Playfair Display(영문 세리프) · 기기 기본 폰트 |
| **버튼 모서리** | 버튼의 둥근 정도 | 각지게 ~ 아주 많이 둥글게 (7단계) |
| **카드 모서리** | 상품 카드·배너의 둥근 정도 | 위와 같은 7단계 |
| **입력창 모서리** | 검색창·입력칸의 둥근 정도 | 위와 같은 7단계 |
| **태그·칩 모서리** | 태그·필터칩처럼 작고 둥근 요소 | 완전 둥글게(알약형) · 조금 둥근 사각형 · 거의 각진 사각형 · 각지게 |
| **섹션 간격** | 메인 화면 섹션 사이의 위아래 여백 | 아주 좁게 · 좁게 · 보통 · 넓게 · 아주 넓게 |
| **본문 최대 너비** | 화면이 넓을 때 내용이 퍼지는 최대 폭 | 좁게(1024px) · 보통(1152px) · 넓게(1280px) · 아주 넓게(1440px) · 화면 전체 |

**모서리 7단계 감 잡기**

| 선택지 | 느낌 |
|---|---|
| 각지게 | 딱딱하고 정제된 인상 |
| 아주 살짝 ~ 보통 | 가장 무난합니다 |
| 둥글게 ~ 아주 많이 둥글게 | 캐주얼하고 부드러운 인상 |

> **폰트는 이 몰이 실제로 불러오는 글꼴만 목록에 있습니다.** 다른 글꼴 이름을 넣을 수 있게 하면 "저장은 됐는데 화면은 그대로"인 상태가 되기 때문입니다.

> **"현재 값 유지" 라는 선택지가 보인다면** — 예전에 직접 입력해 둔 값이 지금 목록에 없다는 뜻입니다. 그대로 두면 유지되고, 다른 것을 고르면 그 값으로 바뀝니다. 한 번 다른 값을 고르면 "현재 값 유지"는 다시 나타나지 않습니다.

### 3-3. 상품 카드 스타일

상품 카드의 테두리 처리를 셋 중에 고릅니다.

| 선택지 | 모양 |
|---|---|
| **그림자** | 카드에 그림자가 깔립니다. 카드가 떠 보입니다. |
| **테두리** | 얇은 테두리 선이 둘러집니다. |
| **민무늬** | 그림자도 테두리도 없습니다. 여백으로만 구분합니다. |

### 3-4. 스토어프론트에 주입되는 CSS (개발자용)

접혀 있는 영역입니다. 제목을 클릭하면 펼쳐지며, 지금 설정이 실제로 어떤 값으로 고객 화면에 들어가는지 보여 줍니다. **읽기 전용이고, 운영에는 볼 일이 없습니다.** 개발자와 문제를 이야기할 때만 쓰세요.

---

## 4. 저장하면 바로 반영됩니다

**저장**을 누르면 `저장되었습니다. 스토어프론트에 즉시 반영됩니다.` 가 뜹니다. 고객 화면을 새로고침하면 곧바로 보입니다.

**되돌리기** 버튼을 누르면 저장된 값으로 화면을 다시 불러옵니다. 저장하기 전에 실수를 취소할 때 쓰세요.

> **고른 즉시 저장되는 것은 아닙니다.** 목록에서 골라도 아래 **저장**을 눌러야 반영됩니다.

> **미리보기가 없습니다.** 저장해야 결과를 볼 수 있습니다. 고객이 실제로 보고 있는 몰이라면 **한 번에 여러 항목을 바꾸지 말고 하나씩 바꿔 가며** 확인하세요. 잘못 골라도 **— 기본** 이 붙은 선택지로 되돌리면 원래대로 돌아옵니다.

---

## 5. 실제로 어떻게 다듬나 — 예시

**딱딱하고 정제된 인상으로 바꾸고 싶을 때**

| 항목 | 고를 값 |
|---|---|
| 버튼 모서리 | 각지게 |
| 카드 모서리 | 각지게 |
| 입력창 모서리 | 각지게 |
| 상품 카드 스타일 | 민무늬 |

**부드럽고 캐주얼한 인상으로 바꾸고 싶을 때**

| 항목 | 고를 값 |
|---|---|
| 버튼 모서리 | 많이 둥글게 |
| 카드 모서리 | 많이 둥글게 |
| 상품 카드 스타일 | 그림자 |

**화면이 답답할 때**

- **섹션 간격**을 `보통` → `넓게` 로 올려 봅니다.
- **본문 최대 너비**를 `보통(1152px)` → `넓게(1280px)` 로 올리면 넓은 화면에서 더 시원해집니다. 다만 너무 넓히면 글줄이 길어져 읽기 힘들어집니다.

---

## 6. 자주 막히는 상황

**Q. 색상을 바꾸고 싶은데 항목이 없습니다.**
색상은 **사이트 설정 → 브랜드 키 컬러**에서 바꿉니다. 이 화면에는 색상 항목이 없습니다.

**Q. 저장했는데 고객 화면이 그대로입니다.**
① **편집 중인 몰**이 맞는지 확인하고, ② 브라우저를 완전 새로고침(Ctrl+F5)해 보세요.

**Q. 원하는 글꼴이 목록에 없습니다.**
이 몰이 실제로 불러오는 글꼴만 목록에 있습니다. 다른 글꼴을 쓰려면 개발자가 그 글꼴을 몰에 추가해야 합니다. 목록에 없는 이름을 넣어도 화면은 바뀌지 않기 때문에, 아예 고를 수 없게 해 두었습니다.

**Q. "현재 값 유지" 라는 선택지가 있습니다.**
예전에 직접 입력해 둔 값이라 지금 목록에 없다는 뜻입니다. 그대로 두면 유지되고, 다른 것을 고르면 그 값으로 바뀝니다.

**Q. 테마를 다시 적용했더니 여기서 손본 값이 전부 사라졌습니다.**
정상입니다. 테마 재적용은 스타일 값을 그 테마의 기본값으로 되돌립니다. **테마를 먼저 확정하고 나서** 이 화면을 손보세요.

**Q. 테마 1·2·3 을 이 화면에서 고를 수 없나요?**
없습니다. 페이지 빌더의 [테마 설정] 탭에서 고릅니다.

**Q. 원래 디자인으로 완전히 되돌리고 싶습니다.**
모든 항목에서 **— 기본** 이 붙은 선택지를 고르고 저장하면 전부 처음 상태로 돌아옵니다.

---

## 7. 아직 지원하지 않는 것

- **색상 변경** — 사이트 설정을 쓰세요
- **테마를 여러 개 만들어 두고 전환** — 몰마다 활성 테마는 항상 하나입니다
- **테마 1·2·3 선택** — 페이지 빌더에서 합니다
- **실시간 미리보기** — 저장해야 결과를 볼 수 있습니다
- **목록에 없는 글꼴·수치 직접 입력** — 고를 수 있는 값만 씁니다
- 개별 화면(상품 상세·장바구니 등)만 다르게 꾸미기
