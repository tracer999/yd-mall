# 디자인 스타일

고객 화면의 **모서리 둥글기 · 여백 · 글꼴 · 상품 카드 스타일** 같은 세부 디자인 값을 다듬는 메뉴입니다. 몰의 인상을 크게 좌우하지만, 값 하나하나는 아주 작은 조정입니다.

화면 위치: <span class="manual-path">관리자 &gt; 쇼핑몰 관리 &gt; 디자인 스타일</span> (`/admin/theme-settings`)

> **이 화면에는 색상이 없고, 테마 1·2·3을 고르는 기능도 없습니다.** 가장 많이 헷갈리는 부분이니 아래 1장을 먼저 읽으세요.

---

## 1. 먼저 알아야 할 두 가지

### 1-1. 색상과 로고는 여기가 아닙니다

**브랜드 색상·로고·상호는 사이트 설정**에서 관리합니다. 디자인 스타일은 **모양(모서리·간격·글꼴)** 만 다룹니다.

| 바꾸고 싶은 것 | 가야 할 화면 |
|---|---|
| 색상, 로고, 상호, 파비콘 | **사이트 설정** |
| 모서리 둥글기, 글꼴, 여백, 카드 스타일 | **디자인 스타일** (이 화면) |
| 헤더 모양, 메뉴 개수 | **Header 설정** |
| 히어로 배치(테마 1·2·3) | **페이지 빌더 → 테마 설정 탭** |

<div class="manual-goto-bar">
  <span class="manual-goto-label">색상을 바꾸려면</span>
  <a class="manual-goto" href="/admin/site-settings" target="_blank">사이트 설정 → 브랜드 키 컬러</a>
</div>

### 1-2. "테마를 고르는 곳"이 아닙니다

이름이 비슷한 화면이 하나 더 있습니다. **페이지 빌더의 [테마 설정] 탭**입니다. 둘은 하는 일이 다릅니다.

| 화면 | 하는 일 |
|---|---|
| **디자인 스타일** (이 화면) | 이미 적용된 테마의 **모서리·글꼴·간격·카드 스타일**을 다듬습니다 |
| **페이지 빌더 → [테마 설정] 탭** | 테마 1·2·3 중 **어떤 테마를 쓸지 고릅니다**(히어로 배치·메뉴 구성 방식) |

즉 테마 1·2·3 중 **어떤 테마를 쓸지 고르는 것**은 몰을 만들 때, 또는 나중에 **페이지 빌더의 [테마 설정] 탭**에서 합니다. 이 **디자인 스타일** 화면은 그렇게 **이미 적용된 테마를 미세하게 다듬는** 곳입니다.

<div class="manual-goto-bar">
  <span class="manual-goto-label">테마 1·2·3을 바꾸려면</span>
  <a class="manual-goto" href="/admin/page-builder" target="_blank">페이지 빌더 → [테마 설정] 탭</a>
</div>

> **페이지 빌더에서 테마를 다시 적용하면, 이 화면에서 손본 모서리·글꼴·간격 값이 그 테마의 기본값으로 초기화됩니다.** 순서를 지키세요. **① 테마 1·2·3 결정 → ② 이 화면에서 다듬기.** 반대로 하면 공들인 값이 날아갑니다.

---

## 2. 디자인 스타일은 몰마다 따로입니다

값은 **지금 편집 중인 몰**에만 저장됩니다. 오른쪽 위의 몰 이름을 확인한 뒤 시작하세요.

새로 만든 몰에 아직 테마가 없으면, 이 화면을 여는 순간 **기본 테마가 자동으로 만들어져** 편집할 수 있는 상태가 됩니다.

---

## 3. 입력 항목

### 3-1. 테마 이름

이 테마를 부르는 이름입니다. 최대 100자. 관리자에게만 보이는 이름이며 고객 화면에는 나오지 않습니다.

### 3-2. 스타일 토큰

각 칸 오른쪽 위에 **적용 중: ○○** 으로 실제 쓰이고 있는 값이 표시됩니다. 입력칸의 흐린 글씨는 **기본값**입니다.

| 입력란 | 무엇을 바꾸나 | 입력 예 | 기본값 |
|---|---|---|---|
| **폰트 패밀리** | 쇼핑몰 전체 글꼴 | `'Pretendard', sans-serif` | `'Pretendard', -apple-system, BlinkMacSystemFont, sans-serif` |
| **버튼 모서리** | 버튼의 둥근 정도 | `0.5rem`, `8px`, `0` | `0.5rem` |
| **카드 모서리** | 상품 카드의 둥근 정도 | `0.5rem` | `0.5rem` |
| **알약형 모서리** | 태그·뱃지처럼 완전히 둥근 요소 | `9999px` | `9999px` |
| **입력창 모서리** | 검색창·입력칸의 둥근 정도 | `0.375rem` | `0.375rem` |
| **섹션 간격** | 메인 화면 섹션 사이의 위아래 여백 | `3rem` | `3rem` |
| **본문 최대 너비** | 화면이 넓을 때 내용이 퍼지는 최대 폭 | `72rem` | `72rem` |

> **칸을 비우고 저장하면 그 항목은 기본값으로 되돌아갑니다.** 잘못 건드린 값을 원래대로 돌리는 가장 쉬운 방법입니다.

**모서리 값 감 잡기**

| 값 | 느낌 |
|---|---|
| `0` | 각진 모서리. 딱딱하고 정제된 인상 |
| `0.25rem` ~ `0.5rem` | 살짝 둥근 모서리. 가장 무난합니다 |
| `1rem` 이상 | 많이 둥근 모서리. 캐주얼하고 부드러운 인상 |
| `9999px` | 완전한 알약 모양 |

### 3-3. 상품 카드 스타일

상품 카드의 테두리 처리를 셋 중에 고릅니다.

| 선택지 | 모양 |
|---|---|
| **shadow** | 카드에 그림자가 깔립니다. 카드가 떠 보입니다. |
| **border** | 얇은 테두리 선이 둘러집니다. |
| **flat** | 그림자도 테두리도 없는 민무늬. 여백으로만 구분합니다. |

### 3-4. 스토어프론트에 주입되는 CSS

**읽기 전용 영역**입니다. 지금 설정이 실제로 어떤 값으로 고객 화면에 들어가는지 그대로 보여 줍니다. 수정할 수 없으며, 무엇이 반영되고 있는지 확인하는 용도입니다.

---

## 4. 입력 형식이 엄격합니다

이 값들은 **고객 화면의 디자인에 직접 들어갑니다.** 그래서 형식이 정해져 있고, 벗어나면 저장이 거부됩니다.

| 종류 | 허용되는 형식 |
|---|---|
| **길이 값**(모서리·간격·너비) | `0` · `0.5rem` · `12px` · `50%` · `40vw` 처럼 **숫자 + 단위**만. 소수점은 세 자리까지, 정수는 다섯 자리까지. |
| **폰트 패밀리** | 글자·숫자·공백·따옴표·쉼표·하이픈·점만. **200자 이내.** |

형식이 틀리면 화면 위에 빨간 안내가 뜹니다.

```
저장하지 않았습니다. 허용되지 않는 값이 있습니다.
  · 버튼 모서리: "십픽셀" 는 허용되지 않는 값입니다.
```

> **하나라도 형식이 틀리면 아무것도 저장되지 않습니다.** 나머지 정상 항목까지 함께 저장이 취소되므로, 안내에 적힌 항목을 고쳐 다시 저장하세요.

**⚠ 기본값 대체 뱃지** — 어떤 항목 옆에 이 표시가 붙어 있다면, 저장은 되어 있지만 그 값이 형식 검사를 통과하지 못해 **화면에는 기본값이 쓰이고 있다**는 뜻입니다. 값을 올바른 형식으로 고쳐 주세요.

---

## 5. 저장하면 바로 반영됩니다

**저장**을 누르면 `저장되었습니다. 스토어프론트에 즉시 반영됩니다.` 가 뜹니다. 고객 화면을 새로고침하면 곧바로 보입니다.

**되돌리기** 버튼을 누르면 저장된 값으로 화면을 다시 불러옵니다. 저장하기 전에 실수를 취소할 때 쓰세요.

> **미리보기가 없습니다.** 저장해야 결과를 볼 수 있습니다. 고객이 실제로 보고 있는 몰이라면 **값을 한 번에 크게 바꾸지 말고 하나씩 바꿔 가며** 확인하세요. 잘못 바꿔도 칸을 비우고 저장하면 기본값으로 돌아옵니다.

---

## 6. 실제로 어떻게 다듬나 — 예시

**딱딱하고 정제된 인상으로 바꾸고 싶을 때**

| 항목 | 값 |
|---|---|
| 버튼 모서리 | `0` |
| 카드 모서리 | `0` |
| 입력창 모서리 | `0` |
| 상품 카드 스타일 | `flat` |

**부드럽고 캐주얼한 인상으로 바꾸고 싶을 때**

| 항목 | 값 |
|---|---|
| 버튼 모서리 | `1rem` |
| 카드 모서리 | `1rem` |
| 상품 카드 스타일 | `shadow` |

**화면이 답답할 때**

- **섹션 간격**을 `3rem` → `4.5rem` 으로 늘려 봅니다.
- **본문 최대 너비**를 `72rem` → `80rem` 으로 늘리면 넓은 화면에서 더 시원해집니다. 다만 너무 넓히면 글줄이 길어져 읽기 힘들어집니다.

---

## 7. 자주 막히는 상황

**Q. 색상을 바꾸고 싶은데 항목이 없습니다.**
색상은 **사이트 설정 → 브랜드 키 컬러**에서 바꿉니다. 이 화면에는 색상 항목이 없습니다.

**Q. 저장했는데 고객 화면이 그대로입니다.**
① **편집 중인 몰**이 맞는지 확인하고, ② 브라우저를 완전 새로고침(Ctrl+F5)해 보세요.

**Q. "허용되지 않는 값입니다"가 계속 뜹니다.**
길이 값에 단위가 빠졌거나(`8` 대신 `8px`), 한글·수식이 들어간 경우입니다. `0` 만은 단위 없이 쓸 수 있습니다.

**Q. 어떤 항목에 ⚠ 기본값 대체가 붙어 있습니다.**
예전에 저장된 값이 형식에 맞지 않아 화면에는 기본값이 쓰이고 있다는 뜻입니다. 그 칸의 값을 올바른 형식으로 고치거나, 비우고 저장해 기본값으로 정리하세요.

**Q. 테마를 다시 적용했더니 여기서 손본 값이 전부 사라졌습니다.**
정상입니다. 테마 재적용은 스타일 값을 그 테마의 기본값으로 되돌립니다. **테마를 먼저 확정하고 나서** 이 화면을 손보세요.

**Q. 테마 1·2·3 을 이 화면에서 고를 수 없나요?**
없습니다. 페이지 빌더의 [테마 설정] 탭에서 고릅니다.

**Q. 원래 디자인으로 완전히 되돌리고 싶습니다.**
모든 칸을 비우고 저장하면 전부 기본값으로 복귀합니다.

---

## 8. 아직 지원하지 않는 것

- **색상 변경** — 사이트 설정을 쓰세요
- **테마를 여러 개 만들어 두고 전환** — 몰마다 활성 테마는 항상 하나입니다
- **테마 1·2·3 선택** — 페이지 빌더에서 합니다
- **실시간 미리보기** — 저장해야 결과를 볼 수 있습니다
- 개별 화면(상품 상세·장바구니 등)만 다르게 꾸미기
