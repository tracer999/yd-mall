# 브랜드 관리

고객 화면의 **브랜드관**(`/brands`)에 보이는 브랜드를 관리하는 곳입니다. 브랜드를 **만들고 · 지우고 · 순서와 로고를 정하고**, 영문명·별칭·한 줄 소개·브랜드 스토리·공식 여부처럼 브랜드에만 있는 항목을 채우고, 브랜드에 속한 상품을 담고 빼냅니다.

> **브랜드 작업은 전부 이 화면에서 끝납니다.** 예전에는 만들기·로고·순서를 카테고리 관리에서 했지만, 지금은 모두 이 화면으로 옮겨졌습니다. 카테고리 관리에는 더 이상 브랜드 탭이 없습니다.

> **브랜드는 "브랜드 유형의 카테고리"로 저장됩니다.** 그래서 상품 분류와 저장소를 공유하지만, **관리 화면은 완전히 분리**되어 있습니다. 상품 분류는 카테고리 관리, 브랜드는 이 화면입니다.

## 1. 작업 순서 한눈에

```
① 브랜드 관리 › [브랜드 등록]
        ↓  이름 · 로고 · 입점일 · 순서를 넣어 브랜드를 "만든다"
② 목록에서 그 줄을 바로 고치거나, [관리]로 들어가
        ↓  영문명 · 별칭 · 소개 · 스토리 · 공식 여부를 "채운다"
③ [관리] 화면 › [상품 추가]
        ↓  이 브랜드의 상품을 담는다
④ [집계 재계산]
        ↓  인기 점수 · 혜택 수를 갱신해야 고객 화면에 반영된다
```

**목록에서 하는 일과 [관리] 화면에서 하는 일**

| 작업 | 하는 곳 |
|---|---|
| 브랜드 **만들기 · 지우기** | **목록** ([브랜드 등록] / 줄의 [삭제]) |
| **순서 · 로고 · 브랜드명 · 입점일 · 사용 여부** | **목록** (줄에서 바로 고치고 [저장]) |
| **이 몰 표시** | **목록** (바꾸는 즉시 저장) |
| 영문명 · 별칭 · 초성 · 국가 | **[관리]** 화면 |
| 한 줄 소개 · 스토리 · 히어로 이미지 | **[관리]** 화면 |
| 공식 여부 · 브랜드관 확장 · SEO | **[관리]** 화면 |
| 상품 담기 · 빼기 | **[관리]** 화면 |

> **브랜드도 전 몰이 공유하는 한 벌입니다.** 한 몰에서 만든 브랜드가 다른 몰에도 함께 나타납니다. 다만 **상품 수 · 인기 점수 · 혜택 수 · 취급 카테고리는 지금 편집 중인 몰 기준**이라, 몰을 바꾸면 같은 브랜드라도 숫자가 달라집니다.

> **브랜드에는 계층이 없습니다.** 상품 카테고리와 달리 브랜드는 상위·하위를 두지 않고 전부 한 단계입니다.

### 2-1. 탭 — 사용중 / 미사용 포함 전체

목록 위에 탭이 두 개 있습니다. 브랜드는 전 몰이 함께 쓰는 목록이라 **1,400개가 넘는데, 이 몰에서 실제로 쓰는 것은 그중 일부**입니다.

| 탭 | 무엇이 보이나 |
|---|---|
| **사용중 브랜드** (기본) | **지금 편집 중인 몰에 상품이 담긴** 브랜드만 |
| **미사용 포함 전체** | 상품이 하나도 없는 브랜드까지 전부 |

탭 이름 옆 괄호 안 숫자가 각각의 건수입니다. 검색어·공식 여부·정렬 조건은 탭을 옮겨도 그대로 유지됩니다.

> **"방금 만든 브랜드가 목록에 없어요"의 1순위 원인이 이것입니다.** 새로 만든 브랜드는 상품이 0개라 기본 탭(사용중)에 나오지 않습니다. **[미사용 포함 전체]** 탭에서 찾으세요.

> **순서는 [사용중 브랜드] 탭에서만 고칠 수 있습니다.** 전체 탭에서는 순서가 회색 숫자로 보이기만 합니다. 고객 화면에 나오지도 않는 빈 브랜드의 순서를 매기는 것은 의미가 없기 때문입니다.

### 2-2. 검색과 정렬

| 항목 | 값 |
|---|---|
| **검색어** | 브랜드명 · 영문명 · **초성**으로 찾습니다. (예: `ㄴㅇㅋ` → 나이키) |
| **공식 여부** | 전체 / 공식 브랜드만 / 일반 브랜드만 |
| **정렬** | 상품 많은순 / 인기순 / 가나다순 / 입점 최신순 / **출력 순서** |

> **순서를 정리할 때는 정렬을 [출력 순서]로 두세요.** 지금 매겨진 순서대로 줄이 늘어서므로 앞뒤를 보며 고치기 쉽습니다.

### 2-3. 표의 각 칸 — 대부분 줄에서 바로 고칩니다

| 칸 | 뜻 | 고칠 수 있나 |
|---|---|---|
| **순서** | 숫자가 작을수록 앞에 나옵니다. | **사용중 탭에서만** |
| **로고** | 정사각형 권장, 최대 2MB (jpg·png·webp). 없으면 `NO LOGO`. | ⭕ 파일 선택 |
| **브랜드명** | 고객에게 보이는 이름. `공식` 배지와 영문명이 함께 표시됩니다. | ⭕ |
| **입점일** | 신규 입점 브랜드 판정 기준일 | ⭕ |
| **대표 카테고리** | 이 브랜드 상품이 가장 많이 속한 카테고리 (자동 계산) | ❌ |
| **상품** | 이 몰에서 이 브랜드에 속한 상품 수 (**실시간**) | ❌ |
| **인기 점수** | 자동 계산 점수 (**집계 캐시**) | ❌ |
| **구분** | `브랜드` (제조사·자체 브랜드) 또는 `입점몰` (도매·셀러) | [관리] 화면에서 |
| **사용 여부** | **전 몰 공통** 스위치. 끄면 어디에서도 쓰이지 않습니다. | ⭕ |
| **이 몰 표시** | 지금 편집 중인 몰에서만 숨김/표시. **바꾸는 즉시 저장**됩니다. | ⭕ (상품 있는 브랜드만) |
| **관리** | 영문명·스토리 등 확장 항목 편집 + 상품 담기 화면으로 이동 | — |

> **줄에서 고친 값은 그 줄의 [저장]을 눌러야 반영됩니다.** 순서·로고·이름·입점일·사용 여부를 한 줄에서 함께 고치고 [저장] 한 번이면 됩니다. **[이 몰 표시]만 예외로 즉시 저장**됩니다.

> **[저장]은 그 줄의 다른 값을 건드리지 않습니다.** 영문명·스토리·공식 여부 같은 [관리] 화면의 값은 그대로 남습니다. 로고도 새 파일을 고르지 않으면 기존 로고가 유지됩니다.

**화면 위쪽 버튼**

- **[브랜드 등록]** — 새 브랜드를 만듭니다(팝업).
- **[사용여부 저장]** — 지금 페이지에 보이는 줄들의 **사용 여부만** 한 번에 저장합니다. 이름·순서·로고는 건드리지 않습니다.
- **[집계 재계산]** — 인기 점수·혜택 수·대표 카테고리를 다시 계산합니다 (아래 3장).

### 2-4. 브랜드 등록

**[브랜드 등록]**을 누르면 팝업이 뜹니다.

| 입력란 | 필수 | 설명 |
|---|---|---|
| **브랜드명** | 필수 | 고객에게 보이는 이름입니다. |
| **입점일** | 선택 | 기본값은 오늘입니다. |
| **브랜드 로고** | 선택 | 정사각형 권장, 최대 2MB. |
| **출력 순서** | 선택 | 비워 두면 맨 뒤로 붙습니다. |
| **사용** | — | 기본은 켜짐입니다. |

> **초성 검색어는 브랜드명에서 자동으로 만들어집니다.** 등록 직후부터 `ㄴㅇㅋ` 같은 초성 검색에 걸립니다. 영문명·별칭·스토리는 등록 후 **[관리]** 화면에서 채우세요.

> **등록한 브랜드는 [미사용 포함 전체] 탭에 있습니다.** 상품을 담기 전까지는 사용중 탭에 나오지 않습니다.

### 2-5. 브랜드 삭제

줄 오른쪽의 **[삭제]**를 누릅니다. 확장 속성(영문명·스토리 등)도 함께 지워집니다.

| 상황 | 결과 |
|---|---|
| 상품이 하나도 없다 | 삭제됩니다 |
| **상품이 이 브랜드를 쓰고 있다** | `이 브랜드를 참조하는 상품이 (다른 몰 포함) N개 있어 삭제할 수 없습니다.` |

> **"다른 몰 포함"에 주의하세요.** 화면의 상품 수가 `0` 이어도 **다른 몰의 상품이 이 브랜드를 쓰고 있으면** 삭제되지 않습니다. 상품 수는 편집 중인 몰만 세기 때문입니다. 고장이 아니라 정상 동작입니다.

## 3. 집계 재계산 — 잊으면 숫자가 안 맞습니다

목록 위의 파란 안내에 **마지막 집계 시각**이 적혀 있습니다.

> **인기 점수 · 혜택 수 · 대표 카테고리 · 취급 카테고리는 "저장해 둔 계산 결과"입니다.** 상품을 담거나 빼도 **저절로 갱신되지 않습니다.** 화면 오른쪽 위 **[집계 재계산]**을 눌러야 반영됩니다.

> **목록의 [상품] 칸과 탭 건수는 예외로 실시간입니다.** 상품을 담으면 바로 반영되고, [사용중 브랜드] 탭에도 즉시 나타납니다. 집계를 돌리지 않아 생기는 "탭에는 있는데 상품 수는 0" 같은 어긋남이 없도록 맞춰 두었습니다.

상품을 잔뜩 넣었는데 **브랜드관 화면**에서 "상품 0개"로 보인다면 대부분 **집계 재계산을 누르지 않은 것**입니다.

- 재계산은 **지금 편집 중인 몰에 대해서만** 돕니다. 다른 몰 숫자는 그 몰로 바꿔 다시 눌러야 합니다.
- 끝나면 `집계를 갱신했습니다. 브랜드 N건` 이 표시됩니다.
- `아직 집계가 돌지 않았습니다` 라고 나오면 한 번도 실행하지 않은 상태입니다.

## 4. 브랜드 편집 화면

목록에서 브랜드를 누르면 편집 화면이 열립니다. 맨 위에 로고·브랜드명·상품 수·인기 점수·진행 중 혜택 수가 요약되고, **[브랜드관 보기]** 로 고객 화면을 새 창에서 확인할 수 있습니다.

### 4-1. 기본정보

| 입력란 | 필수 | 설명 | 예시 |
|---|---|---|---|
| **브랜드명 (한글)** | 필수 | 고객에게 보이는 이름입니다. | `나이키` |
| **영문명** | 선택 | 브랜드명 아래 작게 표시되고 검색에도 잡힙니다. | `NIKE` |
| **별칭 (콤마 구분)** | 선택 | 브랜드 검색에서 함께 매칭될 다른 이름들. | `나이키코리아, 나이키 공식` |
| **초성 인덱스** | 선택 | 브랜드관의 초성 색인(ㄱ·ㄴ·ㄷ…) 위치. **비우면 브랜드명에서 자동 산출**됩니다. | `ㄴ` |
| **국가** | 선택 | | `대한민국` |
| **한 줄 소개** | 선택 | 브랜드관 헤더와 검색 결과에 노출됩니다. 최대 200자. | `Just Do It` |
| **입점일** | 선택 | '신규 입점 브랜드' 판정 기준일입니다. | `2026-03-01` |
| **노출** | — | `노출` / `숨김` | |

> **초성 인덱스는 특별한 이유가 없으면 "자동 (브랜드명에서 산출)" 로 두세요.** 직접 고른 값이 있으면 그것이 우선합니다. 잘못 고르면 고객이 브랜드관 색인에서 브랜드를 못 찾습니다. 칸 아래에 **현재 초성 검색어**가 표시되니 참고하세요.

> **입점일은 시스템이 추정해 채워 둔 값일 수 있습니다.** 그 브랜드의 최초 상품 등록일로 백필된 경우가 있으니, 실제 입점일과 다르면 직접 고쳐 주세요.

### 4-2. 운영정보

> **모든 브랜드는 기본 브랜드관이 자동으로 생깁니다.** 아래 설정은 "그 위에 무엇을 더 얹을지"를 정하는 것입니다.

| 입력란 | 값 | 동작 |
|---|---|---|
| **공식 브랜드 여부** | `일반` / `공식 (배지 노출)` | 공식으로 두면 목록과 브랜드관에 **공식 배지**가 붙습니다. |
| **공식 브랜드관 확장** | `사용 안 함` / `사용 (히어로·스토리)` | 켜면 브랜드관에 **히어로 이미지와 브랜드 스토리**가 추가로 노출됩니다. |
| **히어로 이미지 URL** | 경로 입력 | 예: `/uploads/brands/hero-xxx.jpg` |
| **브랜드 스토리** | 여러 줄 | **공식 브랜드관 확장이 켜져 있을 때만 노출**됩니다. |
| **입점 셀러명** | 텍스트 | 상품의 공급처 값에서 자동으로 채워졌을 수 있습니다. |
| **구분** | `브랜드 (제조사·자체 브랜드)` / `입점몰 (도매·셀러)` | 예: 나이키 = 브랜드 / 오너클랜 = 입점몰 |

> **스토리를 열심히 써 놓고 안 보인다면 [공식 브랜드관 확장]이 꺼져 있는 것입니다.** 스토리와 히어로 이미지는 이 스위치가 켜져야 화면에 나옵니다.

### 4-3. SEO

| 입력란 | 설명 |
|---|---|
| **SEO 제목** | 검색엔진에 노출될 제목. 최대 200자. |
| **SEO 설명** | 검색엔진에 노출될 설명. 최대 300자. |

### 4-4. 취급 카테고리 (읽기 전용)

이 브랜드 상품들이 어느 카테고리에 속해 있는지를 **자동으로 뽑아** 보여 줍니다. 직접 고칠 수 없고, **[집계 재계산]** 을 눌러야 갱신됩니다.

여기까지 입력했으면 아래의 **[저장]**을 누릅니다.

> **[저장]을 눌러야 반영됩니다.** 화면 아래쪽 "이 브랜드의 상품" 영역은 저장과 무관하게 **즉시 반영**되지만, 위쪽 입력값들은 저장을 눌러야 합니다.

---

## 5. 로고 이미지

**목록의 로고 칸**에서 파일을 고르고 그 줄의 **[저장]**을 누릅니다. 등록 팝업에서도 올릴 수 있습니다.

- 정사각형 권장, 최대 2MB, jpg·png·webp
- 파일을 고르면 저장 전에 미리보기가 바뀝니다. **[저장]을 눌러야 실제로 반영**됩니다.
- 새 파일을 고르지 않으면 **기존 로고가 그대로 유지**됩니다 (줄의 다른 값만 고쳐도 로고는 사라지지 않습니다)
- 로고가 없으면 목록에 `NO LOGO` 로 표시됩니다

## 6. 이 브랜드의 상품 담기 · 빼기

편집 화면 맨 아래 **이 브랜드의 상품** 영역에서 합니다. 제목 옆에 **현재 편집 몰 이름과 총 개수**가 표시됩니다.

**담는 순서**

1. **[상품 추가]**를 누릅니다. (버튼 옆 `미설정 N` 은 아직 브랜드가 지정되지 않은 상품 수입니다.)
2. 조건을 넣고 **[검색]** 합니다.

   | 검색 조건 | 값 |
   |---|---|
   | **검색어** | 상품명 또는 상품코드 |
   | **재고** | 전체 / 재고 있음 / 재고 없음 |
   | **노출** | 전체 / 노출 / 숨김 / 회원전용 |

3. 담을 상품을 체크합니다. **[결과 전체 선택]** 도 쓸 수 있습니다.
4. **[선택 배정]**을 누릅니다.

빼낼 때는 상품 줄 오른쪽의 **[제거]**를 누릅니다. **연결만 끊길 뿐 상품은 삭제되지 않습니다.**

> **팝업에는 "브랜드가 아직 지정되지 않은 상품"만 나옵니다.** 이미 다른 브랜드에 속한 상품은 검색해도 나오지 않습니다. **브랜드를 바꾸려면 원래 브랜드에서 [제거]한 뒤 다시 담으세요.**

> 검색 결과는 **최대 100건**까지입니다. `결과가 100개에 도달했습니다` 안내가 뜨면 검색어로 좁히세요.

> **상품을 담고 나서 [집계 재계산]을 잊지 마세요.** 담기만 해서는 목록의 상품 수와 브랜드관 화면이 갱신되지 않습니다.

## 9. 자주 막히는 상황

**Q. 브랜드를 만들었는데 목록에 없습니다.**
새 브랜드는 상품이 0개라 기본 탭(**사용중 브랜드**)에 나오지 않습니다. **[미사용 포함 전체]** 탭에서 찾으세요. 그래도 없다면 검색어·공식 여부 필터에 걸린 것입니다.

**Q. 순서 칸이 회색 숫자라 고칠 수 없습니다.**
**[미사용 포함 전체]** 탭입니다. 순서는 **[사용중 브랜드]** 탭에서만 고칩니다.

**Q. 상품을 담았는데 브랜드관 화면의 숫자가 0입니다.**
**[집계 재계산]**을 누르지 않았습니다. 목록 오른쪽 위 버튼입니다. (관리 목록의 [상품] 칸은 실시간이라 이미 맞게 보입니다.)

**Q. 브랜드 스토리를 썼는데 고객 화면에 안 보입니다.**
**[공식 브랜드관 확장]** 이 `사용 안 함` 입니다. `사용 (히어로·스토리)` 로 바꾸고 저장하세요.

**Q. 줄에서 이름을 고쳤는데 저장되지 않습니다.**
그 줄의 **[저장]**을 눌러야 합니다. 화면 위의 **[사용여부 저장]**은 사용 여부만 저장합니다.

**Q. 상품 추가 팝업에서 상품이 검색되지 않습니다.**
그 상품은 **이미 다른 브랜드에 속해 있습니다.** 원래 브랜드에서 먼저 **[제거]**하세요.

**Q. 삭제가 안 됩니다.**
상품이 이 브랜드를 쓰고 있습니다. **[관리]** 화면에서 상품을 **[제거]**한 뒤 다시 시도하세요. 상품 수가 0인데도 막히면 **다른 몰의 상품**이 쓰고 있는 것입니다.

**Q. 예전에 카테고리 관리에서 브랜드를 관리했는데 탭이 없어졌습니다.**
브랜드 탭은 이 화면으로 옮겨졌습니다. 순서·로고·이름·입점일·사용 여부·이 몰 표시·삭제·등록 전부 여기서 합니다.

**Q. 몰을 바꿨더니 같은 브랜드의 상품 수가 달라졌습니다.**
정상입니다. 브랜드 자체는 전 몰 공통이지만 **상품은 몰별**이라 숫자가 다릅니다.

**Q. 브랜드명을 고쳤더니 다른 몰에서도 바뀌었습니다.**
정상입니다. 브랜드는 전 몰이 공유하는 한 벌입니다.

**Q. 초성 검색이 안 됩니다.**
브랜드명을 바꾼 뒤 **저장**하면 초성 검색어가 다시 만들어집니다. 편집 화면의 `현재 초성 검색어` 표시를 확인하세요.

## 10. 아직 지원하지 않는 것

- **브랜드 계층** — 브랜드는 상위·하위 없이 전부 한 단계입니다
- **엑셀/CSV 일괄 등록**
- **중복 브랜드 병합** — 같은 브랜드를 두 개 만들었다면 상품을 한쪽으로 옮긴 뒤 남은 쪽을 지워야 합니다
- **자동 집계 갱신** — 반드시 [집계 재계산]을 눌러야 합니다
- **취급 카테고리 직접 지정** — 상품에서 자동으로 도출되는 읽기 전용 값입니다
