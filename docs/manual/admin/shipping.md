# 배송 관리

**배송할 주문**을 모아 보고, **택배 송장 번호**를 입력하는 메뉴입니다.

---

## 이 메뉴로 무엇을 하나요?

- **결제가 완료되었지만 아직 배송하지 않은 주문**을 한곳에서 볼 수 있습니다.
- 각 주문에 **택배사·송장 번호**를 입력하면, 고객이 주문 조회나 문자/이메일로 **배송 추적**을 할 수 있습니다.

---

## 화면 구성

- **배송 대기 목록:** 상태가 "결제완료" 또는 "상품준비중"인 주문이 보입니다. (시스템 설정에 따라 다름)
- **주문 번호, 주문자, 수령인, 배송지, 주문일** 등이 표시됩니다.
- **송장 입력** 버튼 또는 입력란을 통해 **택배사**와 **송장 번호**를 넣을 수 있습니다.

---

## 송장 입력 방법

1. 배송할 **주문 한 건**을 선택하거나, 해당 행의 **송장 입력**을 클릭합니다.
2. **택배사**를 선택합니다. (우체국, CJ대한통운, 한진 등)
3. **송장 번호**(운송장 번호)를 입력합니다. 택배사에서 부여한 숫자만 정확히 넣습니다.
4. **저장** 또는 **등록**을 누릅니다.

저장이 되면 주문 상태가 **"배송중"**으로 바뀌고, 고객에게 배송 안내 문자가 가거나 주문/마이페이지에서 조회가 가능해질 수 있습니다. (시스템에 따라 다름)

**송장은 한 건씩 입력합니다.** 여러 주문을 체크해서 한 번에 송장을 넣는 **일괄 송장 입력 기능은 없습니다.**

---

## 배송완료 처리

상품이 고객에게 도착하면 **배송완료** 버튼으로 상태를 바꿉니다.

**⚠️ 배송완료 처리는 제때 해야 합니다.**

> 고객이 **반품을 신청할 수 있는 기간(7일)** 은 **배송완료 처리한 시점부터** 계산됩니다.
>
> 배송완료 처리를 늦게 하면 **반품 가능 기간도 그만큼 뒤로 밀립니다.** 실제로는 도착한 지 열흘이 지났는데 아직 반품이 가능한 상태로 남을 수 있습니다.

배송이 끝난 주문은 **바로바로 배송완료로 바꿔 두세요.**

---

## 배송비 정책

**배송비 정책** 화면에서 배송비 계산 기준을 설정합니다. **몰마다 따로** 저장됩니다.

| 항목 | 설명 |
|------|------|
| **기본 배송비** | 주문에 붙는 기본 배송 금액 |
| **무료배송 기준액** | 이 금액 이상 사면 배송비가 무료가 됩니다. |
| **제주·도서산간 추가 배송비** | 특정 지역에 추가로 붙는 금액 |
| **우편번호 대역** | 추가 배송비를 받을 지역을 우편번호 범위로 **추가·삭제**합니다. |

### 꼭 알아야 할 3가지

**① 무료배송 판정은 "쿠폰·적립금을 빼기 전" 상품 금액 기준입니다.**
예를 들어 무료배송 기준이 5만 원이고 고객이 5만 원어치를 담은 뒤 쿠폰으로 1만 원을 할인받아 4만 원을 결제해도, **무료배송이 적용됩니다.**

**② 무료배송이 되어도 제주·도서산간 추가 배송비는 그대로 청구됩니다.**
"무료배송인데 왜 배송비가 붙었냐"는 문의는 대부분 이 경우입니다. 추가 배송비는 무료배송과 별개로 계산됩니다.

**③ 정책을 꺼 두면 배송비가 전액 0원이 됩니다.**
배송비를 받으려면 **정책이 켜져 있어야 합니다.** 배송비가 갑자기 0원으로 나온다면 정책이 꺼져 있는지 먼저 확인하세요.
