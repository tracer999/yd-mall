# 방문자 통계

**쇼핑몰에 얼마나 많은 사람이 방문했는지** 기간별로 확인하는 메뉴입니다.

---

## 무엇을 보나요?

- **선택한 기간** 동안 **몇 명이 방문했는지** (중복 제거한 수)
- **날짜별·주별** 방문자 수 **추이** (그래프로 보이는 경우가 많음)

관리자 사이트(`/admin`) 접속은 보통 **방문자 수에 포함되지 않습니다.** 고객이 쇼핑몰 메인·상품 페이지 등에 접속한 수만 집계됩니다.

---

## 어떻게 쓰나요?

1. **방문자 통계** 메뉴에 들어갑니다.
2. **기간**을 선택합니다. (오늘, 최근 7일, 최근 30일, 직접 날짜 지정 등)
3. **조회** 또는 **적용** 버튼을 누르면 해당 기간의 **방문자 수**와 **차트**가 나옵니다.

---

## 활용 예

- **이벤트·홍보 전후**로 방문자가 늘었는지 비교할 수 있습니다.
- **요일별·시간대별** 그래프가 있다면, 언제 접속이 많은지 파악해 운영·광고에 참고할 수 있습니다.
