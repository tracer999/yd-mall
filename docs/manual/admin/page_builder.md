# 메인 화면(페이지) 만들기

쇼핑몰의 **메인 화면(홈)** 을 직접 구성하는 메뉴입니다. 홈뿐 아니라 **신상품 같은 랜딩 페이지**도 같은 방법으로 편집할 수 있습니다.

---

## 기본 개념: 섹션을 쌓아 만듭니다

메인 화면은 **섹션**이라는 블록을 위에서부터 차례로 **쌓아서** 만듭니다. (예: 맨 위에 히어로 배너 → 그 아래 퀵메뉴 → 그 아래 베스트 랭킹 …)

1. 화면 상단에서 **편집할 페이지**를 고릅니다. (홈 / 신상품 등)
2. **섹션 추가**로 블록을 넣습니다.
3. 각 섹션을 열어 **내용과 노출 조건**을 설정합니다.
4. **순서**를 조정합니다.
5. **발행**합니다. ← **이걸 눌러야 고객 화면에 나갑니다.**

---

## 홈 화면은 세 가지 방식으로 만듭니다

**홈**을 편집하면 상단에 **[테마 설정] / [페이지 설정]** 탭이 나타납니다. (신상품 같은 랜딩 페이지는 이 탭 없이 아래 "직접설정"만 씁니다.)

| 방식 | 위치 | 무엇을 하나 | 히어로(상단 큰 영역) |
|------|------|-------------|----------------------|
| **테마 설정** | [테마 설정] 탭 | 테마 1·2·3 중 하나를 골라 메인을 통째로 다시 구성. 그 테마 기본 구성(히어로 + 베스트·쇼핑특가·베스트 카테고리·베스트 브랜드)으로 교체·발행. 내비·메뉴·디자인 토큰도 그 테마 기준으로 되돌아감 | **삭제 후 재생성** |
| **이지모드** | [페이지 설정] 탭 → 이지모드 | 미리 만든 3종 구성 중 하나로 캐러셀만 교체·발행 | **그대로 보존** |
| **직접설정** | [페이지 설정] 탭 → 직접설정 (기본) | 섹션을 하나씩 추가·편집 (아래 설명) | 편집한 대로 |

> ⚠️ **[테마 설정] 탭은 현재 홈 섹션을 전부 삭제**하고 테마 기본 구성으로 교체합니다. 지금 홈을 그대로 두고 부분만 바꾸려면 [페이지 설정] 탭을 쓰세요. 또한 테마를 다시 적용하면 **테마 설정(`/admin/theme-settings`)에서 다듬은 모서리·글꼴·간격 값도 초기화**됩니다.

### 이지모드 번들 3종

[페이지 설정] → 이지모드에는 3개 카드가 있고, "이 구성 적용"을 누르면 **상단 히어로는 그대로 둔 채** 그 아래 캐러셀만 아래 구성으로 교체·발행됩니다. (내비·메뉴·테마는 건드리지 않습니다.)

| 번들 | 구성(위→아래) |
|------|----------------|
| **상품 중심** | 특가 → 신상품 → 베스트 카테고리 → 베스트 브랜드 |
| **혜택 강조** | 퀵메뉴 → 프로모션 배너(기획전) → 특가 → 베스트 카테고리 |
| **심플** | 베스트 카테고리 → 베스트 브랜드 → 신상품 |

---

## ⚠️ 가장 중요: "발행"을 눌러야 반영됩니다

> **편집만 하고 저장해도 고객 화면은 그대로입니다.**
>
> - **미리보기** = 지금 **편집 중인** 내용
> - **실제 고객 홈** = 마지막으로 **발행된** 내용
>
> 다 고쳤으면 **반드시 "발행" 버튼**을 누르세요.

발행 이력이 남으므로, 잘못 발행했다면 **이전 버전으로 되돌리기(롤백)** 를 할 수 있습니다.

---

## 섹션 둘러보기

어떤 섹션인지 이름만으로는 감이 안 올 때, **"섹션 둘러보기"** 버튼을 누르면 **실제 상품·배너 데이터로 미리 그려진 섹션들**을 눈으로 보고 고를 수 있습니다. 처음 홈을 구성할 때 여기서부터 시작하는 것이 가장 빠릅니다.

---

## 섹션 종류 (18가지)

| 섹션 | 어떤 모습인가요 |
|------|------------------|
| **히어로** | 맨 위 큰 배너 (슬라이드) |
| **테마 히어로** | 테마가 까는 상단 대형 영역 (표현: 쇼케이스/배너/에디토리얼) |
| **상품 그리드** | 상품을 격자로 나열 |
| **베스트/랭킹** | 인기 상품 순위 |
| **카테고리별 상품** | 카테고리를 카드로 보여 줌 |
| **카카오 상담 CTA** | 카카오 상담 유도 영역 |
| **상품 캐러셀** | 상품을 옆으로 넘기는 줄 |
| **쇼핑특가 캐러셀** | 쇼핑특가 상품을 옆으로 넘기는 줄 |
| **브랜드 캐러셀** | 브랜드를 옆으로 넘기는 줄 |
| **랭킹 탭** | 기간 탭이 달린 순위 영역 |
| **프로모션 배너** | 중간에 넣는 띠 배너 |
| **혜택 벤토** | 혜택을 타일로 묶어 보여 줌 |
| **퀵 메뉴** | 아이콘 바로가기 모음 |
| **최근 본 상품** | 고객이 최근 본 상품 |
| **커스텀 HTML** | 원하는 내용을 직접 작성 |
| **카테고리별 신상품** | 카테고리 기준 신상품 |
| **브랜드별 신상품** | 브랜드 기준 신상품 |
| **신규 입점 브랜드** | 새로 입점한 브랜드 |

---

## 상품을 보여 주는 섹션은 "상품 그룹"을 씁니다

상품 그리드·상품 캐러셀 같은 섹션은 **어떤 상품을 보여 줄지**를 **상품 그룹**으로 지정합니다. 상품 그룹은 두 가지 방식으로 만듭니다.

| 방식 | 설명 |
|------|------|
| **직접 고르기** | 보여 줄 상품을 하나씩 검색해서 담습니다. |
| **조건으로 자동** | 조건에 맞는 상품이 자동으로 채워집니다. 조건: **뱃지 · 카테고리 · 최소 할인율 · 재고** |

행사처럼 상품이 자주 바뀌면 **조건으로 자동**이 편하고, 꼭 그 상품만 보여 주고 싶으면 **직접 고르기**가 확실합니다.

---

## 섹션별 노출 조건

각 섹션마다 아래를 정할 수 있습니다.

- **노출 기간:** 시작일·종료일을 지정하면 그 기간에만 보입니다.
- **PC / 모바일 노출:** 어느 화면에서 보일지 각각 켜고 끕니다.

---

## "프론트 미노출" 진단과 설정 이동

직접설정 섹션 목록에서, 섹션이 실제로는 고객 화면에 안 나오면 **"프론트 미노출"** 배지와 그 이유가 함께 표시됩니다.

- 이유가 **비활성·노출기간·PC/모바일 꺼짐**이면 그 자리(섹션 설정)에서 바로 고칩니다.
- 이유가 **"보여 줄 데이터가 없어서"**(예: 특가 캐러셀인데 특가가 하나도 없음)이면, 데이터를 채울 화면으로 가는 **"〇〇로 이동"** 버튼이 함께 나옵니다. (예: 특가 캐러셀 → [특가 관리로 이동], 브랜드/카테고리 섹션 → [카테고리 관리로 이동], 상품 섹션 → [상품 그룹 관리로 이동])

데이터를 채운 뒤 섹션을 다시 저장하면 배지가 사라집니다.

---

## 순서 바꾸기

**위/아래 버튼**으로 섹션 순서를 옮깁니다.

- **드래그로 옮기는 기능은 없습니다.**
- **예약 발행 기능도 없습니다.** 미래 시점에 자동으로 바꾸고 싶다면, 섹션의 **노출 기간**을 활용하세요.

---

## 퀵메뉴 설정

퀵메뉴는 **주소를 직접 타이핑하지 않습니다.** **연결할 페이지를 목록에서 골라** 지정합니다. 오타로 인한 링크 오류가 생기지 않습니다.
