# 기획전 · 전문관 관리

여러 상품을 하나의 주제로 모아 보여 주는 **전시 랜딩 화면**을 만드는 메뉴입니다. "가을 보양 기획전", "설 선물세트 모음"처럼 **기간을 정해 운영하는 기획전**과, 종료일 없이 **상시 운영하는 전문관**을 이 화면 한 곳에서 함께 관리합니다.

> **기획전은 "전시 화면"입니다.** 고객은 기획전에서 상품을 둘러보고, **실제 결제는 각 상품 상세 화면에서** 합니다. 기획전 화면에서 바로 결제하지 않습니다.

---

## 1. 시작하기 전에 — 준비물

기획전을 만들기 전에 아래가 준비되어 있어야 합니다. 준비 없이 시작하면 중간에 막힙니다.

| 준비물 | 왜 필요한가 | 어디서 만드나 |
|---|---|---|
| **담을 상품** | 기획전은 이미 등록된 상품을 골라 담는 화면입니다. 상품을 여기서 새로 만들 수는 없습니다. | 상품 관리 |
| **이미지 2종** | **목록 썸네일**과 **공유용** 이미지. (PC·모바일 대표 이미지는 전문관에만 쓰입니다 — 아래 3-3 참고) | 이미지 편집 도구 |
| **브랜드 분류** (유형이 `브랜드`인 경우) | 대상 브랜드를 지정해야 그 브랜드 상품만 담을 수 있습니다. | 카테고리 관리 |
| **카테고리** (유형이 `카테고리`인 경우) | 대상 카테고리를 지정해야 그 분류의 상품만 담을 수 있습니다. | 카테고리 관리 |

> **지금 편집 중인 몰을 반드시 확인하세요.** 기획전은 **몰별로 따로 저장**됩니다. 관리자 화면 상단의 편집 몰이 A몰인데 B몰에 만들 생각이었다면, 저장해도 B몰에는 나타나지 않습니다.

## 2. 전체 흐름 한눈에 보기

```
① 유형 고르기 + 기본 정보 입력 → 저장   (여기까지 해야 기획전이 "생깁니다")
        ↓
② 섹션(탭) 만들기                       (예: 베스트 / 신상품 / 1+1)
        ↓
③ [상품 선택] 팝업에서 골라 담기        (여러 개를 한 번에 + 배지·코멘트 달기)
        ↓
④ 상태를 "발행"으로 변경                (+ 목록 노출 켜기)
        ↓
⑤ 고객 화면에서 확인
```

**①번 저장을 먼저 해야 ②③이 가능합니다.** 등록 화면에서는 섹션·상품 영역이 아직 열리지 않습니다. 저장 후 이어지는 수정 화면에서 계속 작업하세요.

### 3-1. 이름과 설명

| 입력란 | 필수 | 설명 | 예시 |
|---|---|---|---|
| **기획전명** | 필수 | 고객에게 보이는 제목입니다. 최대 200자. | `가을 보양 기획전` |
| **주소용 이름(slug)** | 선택 | 기획전 주소에 들어가는 영문 이름입니다. **비워 두면 자동으로 만들어집니다.** | `autumn-boyang` |
| **짧은 설명** | 선택 | 목록에서 제목 아래 한 줄로 보입니다. 최대 500자. | `환절기 건강을 챙기는 30가지` |
| **상세 설명** | 선택 | 기획전 화면 상단에 들어가는 소개글입니다. | 여러 줄 입력 가능 |

> **주소용 이름은 나중에 바꾸지 마세요.** 고객에게 링크를 뿌린 뒤 이 값을 바꾸면 **기존 링크가 전부 깨집니다.** 문자·카카오톡으로 발송한 링크도 마찬가지입니다.

### 3-2. 유형 · 상태 · 기간

**기획전 유형** — **가장 먼저 고르는 값**입니다. 성격을 분류할 뿐 아니라 **어떤 상품을 담을 수 있는지(편성 범위)를 결정**합니다.

| 유형 | 담을 수 있는 상품 | 목록 노출 |
|---|---|---|
| **브랜드** | 지정한 **그 브랜드의 상품만** | 기획전 목록 |
| **카테고리** | 지정한 **그 분류와 하위 분류의 상품만** | 기획전 목록 |
| 테마 / 시즌 / 콜라보 / 방송연계 | **전체 상품** (자유롭게) | 기획전 목록 |
| **전문관 (상시 · 종료일 비움)** | 전체 상품 | **기획전 목록에서 빠지고 전문관 메뉴에만** — 아래 6장 |

**유형을 고르면 오른쪽에 그 유형에 필요한 설정이 나타납니다.**

| 고른 유형 | 나타나는 설정 | 입력 방법 |
|---|---|---|
| **브랜드** | **대상 브랜드** | 입력칸을 **클릭만 해도** 이 몰에 상품이 있는 브랜드가 상품 수와 함께 목록으로 뜹니다. 브랜드가 많으면 이름·초성으로 검색해 좁히세요. |
| **카테고리** | **대상 카테고리** | **대분류 → 중분류 → 소분류** 순서로 고릅니다. 앞 단계를 고르면 다음 단계가 열립니다. **대분류만 골라도 그 아래 상품이 전부 대상**이 됩니다. |
| 그 외 | 없음 (전체 상품 안내만 표시) | — |

> **유형을 바꾸면 이전 유형의 설정은 지워집니다.** 브랜드 기획전으로 만들었다가 테마로 바꾸면 대상 브랜드가 해제됩니다. 화면에 보이지 않는 조건이 상품 편성을 계속 막지 않도록 하기 위해서입니다.

> **유형을 먼저 정하고 상품을 담으세요.** 상품을 다 담은 뒤 유형을 바꾸면, 이미 담긴 상품은 그대로 남지만 **새로 담을 수 있는 범위만 바뀝니다.** 범위 밖 상품이 남아 있으면 직접 [제외]해야 합니다.

**상태**

| 상태 | 의미 |
|---|---|
| **임시저장** | 준비 중. 고객에게 보이지 않습니다. 새로 만들면 기본값입니다. |
| **발행** | 고객에게 공개됩니다. |
| **숨김** | 공개를 잠시 내립니다. 내용은 그대로 남습니다. |

**기간**

- **시작일**은 **필수**입니다. 날짜와 시각을 함께 지정합니다.
- **종료일**은 비울 수 있습니다. **비우면 종료 없이 계속 진행**됩니다. 전문관은 반드시 비웁니다.
- 기간에 따라 목록에 **예정 / 진행중 / 종료** 가 자동으로 표시됩니다.

**상세 템플릿** — 기획전 상세 화면의 생김새입니다.

> 현재 **`기본 탭형`만 실제로 동작합니다.** 목록에 `(2차)`라고 적힌 이미지 스토리형·카테고리 매장형·브랜드관형을 골라도 화면은 기본 탭형으로 나옵니다. 혼란을 피하려면 **기본 탭형을 그대로 두세요.**

### 3-3. 이미지

| 이미지 | 어디에 보이나 | 없으면 |
|---|---|---|
| **목록 썸네일** | 기획전 목록의 작은 카드 이미지 | 빈 칸으로 보입니다 |
| **공유 이미지** | 카카오톡·SNS로 링크를 보냈을 때 뜨는 미리보기 | 미리보기가 안 뜨거나 엉뚱한 이미지가 잡힙니다 |
| **PC 대표 이미지** | **전문관 화면 상단**에만 쓰입니다. 기획전 상세에는 나오지 않습니다. | 전문관 상단이 비어 보입니다 (기획전은 영향 없음) |
| **모바일 대표 이미지** | 전문관의 모바일 전용 상단 이미지 | **PC 이미지가 대신 쓰입니다** |

> **기획전 상세 화면에는 상단 이미지 배너가 없습니다.** 예전에는 PC 대표 이미지가 상세 맨 위에 크게 깔렸지만, 지금은 그 자리에 **추천 상품**이 들어갑니다(아래 5-5 참고). 이미지 한 장보다 살 수 있는 상품을 먼저 보여 주는 편이 낫기 때문입니다.
>
> 기획전에서 **PC·모바일 대표 이미지를 채울 필요는 없습니다.** 다만 공유 이미지를 비워 두면 이 이미지가 공유 미리보기로 대신 쓰입니다.

**기획전 목록 맨 위의 큰 배너는 이 화면에서 만들지 않습니다.** 배너 관리에서 등록합니다 — 아래 9장 참고.

수정 화면에서는 각 이미지 옆에 **삭제 체크박스**가 있습니다. 체크하고 저장하면 그 이미지가 지워집니다.

### 3-4. 노출 설정 (체크박스 3개)

| 항목 | 켜면 |
|---|---|
| **목록 노출** | 기획전 목록 화면에 나타납니다. |
| **검색 노출** | 사이트 내 검색 결과에 잡힙니다. |
| **공유 버튼** | 고객 화면에 SNS 공유 버튼이 표시됩니다. |

> **고객에게 보이려면 `상태 = 발행` + `목록 노출 = 켜짐` 두 가지가 모두** 필요합니다. 하나만 해 두고 "안 보인다"고 하는 경우가 가장 많습니다.

### 3-5. 종료 후 정책

기간이 끝난 뒤 기획전을 어떻게 할지 정합니다.

**종료 후 접근**

| 값 | 동작 |
|---|---|
| **접근 허용** | 기간이 지나도 화면이 그대로 열립니다. |
| **종료 안내 표시** | 화면은 열리되 종료되었다는 안내가 붙습니다. |
| **접근 차단** | 상세 화면 자체가 열리지 않습니다. |

**종료 후 구매**

| 값 | 동작 |
|---|---|
| **구매 허용** | 기간이 지나도 상품을 살 수 있습니다. |
| **구매 차단** | 구매 버튼이 막힙니다. |

> **외부에 링크를 많이 뿌린 기획전이라면 `접근 허용`(또는 `종료 안내 표시`) + `구매 차단` 조합을 권장합니다.** 접근까지 차단하면 즐겨찾기·검색으로 들어온 고객이 빈 화면을 보게 되어 문의가 몰립니다.

### 3-6. 기타

- **품절 상품 숨기기** — 켜면 품절된 상품이 기획전 화면에서 빠집니다.
- **상단 추천 영역 제목** — 상세 맨 위 추천 영역의 제목입니다. 비우면 `추천 상품` 으로 나옵니다. (아래 5-5 참고)
- **안내 문구** — 기획전 화면에 표시할 유의사항 등을 적습니다.

여기까지 입력했으면 **저장**을 누릅니다. 저장하면 수정 화면으로 넘어가고, 이제 섹션과 상품을 넣을 수 있습니다.

## 4. 2단계 — 섹션(탭) 만들기

**섹션은 고객 화면에서 탭이 됩니다.** "베스트 / 신상품 / 1+1"처럼 나눠 보여 주고 싶을 때 사용합니다.

- 섹션을 **하나도 만들지 않으면** 탭 없이 상품이 한 덩어리로 보입니다. 상품이 적으면 이대로도 충분합니다.
- 섹션을 **2개 이상** 만들면 고객 화면 상단에 탭이 생깁니다.

수정 화면의 **섹션 영역**에서 **섹션 추가**를 누르고 다음을 입력합니다.

| 입력란 | 설명 |
|---|---|
| **표시명** | 탭에 보이는 이름 (예: `베스트`) |
| **탭 주소** | 주소의 `?tab=` 뒤에 붙는 값입니다. **표시명에서 자동으로 만들어지며, 입력칸이 없습니다.** 이미 만든 섹션은 기존 주소가 그대로 유지됩니다. |
| **유형** | 아래 표 참고 |
| **내용** | 유형이 `자유 HTML`일 때만 나타납니다. 글자 꾸미기 도구가 있는 편집기로 작성합니다. |

**섹션 유형 3가지**

| 유형 | 화면 |
|---|---|
| **상품 그리드** | 상품이 격자로 쭉 나열됩니다. 상품이 많을 때 기본 선택. |
| **상품 캐러셀** | 상품이 가로로 넘겨 보는 형태로 나옵니다. 상품이 적을 때 어울립니다. |
| **자유 HTML** | 상품 대신 직접 작성한 내용이 들어갑니다. 이벤트 안내·상세 이미지 등에 사용합니다. **편집기로 작성하므로 HTML 을 몰라도 됩니다.** |

> **자유 HTML 은 편집기(워드처럼 쓰는 도구)로 작성합니다.** 굵게·목록·링크·이미지·표를 도구 모음에서 고르면 됩니다. 태그를 직접 적을 필요가 없습니다.

> **섹션을 새로 추가해도 편집기가 함께 만들어집니다.** [섹션 추가] 후 유형을 `자유 HTML` 로 바꾸면 그 자리에 편집기가 나타납니다.

섹션을 입력한 뒤 **섹션 저장** 버튼을 눌러야 반영됩니다.

## 5. 3단계 — 상품 담기

수정 화면의 **상품 연결** 영역에서 **[상품 선택]** 버튼을 눌러 팝업에서 고릅니다. **여러 상품을 한 번에 담을 수 있습니다.**

### 5-1. 담는 순서

1. **담을 위치를 먼저 고릅니다.** (`전체 탭 (섹션 미배정)`을 고르면 어느 섹션에도 속하지 않고 전체에 보입니다.)
2. **[상품 선택]** 버튼을 누릅니다. 팝업이 열리면서 **담을 수 있는 상품이 바로 보입니다.** 검색어를 몰라도 됩니다.
3. 필요하면 아래 필터로 좁힙니다.
4. 담을 상품의 **체크박스를 켭니다.** 여러 개를 골라도 됩니다. 목록 전체를 한 번에 고르려면 상단의 **[이 목록 전체 선택]**.
5. 맨 아래 **[선택한 상품 담기]** 를 누릅니다.

담은 뒤에는 화면 위에 `N개 상품을 담았습니다` 안내가 뜹니다. 이미 담겨 있던 상품이 섞여 있으면 **그 상품만 빼고 나머지가 담깁니다** (`중복 1개 제외`처럼 함께 표시됩니다).

### 5-2. 팝업의 필터 3가지

| 필터 | 설명 |
|---|---|
| **검색어** | 상품명 또는 상품코드로 찾습니다. **비워 두면 전체**가 나옵니다. |
| **브랜드** | 입력칸을 **클릭만 해도** 이 몰에 상품이 있는 브랜드가 목록으로 뜹니다. 골라서 좁힙니다. |
| **카테고리** | **대분류 → 중분류 → 소분류** 순서로 좁힙니다. 대분류만 골라도 **그 아래 상품이 전부** 나옵니다. |

> **기획전 유형이 범위를 잠급니다.**
> - 유형이 **브랜드**면 브랜드 필터가 사라지고 **그 브랜드 상품만** 나옵니다.
> - 유형이 **카테고리**면 **지정한 분류 안에서만** 더 좁힐 수 있습니다.
>
> 잠긴 범위는 바꿀 수 없습니다. 다른 브랜드 상품을 담아야 한다면 **기본 정보의 유형·대상 브랜드를 먼저 바꾸고 저장**하세요.

### 5-3. 팝업에서 알아둘 것

- **이미 담긴 상품은 흐리게 표시되고 `담김` 이라고 나옵니다.** 체크할 수 없습니다. 이 판정은 **위에서 고른 담을 위치(섹션) 기준**입니다. 같은 상품이라도 **다른 섹션에는 담을 수 있습니다.**
- 목록은 한 번에 24개씩 보여 줍니다. 더 있으면 아래 **[더 보기]** 를 누르세요.
- **검색어·필터를 바꾸면 그때까지의 선택이 풀립니다.** 필터를 먼저 정하고 고르는 편이 편합니다.
- 유형이 `브랜드`인데 대상 브랜드를 비워 뒀다면, [상품 선택] 옆에 **경고 문구**가 뜹니다. 이 상태에서는 범위가 잠기지 않아 전체 상품이 나옵니다.

### 5-4. 담긴 상품 다듬기

담긴 상품은 아래 표에 줄줄이 나오며, 각 줄마다 다음을 지정할 수 있습니다.

| 항목 | 설명 |
|---|---|
| **순서** | 숫자가 작을수록 앞에 나옵니다. |
| **섹션** | 이 상품이 속할 탭. 나중에 바꿀 수 있습니다. |
| **배지** | 상품 카드 위에 붙는 짧은 문구. 최대 50자 (예: `단독`, `1+1`) |
| **MD 코멘트** | 상품 아래 붙는 한 줄 추천글. 최대 200자 |
| **상단 추천** | 켜면 상세 화면 **맨 위 추천 영역**에 크게 나옵니다. 아래 5-5 참고. |
| **노출** | 끄면 이 기획전에서만 상품이 숨겨집니다. |
| **구매 허용** | 끄면 보여만 주고 구매 버튼이 막힙니다. |
| **상단 고정** | 켜면 **그 목록 안에서** 순서와 관계없이 위로 올라옵니다. (추천 영역과 다릅니다) |
| **제외** | 이 기획전에서 상품을 빼냅니다. (상품 자체는 삭제되지 않습니다.) |

> **`상단 고정`과 `상단 추천`은 다른 기능입니다.** 고정은 *그 목록 안에서 앞자리로* 보내는 것이고, 추천은 *화면 맨 위 별도 영역에* 크게 세우는 것입니다. 둘 다 켜도 되고, 하나만 켜도 됩니다.

**표를 수정한 뒤에는 반드시 상품 저장 버튼을 누르세요.** 누르지 않고 화면을 벗어나면 변경이 사라집니다.

### 5-5. 상단 추천 상품 영역

기획전 상세 화면 **맨 위**(제목 바로 아래, 탭 위)에 **골라 놓은 상품 몇 개를 크게** 보여 주는 영역입니다. 예전에 대표 이미지 배너가 깔려 있던 자리입니다.

**켜는 방법**

1. 상품 연결 표에서 올리고 싶은 상품의 **상단 추천** 체크박스를 켭니다.
2. **상품 설정 저장** 을 누릅니다.

그게 전부입니다. **하나도 켜지 않으면 영역 자체가 나오지 않습니다**(빈 칸이 생기지 않습니다).

| 알아둘 것 | 내용 |
|---|---|
| **최대 개수** | **8개**까지 나옵니다. 더 켜면 앞 순서 8개만 나오고, 화면에 `앞의 8개만 노출됩니다` 안내가 뜹니다. |
| **순서** | 표의 **순서** 값을 따릅니다. (상단 고정을 켠 상품이 먼저) |
| **아래 목록에도 나오나요** | **나옵니다.** 위에서 먼저 보여 주고 원래 자리에도 그대로 둡니다. 중복이 아니라 의도된 동작입니다. |
| **영역 제목** | 기본값은 `추천 상품` 입니다. 바꾸려면 **전시 설정 &gt; 상단 추천 영역 제목** 에 입력하세요 (예: `MD가 고른 베스트`). |
| **숨겨진 상품** | 노출을 끈 상품·품절 숨김에 걸린 상품은 추천을 켜도 나오지 않습니다. 아래 목록과 같은 기준입니다. |

> **전문관에서도 똑같이 쓸 수 있습니다.** 전문관은 대표 이미지 아래에 추천 영역이 붙습니다.

> **상품 원본이 우선입니다.** 기획전에서 `노출`을 켜 두어도, 상품 관리에서 그 상품이 **판매중지·품절**이면 기획전에서도 정상적으로 팔리지 않습니다. 오픈 전에 상품 관리에서 재고와 판매 상태를 확인하세요.

2. **종료일을 비워 둡니다.** (시작일은 필수이므로 오픈 날짜를 넣습니다.)

특정 브랜드 프로모션이라면 반드시 지정하세요. 지정하지 않으면 기획전 목록에서만 볼 수 있습니다.

> **이 배너는 기획전 상세 화면에는 따라 나오지 않습니다.** 목록 전용입니다.

## 10. 자주 막히는 상황

**Q. 기획전을 만들었는데 고객 화면에 안 보입니다.**
아래를 순서대로 확인하세요.
1. **상태**가 `발행`인가? (`임시저장`이면 안 보입니다)
2. **목록 노출**이 켜져 있는가?
3. **기간**이 오늘을 포함하는가? (시작일이 미래면 `예정`이라 안 보입니다)
4. **유형**이 `전문관`으로 되어 있지 않은가? (전문관은 기획전 목록에 안 나옵니다)
5. **편집 중인 몰**이 맞는가? (다른 몰에 만들었을 수 있습니다)

**Q. 섹션·상품 영역이 화면에 없습니다.**
아직 저장 전입니다. 기본 정보를 먼저 **저장**하면 수정 화면에서 나타납니다.

**Q. 상품을 담았는데 화면에 안 나옵니다.**
① 상품 줄의 **노출**이 꺼져 있는지, ② **품절 상품 숨기기**가 켜진 상태에서 그 상품이 품절인지, ③ **상품 저장** 버튼을 눌렀는지 확인하세요.

**Q. 담고 싶은 상품이 [상품 선택] 팝업에 없습니다.**
1. **기획전 유형이 범위를 잠갔을 수 있습니다.** 유형이 `브랜드`면 지정한 브랜드 상품만, `카테고리`면 지정한 분류의 상품만 나옵니다. 팝업 안에 잠금 안내(자물쇠 표시)가 있는지 보세요. 담아야 한다면 **기본 정보에서 유형이나 대상을 바꾸고 저장**한 뒤 다시 여세요.
2. **검색어·브랜드·카테고리 필터**가 걸려 있는지 확인하세요. 검색어를 비우면 전체가 나옵니다.
3. 이미 담은 상품은 `담김` 으로 흐리게 표시되어 다시 고를 수 없습니다.
4. **다른 몰의 상품은 나오지 않습니다.** 편집 중인 몰을 확인하세요.

**Q. 팝업에서 여러 개를 골랐는데 일부만 담겼습니다.**
정상입니다. **이미 담겨 있던 상품은 건너뜁니다.** 담은 뒤 안내에 `중복 N개 제외`로 몇 개가 빠졌는지 표시됩니다.

**Q. 기획전 상세 맨 위에 대표 이미지가 안 나옵니다.**
정상입니다. 기획전 상세에는 **상단 이미지 배너가 없습니다.** 그 자리는 추천 상품 영역이 씁니다. PC·모바일 대표 이미지는 전문관에만 쓰입니다. (위 3-3 참고)

**Q. 상단 추천 영역이 안 보입니다.**
① 상품 연결 표에서 **상단 추천**을 켠 상품이 하나도 없으면 영역째 나오지 않습니다. ② 켠 뒤 **상품 설정 저장**을 눌렀는지 확인하세요. ③ 추천을 켠 상품이 **노출 꺼짐·품절 숨김**에 걸려 있으면 나오지 않습니다. (위 5-5 참고)

**Q. 추천으로 켠 상품이 아래 목록에도 또 보입니다.**
정상입니다. 위에서 크게 한 번 보여 주고 원래 자리에도 그대로 둡니다.

**Q. 기획전 목록 맨 위에 아무것도 안 보입니다.**
목록 상단 배너는 **배너 관리 &gt; 메뉴별 배너 &gt; 기획전**에서 등록합니다. 등록하지 않으면 비어 있는 것이 정상입니다. (위 9장 참고)

**Q. 탭이 안 생깁니다.**
섹션이 1개 이하면 탭이 표시되지 않습니다. 2개 이상 만들고 **섹션 저장**을 누르세요.

**Q. 기획전에서 바로 결제가 안 됩니다.**
정상입니다. 기획전은 전시 화면이고, 결제는 상품 상세 화면에서 이뤄집니다.

**Q. 기획전에 적힌 가격과 실제 결제 금액이 다릅니다.**
해당 상품에 쇼핑특가가 걸려 있을 수 있습니다. (위 8장 참고)

**Q. 링크를 눌렀더니 화면이 안 열립니다.**
기간이 끝났고 **종료 후 접근**이 `접근 차단`으로 되어 있을 수 있습니다.

---

## 11. 아직 지원하지 않는 것

다음은 현재 기능이 없어 수동으로 처리해야 합니다.

- 엑셀/CSV로 상품 일괄 담기 (팝업에서 골라 담는 것까지만 가능합니다)
- 조건(카테고리·가격 등)을 걸어 **상품이 자동으로 들고 나게** 하기 — 팝업의 카테고리·브랜드 필터는 **찾을 때만** 쓰이고, 담은 뒤에는 고정됩니다
- 기획전별 매출 집계
- 추천 상품 영역의 **자체 순서 지정** (상품 표의 순서를 따릅니다)
