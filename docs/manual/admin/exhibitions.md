# 기획전 · 전문관 관리

여러 상품을 하나의 주제로 모아 보여 주는 **전시 랜딩 화면**을 만드는 메뉴입니다. "가을 보양 기획전", "설 선물세트 모음"처럼 **기간을 정해 운영하는 기획전**과, 종료일 없이 **상시 운영하는 전문관**을 이 화면 한 곳에서 함께 관리합니다.

> **기획전은 "전시 화면"입니다.** 고객은 기획전에서 상품을 둘러보고, **실제 결제는 각 상품 상세 화면에서** 합니다. 기획전 화면에서 바로 결제하지 않습니다.

---

## 1. 시작하기 전에 — 준비물 3가지

기획전을 만들기 전에 아래가 준비되어 있어야 합니다. 준비 없이 시작하면 중간에 막힙니다.

| 준비물 | 왜 필요한가 | 어디서 만드나 |
|---|---|---|
| **담을 상품** | 기획전은 이미 등록된 상품을 골라 담는 화면입니다. 상품을 여기서 새로 만들 수는 없습니다. | 상품 관리 |
| **이미지 4종** | 목록 썸네일 / PC 대표 / 모바일 대표 / 공유용. 미리 만들어 두면 등록이 한 번에 끝납니다. | 이미지 편집 도구 (용도는 아래 3-3 참고) |
| **브랜드 분류** (브랜드 기획전인 경우만) | 브랜드관에도 함께 노출하려면 브랜드가 먼저 있어야 합니다. | 카테고리 관리 |

> **지금 편집 중인 몰을 반드시 확인하세요.** 기획전은 **몰별로 따로 저장**됩니다. 관리자 화면 상단의 편집 몰이 A몰인데 B몰에 만들 생각이었다면, 저장해도 B몰에는 나타나지 않습니다.

## 2. 전체 흐름 한눈에 보기

```
① 기본 정보 입력 → 저장    (여기까지 해야 기획전이 "생깁니다")
        ↓
② 섹션(탭) 만들기          (예: 베스트 / 신상품 / 1+1)
        ↓
③ 상품 검색해서 담기       (섹션에 배정 + 배지·코멘트 달기)
        ↓
④ 상태를 "발행"으로 변경   (+ 목록 노출 켜기)
        ↓
⑤ 고객 화면에서 확인
```

**①번 저장을 먼저 해야 ②③이 가능합니다.** 등록 화면에서는 섹션·상품 영역이 아직 열리지 않습니다. 저장 후 이어지는 수정 화면에서 계속 작업하세요.

### 3-1. 이름과 설명

| 입력란 | 필수 | 설명 | 예시 |
|---|---|---|---|
| **기획전명** | 필수 | 고객에게 보이는 제목입니다. 최대 200자. | `가을 보양 기획전` |
| **주소용 이름(slug)** | 선택 | 기획전 주소에 들어가는 영문 이름입니다. **비워 두면 자동으로 만들어집니다.** | `autumn-boyang` |
| **짧은 설명** | 선택 | 목록에서 제목 아래 한 줄로 보입니다. 최대 500자. | `환절기 건강을 챙기는 30가지` |
| **상세 설명** | 선택 | 기획전 화면 상단에 들어가는 소개글입니다. | 여러 줄 입력 가능 |

> **주소용 이름은 나중에 바꾸지 마세요.** 고객에게 링크를 뿌린 뒤 이 값을 바꾸면 **기존 링크가 전부 깨집니다.** 문자·카카오톡으로 발송한 링크도 마찬가지입니다.

### 3-2. 유형 · 상태 · 기간

**기획전 유형** — 기획전의 성격을 고르는 값입니다. 대부분은 분류·검색용이지만, **`전문관`만 동작이 완전히 다릅니다.**

| 유형 | 동작 |
|---|---|
| 테마 / 브랜드 / 시즌 / 카테고리 / 콜라보 / 방송연계 | 일반 기획전. **기획전 목록에 노출**됩니다. |
| **전문관 (상시 · 종료일 비움)** | **기획전 목록에서 빠지고, 전문관 메뉴에만 노출**됩니다. 아래 6장 참고. |

**상태**

| 상태 | 의미 |
|---|---|
| **임시저장** | 준비 중. 고객에게 보이지 않습니다. 새로 만들면 기본값입니다. |
| **발행** | 고객에게 공개됩니다. |
| **숨김** | 공개를 잠시 내립니다. 내용은 그대로 남습니다. |

**기간**

- **시작일**은 **필수**입니다. 날짜와 시각을 함께 지정합니다.
- **종료일**은 비울 수 있습니다. **비우면 종료 없이 계속 진행**됩니다. 전문관은 반드시 비웁니다.
- 기간에 따라 목록에 **예정 / 진행중 / 종료** 가 자동으로 표시됩니다.

**상세 템플릿** — 기획전 상세 화면의 생김새입니다.

> 현재 **`기본 탭형`만 실제로 동작합니다.** 목록에 `(2차)`라고 적힌 이미지 스토리형·카테고리 매장형·브랜드관형을 골라도 화면은 기본 탭형으로 나옵니다. 혼란을 피하려면 **기본 탭형을 그대로 두세요.**

### 3-3. 이미지 4종

| 이미지 | 어디에 보이나 | 없으면 |
|---|---|---|
| **목록 썸네일** | 기획전 목록의 작은 카드 이미지 | 빈 칸으로 보입니다 |
| **PC 대표 이미지** | 기획전 화면 최상단의 큰 가로 이미지 | 상단이 비어 허전해집니다 |
| **모바일 대표 이미지** | 모바일 전용 상단 이미지 | **PC 이미지가 대신 쓰입니다** (가로형이라 잘려 보일 수 있음) |
| **공유 이미지** | 카카오톡·SNS로 링크를 보냈을 때 뜨는 미리보기 | 미리보기가 안 뜨거나 엉뚱한 이미지가 잡힙니다 |

수정 화면에서는 각 이미지 옆에 **삭제 체크박스**가 있습니다. 체크하고 저장하면 그 이미지가 지워집니다.

### 3-4. 노출 설정 (체크박스 3개)

| 항목 | 켜면 |
|---|---|
| **목록 노출** | 기획전 목록 화면에 나타납니다. |
| **검색 노출** | 사이트 내 검색 결과에 잡힙니다. |
| **공유 버튼** | 고객 화면에 SNS 공유 버튼이 표시됩니다. |

> **고객에게 보이려면 `상태 = 발행` + `목록 노출 = 켜짐` 두 가지가 모두** 필요합니다. 하나만 해 두고 "안 보인다"고 하는 경우가 가장 많습니다.

### 3-5. 종료 후 정책

기간이 끝난 뒤 기획전을 어떻게 할지 정합니다.

**종료 후 접근**

| 값 | 동작 |
|---|---|
| **접근 허용** | 기간이 지나도 화면이 그대로 열립니다. |
| **종료 안내 표시** | 화면은 열리되 종료되었다는 안내가 붙습니다. |
| **접근 차단** | 상세 화면 자체가 열리지 않습니다. |

**종료 후 구매**

| 값 | 동작 |
|---|---|
| **구매 허용** | 기간이 지나도 상품을 살 수 있습니다. |
| **구매 차단** | 구매 버튼이 막힙니다. |

> **외부에 링크를 많이 뿌린 기획전이라면 `접근 허용`(또는 `종료 안내 표시`) + `구매 차단` 조합을 권장합니다.** 접근까지 차단하면 즐겨찾기·검색으로 들어온 고객이 빈 화면을 보게 되어 문의가 몰립니다.

### 3-6. 기타

- **품절 상품 숨기기** — 켜면 품절된 상품이 기획전 화면에서 빠집니다.
- **안내 문구** — 기획전 화면에 표시할 유의사항 등을 적습니다.

여기까지 입력했으면 **저장**을 누릅니다. 저장하면 수정 화면으로 넘어가고, 이제 섹션과 상품을 넣을 수 있습니다.

## 4. 2단계 — 섹션(탭) 만들기

**섹션은 고객 화면에서 탭이 됩니다.** "베스트 / 신상품 / 1+1"처럼 나눠 보여 주고 싶을 때 사용합니다.

- 섹션을 **하나도 만들지 않으면** 탭 없이 상품이 한 덩어리로 보입니다. 상품이 적으면 이대로도 충분합니다.
- 섹션을 **2개 이상** 만들면 고객 화면 상단에 탭이 생깁니다.

수정 화면의 **섹션 영역**에서 **섹션 추가**를 누르고 다음을 입력합니다.

| 입력란 | 설명 |
|---|---|
| **표시명** | 탭에 보이는 이름 (예: `베스트`) |
| **탭 주소** | 주소의 `?tab=` 뒤에 붙는 값입니다. **표시명에서 자동으로 만들어지며, 입력칸이 없습니다.** 이미 만든 섹션은 기존 주소가 그대로 유지됩니다. |
| **유형** | 아래 표 참고 |
| **내용** | 유형이 `자유 HTML`일 때만 나타납니다. 글자 꾸미기 도구가 있는 편집기로 작성합니다. |

**섹션 유형 3가지**

| 유형 | 화면 |
|---|---|
| **상품 그리드** | 상품이 격자로 쭉 나열됩니다. 상품이 많을 때 기본 선택. |
| **상품 캐러셀** | 상품이 가로로 넘겨 보는 형태로 나옵니다. 상품이 적을 때 어울립니다. |
| **자유 HTML** | 상품 대신 직접 작성한 내용이 들어갑니다. 이벤트 안내·상세 이미지 등에 사용합니다. **편집기로 작성하므로 HTML 을 몰라도 됩니다.** |

> **자유 HTML 은 편집기(워드처럼 쓰는 도구)로 작성합니다.** 굵게·목록·링크·이미지·표를 도구 모음에서 고르면 됩니다. 태그를 직접 적을 필요가 없습니다.

> **섹션을 새로 추가해도 편집기가 함께 만들어집니다.** [섹션 추가] 후 유형을 `자유 HTML` 로 바꾸면 그 자리에 편집기가 나타납니다.

섹션을 입력한 뒤 **섹션 저장** 버튼을 눌러야 반영됩니다.

## 5. 3단계 — 상품 담기

수정 화면의 **상품 검색해서 담기** 영역을 사용합니다.

1. 담을 **섹션을 먼저 선택**합니다. (`전체 탭`을 고르면 어느 섹션에도 속하지 않고 전체에 보입니다.)
2. 상품명을 검색합니다.
3. 검색 결과에서 담을 상품을 골라 추가합니다.

담긴 상품은 아래 표에 줄줄이 나오며, 각 줄마다 다음을 지정할 수 있습니다.

| 항목 | 설명 |
|---|---|
| **순서** | 숫자가 작을수록 앞에 나옵니다. |
| **섹션** | 이 상품이 속할 탭. 나중에 바꿀 수 있습니다. |
| **배지** | 상품 카드 위에 붙는 짧은 문구. 최대 50자 (예: `단독`, `1+1`) |
| **MD 코멘트** | 상품 아래 붙는 한 줄 추천글. 최대 200자 |
| **노출** | 끄면 이 기획전에서만 상품이 숨겨집니다. |
| **구매 허용** | 끄면 보여만 주고 구매 버튼이 막힙니다. |
| **상단 고정** | 켜면 순서와 관계없이 위로 올라옵니다. |
| **제외** | 이 기획전에서 상품을 빼냅니다. (상품 자체는 삭제되지 않습니다.) |

**표를 수정한 뒤에는 반드시 상품 저장 버튼을 누르세요.** 누르지 않고 화면을 벗어나면 변경이 사라집니다.

> **상품 원본이 우선입니다.** 기획전에서 `노출`을 켜 두어도, 상품 관리에서 그 상품이 **판매중지·품절**이면 기획전에서도 정상적으로 팔리지 않습니다. 오픈 전에 상품 관리에서 재고와 판매 상태를 확인하세요.

## 6. 전문관 만들기 (기간 없는 상시 매장)

전문관은 **"건강기능식품 전문관"처럼 늘 열려 있는 매장**입니다. 만드는 방법은 기획전과 같고, **세 가지만 다르게** 하면 됩니다.

<div class="manual-goto-bar">
  <span class="manual-goto-label">전문관 만들기</span>
  <a class="manual-goto" href="/admin/exhibitions/add" target="_blank">등록 화면 열기</a>
</div>

1. **기획전 유형**을 **`전문관 (상시 · 종료일 비움)`** 으로 선택합니다.
2. **종료일을 비워 둡니다.** (시작일은 필수이므로 오픈 날짜를 넣습니다.)
3. 나머지(섹션·상품·이미지)는 기획전과 똑같이 구성합니다.

**전문관과 기획전의 차이**

| | 일반 기획전 | 전문관 |
|---|---|---|
| 유형 값 | 테마·브랜드·시즌 등 | **전문관** |
| 종료일 | 보통 지정 | **비움 (상시)** |
| 기획전 목록 | 나옴 | **안 나옴** |
| 전문관 메뉴 | 안 나옴 | **나옴** |

> **"전문관을 만들었는데 기획전 목록에 없어요"는 정상입니다.** 두 목록은 서로 배타적입니다. 반대로 **일반 기획전은 전문관 메뉴에 절대 나오지 않습니다.** 만든 뒤 안 보인다면 가장 먼저 유형을 확인하세요.

**전문관 메뉴가 고객 화면에 보이지 않는다면**, 메뉴 자체가 꺼져 있을 수 있습니다.

## 7. 브랜드에 연결하기

등록·수정 화면의 **브랜드 지정** 칸에서 브랜드를 고르면, 그 **브랜드관 화면에 "브랜드 행사"로 함께 노출**됩니다.

특정 브랜드 프로모션이라면 반드시 지정하세요. 지정하지 않으면 기획전 목록에서만 볼 수 있습니다.

## 8. 쇼핑특가와 겹칠 때 (가격 주의)

기획전에 담은 상품에 **쇼핑특가가 걸려 있으면, 기획전 화면에서도 특가 가격으로 보입니다.**

기획전 배너에 "정가 대비 20% 할인"이라 적어 두었는데 그 상품이 특가 중이라면, **고객이 보는 금액이 배너 문구와 달라집니다.** 상품을 담기 전에 진행 중인 특가를 확인하세요.

## 9. 자주 막히는 상황

**Q. 기획전을 만들었는데 고객 화면에 안 보입니다.**
아래를 순서대로 확인하세요.
1. **상태**가 `발행`인가? (`임시저장`이면 안 보입니다)
2. **목록 노출**이 켜져 있는가?
3. **기간**이 오늘을 포함하는가? (시작일이 미래면 `예정`이라 안 보입니다)
4. **유형**이 `전문관`으로 되어 있지 않은가? (전문관은 기획전 목록에 안 나옵니다)
5. **편집 중인 몰**이 맞는가? (다른 몰에 만들었을 수 있습니다)

**Q. 섹션·상품 영역이 화면에 없습니다.**
아직 저장 전입니다. 기본 정보를 먼저 **저장**하면 수정 화면에서 나타납니다.

**Q. 상품을 담았는데 화면에 안 나옵니다.**
① 상품 줄의 **노출**이 꺼져 있는지, ② **품절 상품 숨기기**가 켜진 상태에서 그 상품이 품절인지, ③ **상품 저장** 버튼을 눌렀는지 확인하세요.

**Q. 탭이 안 생깁니다.**
섹션이 1개 이하면 탭이 표시되지 않습니다. 2개 이상 만들고 **섹션 저장**을 누르세요.

**Q. 기획전에서 바로 결제가 안 됩니다.**
정상입니다. 기획전은 전시 화면이고, 결제는 상품 상세 화면에서 이뤄집니다.

**Q. 기획전에 적힌 가격과 실제 결제 금액이 다릅니다.**
해당 상품에 쇼핑특가가 걸려 있을 수 있습니다. (위 8장 참고)

**Q. 링크를 눌렀더니 화면이 안 열립니다.**
기간이 끝났고 **종료 후 접근**이 `접근 차단`으로 되어 있을 수 있습니다.

## 10. 아직 지원하지 않는 것

다음은 현재 기능이 없어 수동으로 처리해야 합니다.

- 엑셀/CSV로 상품 일괄 담기 (상품은 검색해서 하나씩 담습니다)
- 조건(카테고리·가격 등)을 걸어 상품 자동 담기
- 기획전별 매출 집계
