# 상품 관리

쇼핑몰에 **판매할 상품을 등록·수정·삭제**하고, **가격·재고·이미지·판매 상태**를 관리하는 메뉴입니다.

---

## 상품 목록에서 할 수 있는 일

- **등록된 상품 전체**를 한 화면에서 볼 수 있습니다.
- 각 상품의 **썸네일, 이름, 카테고리, 재고, 가격, 판매 상태**가 표시됩니다.
- **상품 등록**, **수정**, **상세 보기**, **삭제**, **판매 상태 일괄 변경**을 할 수 있습니다.

---

## 상품 등록하기

1. **상품 관리** 화면에서 **상품 등록** 또는 **추가** 버튼을 클릭합니다.
2. **기본 정보**를 입력합니다.
   - **상품명:** 고객에게 보이는 이름 (필수)
   - **카테고리:** 대분류→중분류→소분류를 **단계별 드롭다운**으로 고릅니다(선택한 상위 아래에 하위만 나타남). 목록에 없으면 **직접 입력**에 `대>중>소` 경로를 적으면 없는 단계가 **자동으로 만들어집니다**(최대 3단계). 카테고리는 전 몰 공용 마스터에 저장됩니다.
   - **공급사:** 제조사·브랜드명 등 (선택)
   - **판매 상태:** 판매중 / 품절 / 숨김 중 선택
3. **가격·재고**를 입력합니다.
   - **정가(원가):** 소비자에게 보여 주는 기준 가격
   - **할인율:** % 단위 (입력하면 판매가가 자동 계산될 수 있음)
   - **판매가:** 실제 판매 금액
   - **재고 수량:** 보유 수량 (품절 시 0으로 두거나 상태를 "품절"로 변경). **옵션이 있는 상품은 재고를 조합(SKU)별로 관리**하며, 이때 상품 목록의 재고는 **각 조합 재고의 합**으로, 판매가는 **범위(최저~최고)** 로 표시됩니다. 아래 "옵션·SKU 관리" 참고.
4. **대표 이미지:** 상품 목록·상세에 쓰일 메인 사진을 업로드합니다. (권장: 정사각형 또는 가로형, 5MB 이하)
5. **썸네일 이미지:** 목록용 작은 이미지 (없으면 대표 이미지를 쓰는 경우가 많습니다)
6. **상세 설명:** 상품 상세 페이지에 들어갈 설명·이미지를 작성합니다. 에디터에서 글꼴·이미지·표 등을 넣을 수 있습니다.
7. **서브 이미지:** 상세 페이지에 추가로 보여 줄 이미지 (최대 10장 등 제한이 있을 수 있음)
8. **동영상:** 상품 동영상이 있으면 업로드할 수 있습니다. (선택)
9. 모든 입력이 끝나면 **저장** 또는 **등록** 버튼을 눌러 등록합니다.

**상세 설명 안에 이미지 넣기:** 에디터에서 "이미지 삽입"을 선택한 뒤, PC에 있는 이미지를 선택하면 업로드되어 본문에 들어갑니다.

---

## 상품 수정하기

1. 상품 목록에서 해당 상품의 **수정** 버튼을 클릭합니다.
2. 등록할 때와 같은 폼이 열리며, 기존 값이 채워져 있습니다.
3. 바꾸고 싶은 항목만 수정한 뒤 **저장**합니다.
4. **이미지를 바꾸지 않으려면** 기존 이미지를 그대로 두면 됩니다. 새 파일을 선택하면 기존 이미지가 교체됩니다.

---

## 상품 삭제하기

1. 상품 목록에서 해당 상품의 **삭제** 버튼을 클릭합니다.
2. "정말 삭제하시겠습니까?" 같은 확인 메시지가 나오면 **확인**을 누릅니다.
3. 삭제되면 목록에서 사라집니다.

**주의:** 이미 주문된 상품은 삭제해도 주문 내역에는 남을 수 있습니다. 판매만 중단하고 싶다면 **판매 상태를 "숨김"**으로 바꾸는 방법을 사용할 수 있습니다.

---

## 판매 상태 일괄 변경

여러 상품을 한 번에 **판매중 / 품절 / 숨김**으로 바꿀 수 있습니다.

1. 상품 목록에서 상태를 바꿀 상품에 **체크**를 합니다.
2. 상단 또는 하단의 **상태 변경** 드롭다운에서 원하는 상태(판매중, 품절, 숨김)를 선택합니다.
3. **적용** 버튼을 누르면 선택한 상품들의 상태가 한 번에 바뀝니다.

재고가 없을 때는 **품절**로, 일시적으로 노출을 끄고 싶을 때는 **숨김**으로 설정하면 됩니다.

---

## 상품 유형 (단일 / 옵션)

상품은 크게 두 가지로 나뉩니다. 상품 목록에 **단일 / 옵션 배지**로 구분되어 표시됩니다.

- **단일 상품:** 선택지가 없는 보통 상품. 상품 자체가 하나의 재고 단위입니다.
- **옵션 상품:** 색상·사이즈처럼 **고객이 골라야 하는 선택지**가 있는 상품. 옵션 조합마다 재고·가격을 따로 두는 **SKU** 단위로 관리합니다.

> 여러 상품을 하나로 묶어 파는 **세트·기획상품**(묶음·세트·선물세트·선택형세트)은 별도 메뉴에서 관리합니다 → [세트·기획상품 관리](./derived-products.md).

## 옵션·SKU 관리 (색상·사이즈 등)

색상·사이즈처럼 **선택지가 있는 상품**은 옵션과 조합별 재고·가격을 따로 설정합니다. 예전에는 옵션별로 상품을 여러 개 만들어야 했지만, 이제 **한 상품 안에서** 관리합니다.

### 어디서 여나

- **상품 목록**에서 해당 상품 줄의 **옵션·SKU** 버튼
- 또는 상품 **수정 화면 하단**의 **옵션·SKU 관리** 버튼

옵션 편집은 상품 기본 정보 폼과 분리된 **전용 화면**에서 이뤄집니다.

### 무엇을 하나

1. **옵션명**을 정합니다. (예: 색상, 사이즈)
2. 각 옵션명의 **옵션값**을 나열합니다. (예: 색상 = 블랙/화이트, 사이즈 = S/M/L)
3. 값들의 **조합마다 SKU 한 개**가 만들어집니다. (예: 블랙·M, 화이트·L …)
4. **조합(SKU)별로 재고·판매가·SKU 코드**를 입력합니다.

> **카테고리 추천 옵션:** 카테고리별로 자주 쓰는 옵션 템플릿을 미리 정해 두면(관리자 상품 관리의 **카테고리 옵션** 화면), 그 카테고리 상품의 옵션 편집을 열 때 추천 옵션이 자동으로 채워져 입력이 빨라집니다. (강제는 아니며 지우거나 바꿀 수 있습니다.)

### 고객 화면에서는

옵션 상품은 상세 페이지에 **옵션 선택 UI**가 뜹니다. 고객이 색상·사이즈를 고르면 그 조합(SKU)의 **가격·재고로 바뀌고**, 조합을 골라야 장바구니·바로구매가 활성화됩니다. 없는 조합을 고르면 "판매하지 않습니다"로 안내됩니다.

---

## 판매 시작일과 신상품

상품이 **신상품 코너에 나오는 기준**은 **판매 시작일**입니다.

- 상품 등록·수정 화면의 **판매 시작일**을 기준으로, **100일 동안 자동으로 신상품에 노출**됩니다. (이 기간은 시스템 설정에서 바꿀 수 있습니다.)
- 여러 상품의 판매 시작일을 **상품 목록에서 한꺼번에 지정**할 수도 있습니다. 상품을 체크한 뒤 판매 시작일을 지정하면 됩니다.
- **상품 구분에서 NEW 를 체크**하면 **기간과 상관없이 계속** 신상품에 노출됩니다. 100일이 지나도 내려가지 않으므로, 필요 없어지면 체크를 꼭 해제하세요.

---

## 상품 구분 (뱃지)

상품 등록·수정 화면의 **상품 구분**에 체크하면 상품에 표시가 붙고, 특정 코너에 노출됩니다.

- **NEW:** 기간과 무관하게 신상품 코너에 계속 노출됩니다.
- **추천:** 고객 화면 **추천 메뉴의 "MD 추천" 영역**에 나옵니다.

---

## URL 로 상품 가져오기

상품 등록 화면에서 **상품 페이지 주소(URL)를 입력**하면 상품 정보를 **자동으로 가져올** 수 있습니다.

1. **상품 등록** 화면을 엽니다.
2. **URL 로 가져오기** 입력란에 상품 페이지 주소를 붙여 넣습니다.
3. 가져온 정보가 폼에 채워집니다.
4. **내용을 반드시 확인·수정**한 뒤 저장합니다.

**한 번에 1건씩만** 가져올 수 있습니다. 여러 건을 한꺼번에 가져오는 기능은 없습니다.

---

## 가격이 다르게 보일 때

- **쇼핑특가가 적용된 상품**은 고객 화면에 **특가 가격으로 표시**됩니다. 상품 관리에 입력한 판매가와 다르게 보이는 것은 정상입니다.
- **아울렛에 등록된 상품**은 설정에 따라 **일반 상품 목록에서 빠질 수 있습니다.** (아울렛 설정의 "일반 상품 목록에 아울렛 상품 표시" 항목)

---

## 아직 안 되는 것

아래 기능은 **지원하지 않습니다.** 대체 방법을 참고하세요.

| 안 되는 것 | 대신 |
|-------------|------|
| **엑셀/CSV 일괄 업로드** | 상품을 한 건씩 등록하거나, URL 로 가져오기를 이용합니다. |
| **재고 이력(입출고 기록) 조회** | 현재 재고 수량만 확인할 수 있습니다. |

## AI 추천 문구 (사용 시)

상품 등록·수정 폼에서 **AI 추천 문구 생성** 버튼이 있다면, 상품명·카테고리·공급사를 바탕으로 **상품 소개 문구**를 자동으로 만들어 줍니다. 생성된 문구를 그대로 쓰거나 고쳐서 상세 설명에 넣을 수 있습니다.  
(이 기능을 쓰려면 사이트 설정에서 OpenAI 등 연동이 되어 있어야 할 수 있습니다.)

---

## 상품 SEO 미리보기 (사용 시)

검색엔진·SNS에 노출될 때 어떻게 보이는지 **미리보기** 기능이 있는 경우, 상품 상세 또는 수정 화면에서 "SEO 미리보기"를 클릭해 확인할 수 있습니다. 제목·설명·이미지가 의도대로 나오는지 점검할 때 활용하세요.
