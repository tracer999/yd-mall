# 상품 관리

쇼핑몰에 **판매할 상품을 등록·수정·삭제**하고, **가격·재고·이미지·판매 상태**를 관리하는 메뉴입니다.

---

## 상품 목록에서 할 수 있는 일

- **등록된 상품 전체**를 한 화면에서 볼 수 있습니다.
- 각 상품의 **썸네일, 이름, 카테고리, 재고, 가격, 판매 상태**가 표시됩니다.
- **상품 등록**, **수정**, **상세 보기**, **삭제**, **판매 상태 일괄 변경**을 할 수 있습니다.

---

## 상품 등록하기

1. **상품 관리** 화면에서 **상품 등록** 또는 **추가** 버튼을 클릭합니다.
2. **기본 정보**를 입력합니다.
   - **상품명:** 고객에게 보이는 이름 (필수)
   - **카테고리:** 드롭다운에서 해당하는 카테고리를 선택합니다.
   - **공급사:** 제조사·브랜드명 등 (선택)
   - **판매 상태:** 판매중 / 품절 / 숨김 중 선택
3. **가격·재고**를 입력합니다.
   - **정가(원가):** 소비자에게 보여 주는 기준 가격
   - **할인율:** % 단위 (입력하면 판매가가 자동 계산될 수 있음)
   - **판매가:** 실제 판매 금액
   - **재고 수량:** 보유 수량 (품절 시 0으로 두거나 상태를 "품절"로 변경)
4. **대표 이미지:** 상품 목록·상세에 쓰일 메인 사진을 업로드합니다. (권장: 정사각형 또는 가로형, 5MB 이하)
5. **썸네일 이미지:** 목록용 작은 이미지 (없으면 대표 이미지를 쓰는 경우가 많습니다)
6. **상세 설명:** 상품 상세 페이지에 들어갈 설명·이미지를 작성합니다. 에디터에서 글꼴·이미지·표 등을 넣을 수 있습니다.
7. **서브 이미지:** 상세 페이지에 추가로 보여 줄 이미지 (최대 10장 등 제한이 있을 수 있음)
8. **동영상:** 상품 동영상이 있으면 업로드할 수 있습니다. (선택)
9. 모든 입력이 끝나면 **저장** 또는 **등록** 버튼을 눌러 등록합니다.

**상세 설명 안에 이미지 넣기:** 에디터에서 "이미지 삽입"을 선택한 뒤, PC에 있는 이미지를 선택하면 업로드되어 본문에 들어갑니다.

---

## 상품 수정하기

1. 상품 목록에서 해당 상품의 **수정** 버튼을 클릭합니다.
2. 등록할 때와 같은 폼이 열리며, 기존 값이 채워져 있습니다.
3. 바꾸고 싶은 항목만 수정한 뒤 **저장**합니다.
4. **이미지를 바꾸지 않으려면** 기존 이미지를 그대로 두면 됩니다. 새 파일을 선택하면 기존 이미지가 교체됩니다.

---

## 상품 삭제하기

1. 상품 목록에서 해당 상품의 **삭제** 버튼을 클릭합니다.
2. "정말 삭제하시겠습니까?" 같은 확인 메시지가 나오면 **확인**을 누릅니다.
3. 삭제되면 목록에서 사라집니다.

**주의:** 이미 주문된 상품은 삭제해도 주문 내역에는 남을 수 있습니다. 판매만 중단하고 싶다면 **판매 상태를 "숨김"**으로 바꾸는 방법을 사용할 수 있습니다.

---

## 판매 상태 일괄 변경

여러 상품을 한 번에 **판매중 / 품절 / 숨김**으로 바꿀 수 있습니다.

1. 상품 목록에서 상태를 바꿀 상품에 **체크**를 합니다.
2. 상단 또는 하단의 **상태 변경** 드롭다운에서 원하는 상태(판매중, 품절, 숨김)를 선택합니다.
3. **적용** 버튼을 누르면 선택한 상품들의 상태가 한 번에 바뀝니다.

재고가 없을 때는 **품절**로, 일시적으로 노출을 끄고 싶을 때는 **숨김**으로 설정하면 됩니다.

---

## AI 추천 문구 (사용 시)

상품 등록·수정 폼에서 **AI 추천 문구 생성** 버튼이 있다면, 상품명·카테고리·공급사를 바탕으로 **상품 소개 문구**를 자동으로 만들어 줍니다. 생성된 문구를 그대로 쓰거나 고쳐서 상세 설명에 넣을 수 있습니다.  
(이 기능을 쓰려면 사이트 설정에서 OpenAI 등 연동이 되어 있어야 할 수 있습니다.)

---

## 상품 SEO 미리보기 (사용 시)

검색엔진·SNS에 노출될 때 어떻게 보이는지 **미리보기** 기능이 있는 경우, 상품 상세 또는 수정 화면에서 "SEO 미리보기"를 클릭해 확인할 수 있습니다. 제목·설명·이미지가 의도대로 나오는지 점검할 때 활용하세요.
